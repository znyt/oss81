　　喝完酒，伸了个懒腰，张开嘴，一堆地精魄飞出来，自动落到了他的酒葫芦里，钟馗塞上塞子，在肚子上拍了一下，人迅速缩小，变成了正常体型，仍然比普通人高了两倍，比姚明还高，晃着脑袋来到叶少阳面前，咧嘴笑道：“小天师，你怎么走到哪里都需要人救？”
<br/>　　叶少阳无语。
<br/>　　钟馗拍了拍他肩膀，说道：“玩笑而已，诛仙剑阵，非同寻常，就算被关在里面的是我，靠自己的力量也是出不来的。”
<br/>　　叶少阳听了这话，心里多少有点安慰，不过这次遇伏跟以往还是不一样：以往虽然也总是让道风搭救，但自己在凶险时刻，总是能够反杀，至少不吃亏，这次的确是吃了一个大亏。当下笑道：“能被你搭救，那也是荣幸啊，以往都是我师兄救我。”
<br/>　　钟馗一听，道：“他这次却是来不了了。”
<br/>　　“什么意思？”叶少阳听出话头不对，“你知道他在哪儿？”
<br/>　　“就是不知道，才来你这里。”
<br/>　　“我更不知道，他要是在这，哪里会这样！”叶少阳无奈摇了摇头，想起刚才的战斗，道：“您口味可真重，什么都吃，不过你为啥不把影魅抓来吃了？”
<br/>　　钟馗在他脑门上拍了一把，“那影魅岂是好对付的。”
<br/>　　“你也打不过他？”
<br/>　　钟馗眼珠子转了一圈，想了下道：“小天师，你一向在人间晃荡，见到的邪物总归有限，你记住，三界之中，总有些邪物有着特殊的本领，不能以硬实力比对，譬如这影魅，来去无踪，我也抓不住他。”
<br/>　　当下挠了挠头，道：“这手段确实恼人，阴司怕是没人能捉拿他。”
<br/>　　叶少阳想到了什么，但没有直说，刚要开口，西边的丛林里突然有灵光飞天，像是在战斗，想起之前钟馗的话，知道那边八成是打起来了，正想过去看看，只看到一只大鸟从西边飞来，下方靠近地面的地方，也有一道人影在飞行着。
<br/>　　是鲲鹏和玉玑子。
<br/>　　“天地无极！唯我独尊！”
<br/>　　钟馗将帅旗丢了出去，帅旗迎风招展，不断释放出灵力，形成了一道强大的结界，挡住鲲鹏的去路。
<br/>　　鲲鹏一头撞在结界上，没能撞破，知道知道跑不掉了，降落下来，化作人形，望着钟馗躬身下拜，一脸惶恐得地求饶。
<br/>　　“钟天师，怜我修行不易，饶我一命！”
<br/>　　“饶你一命，之前被你杀得那些无辜生灵，你怎么你饶了他们。”叶少阳捏紧拳头，指了指从壕沟里爬上来的那些生灵，冷冷道：“饶了你，问他们答不答应！”
<br/>　　这些死里逃生的生灵，对鲲鹏这个始作俑者恨之入骨，一个个义愤填膺地看着他。
<br/>　　玉玑子站在鲲鹏身后，一脸的惶恐，不知所措地站在那里。
<br/>　　在他身后，黑压压一群人追了上来，为首的就是小马和吴嘉伟，小马掂着板砖，吼道：“小叶子，别杀他们，放着我来！”
<br/>　　在二人身后……是一大群身穿盔甲的战士。
<br/>　　阴司的鬼武士？
<br/>　　等他们走近些，叶少阳看到了走在最前面的那一群，笑容顿时在脸上绽开。
<br/>　　林三生，小青，小白，美华……甚至连雪琪都来了！
<br/>　　捉鬼联盟的兄弟们，差不多都来了。
<br/>　　来到这片空地上，众人扇形散开，将鲲鹏二人围起来。
<br/>　　两人面色如土，傻傻站着。
<br/>　　鲲鹏一咬牙，往前走了两步，面对叶少阳说道：“叶少阳，要杀便杀，想让我说出什么秘密，那是一个字也没有的。”
<br/>　　“我压根就没想过问你什么。”叶少阳语气带着压抑的平静。
<br/>　　鲲鹏愣了一下，道：“你们人多，想杀我倒是容易的。”
<br/>　　叶少阳笑道：“什么意思啊，不就是想让我跟你单挑吗？”
<br/>　　“你敢吗？”
<br/>　　“我来，老子一板砖拍死你个王八蛋！”小马吼道。
<br/>　　“你对付玉玑子！”
<br/>　　话没说完，叶少阳已经朝鲲鹏冲了过去。鲲鹏高高跃起，摇身一变，现出真身，一只五颜六色的大鹏鸟。
<br/>　　他打不过李浩然，也打不过道风，被这两人都虐过，但他自信自己至少不会败给叶少阳……这个人类。
<br/>　　他在轩辕山，如果不是借着星月奴的宠爱，担当重任，比硬实力的话他排不上号，但他至少是个“神仙”，是上古异兽的后裔，对于凡人，他一样都是看不上的。
<br/>　　叶少阳打出了五张灵符，飘摇着飞向鲲鹏，鲲鹏也全然无惧，翅膀扇动，卷起一阵罡风，将灵符吹落，人也从空中俯冲下来——他会飞，叶少阳不会飞，但他不会蠢到一直在空中呆着，一来反正逃不走，飞再久也只是白白消耗灵力，而且容易被当做靶子。
<br/>　　他在空中蓄力，朝着叶少阳俯冲，身体抖动，身上飞出了红、黄、蓝三支羽毛，如同三把利剑，挟带着三种不同的元素之力，绕着叶少阳一通狂攻。
<br/>　　叶少阳将三张暗金神符贴七星龙泉剑上，不断念咒作法，将各种法术串联起来，行云流水地进行着反击。
<br/>　　那些大难不死的邪物，都站在壕沟边上观战，默默地叶少阳助威。以他们的水平，所看到的只是一阵眼花缭乱的打斗，目不暇接，更不要说看清楚此中的变化和奥义。
<br/>　　钟馗却是看得频频点头，他当然是能看懂的，而且看得相当满意——不光是叶少阳的境界，上仙牌位固然牛逼，但相同境界的两个人，比的就是战斗的综合实力。
<br/>　　说得高大上一点，是对战斗的理解，和对法术的组合和搭配，再复杂一些，就是法术的选择，在什么时候该用什么样的法术。
<br/>　　这些说起来容易，却永远教不会，只能通过不断地战斗和总结，一点点提高，但即使这样，所能提高地也很有限，最重要的，其实是天赋。
<br/>　　钟馗就这么观看了一会，内心深处也是感慨不已：叶少阳，在这好几个方面，差不多都已经做到极致了。他没有短板，尤其在阅读战斗这方面，他简直就是个超级天才……
