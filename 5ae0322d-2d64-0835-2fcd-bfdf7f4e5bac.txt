　　大雷音寺！
<br/>　　遮天界的大雷音寺。
<br/>　　洪荒之中虽然有了西方释教，但真正的佛教，它却还没有成立，但此处却早已有了佛教。而且洪荒之中未来的佛教佛祖释迦牟尼在此方世界也有，但此方世界的释迦牟尼却是佛教的魔，他乃阿弥陀佛大帝的帝尸有了灵智，修行而成。他还在星空彼岸建立了大雷音寺。
<br/>　　此处正是释迦牟尼用来镇压十八位大妖魔的大雷音寺。
<br/>　　一座古庙显现，寂静无声，规模很小，根本谈不上恢宏。仅仅一件古殿，内立一尊石佛，蒙着厚厚的尘埃，旁边一战青铜古灯摇曳出点点光华。
<br/>　　“法器？还是破碎的。”
<br/>　　道君皱了皱眉，这些佛家法器很低级，而且还是破损的，他真有些看不上。
<br/>　　“罢了，还是把它们留在此处，算作给后人的机缘吧。”
<br/>　　说完，道君却不理这间古庙，而是来到古庙前，在这里，才有他需要的东西。
<br/>　　古庙前，一株菩提古树苍劲如虬龙，通体古朴，只有立地两米处零星点缀这五六篇绿叶，每片都晶莹剔透，绿光烁烁，犹如翡翠神玉。
<br/>　　古庙与菩提古树相依相呈，古意盎然，让人似感觉到时光流转，岁月的变迁，带给人以无尽的宁静与苍古。
<br/>　　混沌道君走到菩提古树前，静静地打量着它，心中却在思索。
<br/>　　菩提古树，又名智慧树，悟道树，向来都是智慧的象征。
<br/>　　这菩提古树，乃是佛教圣树，佛陀便是在菩提古树下成道，见菩提树如见佛。
<br/>　　若是在洪荒，菩提古树可是十大先天灵根之一，亦是最为顶级的先天灵根。准提圣人便是菩提古树所化，由此可见，菩提古树的厉害之处。
<br/>　　但在这遮天界中，菩提古树虽然并没有洪荒中那株厉害，但却也是十分难得了。按此界的说法，菩体古树是一种仙药，当年阿弥陀佛大帝身边的一株仙药。可惜此时却只剩下了点滴生机，随时可能死亡。
<br/>　　道君亦是知道，此界中的仙药，乃是仙界中的仙陨落而化的，它的本质是仙，故而能够不断涅槃，从而达到某种意义上的长生不死。
<br/>　　对于此界的仙药，道君表示还是非常感兴趣的。虽然他已经长生不死了，而且他也有许多比这仙药还要高级的混沌仙药。
<br/>　　研究这世界的仙药，就相当于研究此界中的仙道法则，毕竟仙药本身就是一定意义上的仙。
<br/>　　眼前这株干枯的古树苍劲挺拔，六七人也合抱不过来，主干中空，唯有一条垂落道离地面两三米出的枯枝上挂着六面绿叶，晶莹闪烁，若翡翠般漂亮。
<br/>　　“快要枯死的菩提树？这样不好，本道君可没有那么多时间等它再次涅槃了。”道君心念一动，一指点出。
<br/>　　“生命源液。”
<br/>　　一滴晶莹翠绿，如同一颗翠绿色的宝石的液体水珠，飞落在那干枯的菩提古树上。一道极为浓郁到极致的生命之气从那绿水珠上传出。
<br/>　　刹那间，菩提古树之上，枯木逢春，有绿意闪烁，渐渐地，绿意变得浓厚，渐渐充斥了整棵古树。
<br/>　　享受着生命的气息，一片片菩提叶生了出来。
<br/>　　只是眨眼间，它便恢复了上古的荣光，生命灼灼。
<br/>　　生命源液是木灵世界生命树一族的特产，为生命树数万年凝聚出来的，蕴含着浓郁的生命力，不仅能够肉白骨，生死人，而且还天生蕴含着浓郁的生命法则，通过它领悟生命法则，能够省去许多麻烦。
<br/>　　不要说这古树还没有彻底死亡，就算是死了，只要时间不长，生命源液一样能够使它复活。就这一滴生命源液，就比此界的仙药还要神奇。
<br/>　　一滴生命源液就能延寿数万年！
<br/>　　虽然还有许多其他简单的方法让眼前这株菩提古树复苏，但道君却采用了最浪费资源的方法。生命源液不仅能使它恢复上古荣光，更能进一步增强他的底蕴，如今的菩提古树，可比他以前巅峰时候还有强。
<br/>　　“如菩提古树这般好东西，可不能竭泽而渔，得循环利用才对。”道君大手一挥，便将古树收起，心里却想着原著中叶凡的做法，悠悠道。
<br/>　　“从今后，什么菩提子、菩提叶、菩提茶、菩提丹、智慧丹、悟道丹，一切都不是问题。”他相信，菩提古树到了他手中，他能利用它做出比菩提古树本身更有价值的东西。
<br/>　　“该走了，是该到此界地球去看看了。”道君仰头看向远处，眼光穿过无尽虚空，落在一颗水蓝色星球之上，悠声道。
<br/>　　一步踏出，无数星球在他身后倒退，星辰倒旋，星空成道，再一步迈出，混沌道君便降临到了地球之上。
<br/>　　“嗯？这个世界的地球？有些古怪。”
<br/>　　道君刚一降落，此界地球就给了他一个‘惊喜’。他能够感受到这个地方对道则的强烈压制，这个星球上，被人布满了大帝阵纹，若是寻常修士到此，一身修为很快便会被压制到最低，但道君显然不会，他是寻常修士吗就连天道都不能封印他，更何况这点大帝阵纹了。
<br/>　　此界地球上的天地灵气稀薄到了极致，很难出现强大的修士。道君知道，远古时候的地球可不是这样的，它只是后来被人封印了。
<br/>　　道君抬头望去，他看到一座座高耸入云，直上九霄的山川大岳，直插云霄，不只有多少万丈。待再次看去，它却消已失不见了。
<br/>　　“那是昆仑山？”
<br/>　　道君若有所思道，他的本尊就是住在昆仑山，此界的昆仑山虽然不如洪荒的昆仑山，但也是极为神奇的几座名山了。
<br/>　　他可是知道，正是那昆仑山，将地球所有的灵气吸去，神话时代，帝尊寻遍天下名山，找来九十九条大龙脉，布置在昆仑山，为了孕育造化仙器绿鼎。之后，绿鼎破碎，羽化仙朝再次找到此地，送来破碎的绿鼎，希望借此地之势，重新孕育出绿鼎。可惜，他们失败了。
<br/>　　他们千算万算，算不到天下出了一个任性的狠人大帝，成仙鼎又如何，她一个不高兴，一巴掌就将其拍碎。
<br/>　　昆仑山虽然神奇，但它就在那里，又跑不了。道君此刻在意的是，现如今是什么时间，剧情已经开始了，还是没有开始？只有知道这些，他才能好好谋划一番。
<br/>　　“叶梵。”
<br/>　　道君心念一动，念出这样一个名字。
<br/>　　顿时，冥冥之中，好像有一种因果，连接到远处某点。道君脑海之中，顿时出现了一个画面。
<br/>　　一个二十来岁的年轻小伙，正坐在一个摇椅之上，手捧一本书，在仔细看着。
<br/>　　“嗯？还有一个月，剧情就开始了。很好，想必九龙拉棺也不远了。这样么——”道君幽幽道。
<br/>　　“主神计划，可以开始筹划了。”
