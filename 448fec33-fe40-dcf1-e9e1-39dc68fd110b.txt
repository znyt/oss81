　　大雄宝殿比外面保存的还差，门窗都坏了，里面的地板也都碎裂不堪，香案什么的也毁了，只有几尊佛像还在，横着看过去，是佛门三大教主：药师佛、释迦牟尼佛、阿弥陀佛。
<br/>　　这是佛门横三世的三位教主，也是最终极的三位大佬，药师佛掌管东方净琉璃世界，释迦牟尼佛掌管婆娑世界，阿弥陀佛掌管西天极乐世界。
<br/>　　时间还没到。
<br/>　　叶少阳挨个看过去，发现这几尊佛像外层的金漆都不见了，从痕迹看，是被人粗暴地剥去的，估计是佛像不值钱，也没地方卖，所以被盗贼留下了。
<br/>　　被剥去金漆的佛像，露出木头的材质，表情看上去仍然丰润圆满，炯炯有神。叶少阳忍不住赞叹，然后道风告诉他，这天台山建造木雕佛像是出了名的，有一套只有当地人掌握的技术，在周边县城，有很多靠制造木雕佛像生活的匠人，鼎盛的时候，全国的木雕佛像，有六成都是天台地区做的。
<br/>　　“你怎么知道这些？”叶少阳听完，好奇问道。
<br/>　　“我曾游历天下，你以为光是旅游不成？”道风一句话把他噎住了。
<br/>　　两人绕到中间那座释迦牟尼佛的后面，叶少阳招呼道风一起躲在破坏的帷幔后面，静静地等待着。
<br/>　　“对了道风，我记得你早年是道佛双修，可后面你学的那些，三花聚顶五朝元气这些，都是道家法术啊，我好像从来没看你用过佛门法术？”
<br/>　　“我修过佛法，但不是法术。”
<br/>　　停了一下，他解释道：“所谓道佛双绝、万法皆通，不是说什么法术我都会，我修了佛法，用佛法的方式来理解道术，能推演出世间一切法术，只要让我看一遍，我就能懂其中的奥义。”
<br/>　　用佛法来理解道术……叶少阳还是第一次听他这么说，惊的嘴巴班天合不拢。任何法术，都有相应的心法来指导，所以才有先“修心不修术，到头一场空”的说法，修炼道术，自然要以道家心法为纲，就像学汉字要先学汉语拼音一样，你拿英语的语法来学汉语，只会两边都学不好。
<br/>　　语言学不好没啥，但如果修行方式不对，强行修炼法术，结果就是走火入魔，陷入邪魔外道（注：当今有不少练气功的爱好者，都声称自己发功之后能看到鬼怪，其实大部分都是心内幻想出来的，为走火入魔的结果）。
<br/>　　“所以，我入了魔道啊，大家都这么说。”道风淡淡一笑。
<br/>　　叶少阳当然知道他是自嘲，紧跟着问道：“可是你修成了啊，这中间有啥诀窍没有？”
<br/>　　道风瞥了他一眼道：“世人若学我，如同下地狱。”
<br/>　　叶少阳撇了撇嘴，暗骂了一声装逼，又等了一会，外面响起脚步声，等的人来了！
<br/>　　叶少阳想起什么，画了一张隐气符，递给道风，“你不是人，万一被他察觉到气息就不好了。”
<br/>　　道风看了一眼淡淡说道：“不需要。还有——”
<br/>　　用力在他后脑勺上拍了一下，“你才不是人！”
<br/>　　“我去，我说的是实话，还有，我发现你现在越来越喜欢动手动脚了！”叶少阳大声抗议。
<br/>　　说话间，脚步声到了门外，两人赶紧闭嘴，从帷幔后面悄悄看过去。
<br/>　　一个瘦高个的僧人走了进来，身穿常服袈裟，年纪五十岁上下，眉毛很长，看上去很有威仪感，神色凝重，像是满怀心事。
<br/>　　进入大殿之后，他也没多检查，匆匆来到药师佛的背面，在佛像后背上拍了几下，又鼓捣一会，然后竟然从佛像上撕起了一块布，双手抓着，绕着佛像走起来，一点点把布撕下去。
<br/>　　佛像的肚子越来越瘦。
<br/>　　叶少阳虽然穿越过好多次，但也是第一次藏身在这里，眼前这情景也是第一次看到，感到十分惊奇。
<br/>　　等佛像肚子上的布瘦到一定程度，这时候浓眉和尚匆匆去了隔壁，过了一会再回来，手里提着一只木桶，放在药师佛身后，桶里是明晃晃的液体。
<br/>　　浓眉和尚拿出一个包袱，取出一个木壶，打开后将粉末倒了进去，然后又放了几块黑乎乎的药膏一样的东西进去，然后静静地坐在一边等着。
<br/>　　叶少阳和道风不知道他要干什么，也只有一动不动站着，等着下一步。
<br/>　　刺鼻的味道传来，有点像是松香，不过混合了不知道什么东西，这时候，浓眉和尚从怀里取出一支连着剑鞘的剑，贴在佛像后背上被自己撕出来的凹槽上，固定好位置，然后再将撕下来的布缠上去。
<br/>　　这就是太阿剑啊……
<br/>　　叶少阳立刻有点心跳加速。
<br/>　　浓眉和尚又拿出一支毛笔，蘸了桶里的液体，往贴上去的布上刷，应该是起到胶水的作用，缠一层刷一层，后面仔细检查，完好之后，将多余的布料撕掉，再往桶里添了一些粉末，搅拌之后，再次往上刷，这次液体成了金色的，跟药师佛本身的装金颜色一样。
<br/>　　等刷好之后，浓眉和尚又把减下来的多余布料一块块贴补上去，平整那把剑放进去引起的局部的凹凸不平。
<br/>　　这个过程有点费事，弄了足足半个小时。完事之后，浓眉和尚又重新搅拌了一桶“涂料”，不断往上刷，叶少阳猜测这是在充实布料的缝隙，抹平痕迹。
<br/>　　做完这一切，浓眉和尚似乎很累，将木桶拎到隔壁去，回来的时候，手上多了一个火把，对着自己刚处理过的地方烘烤起来，之后将火把熄灭拿走，自己再回来检查了一边，似乎很满意，然后叹了口气，望着藏剑的地方叹道：
<br/>　　“小僧能做的也只有这么多了，将来究竟是谁能得到你，全看机缘了。愿佛祖保佑……”
<br/>　　他说完，又静静地站了一会，这时候窗外吹来了一阵腥风。
<br/>　　浓眉和尚怔了一下，翻窗逃走。
<br/>　　叶少阳和道风等了一下，也紧跟着翻过窗户，然后爬上了房顶，朝远处看过去。
<br/>　　大雄宝殿的对面，就是后山门了，能看到那浓眉和尚出了院门，一直往前飞奔。
