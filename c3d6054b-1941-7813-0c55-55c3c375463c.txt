　　在场的无数众人，都不是第一次见到武风云出手了。
<br/>　　人榜第十一的位置，可不是空手得来的，是要经历无数次厮杀才可以得到的地位。
<br/>　　武风云能够稳居这个位置，他的实力是无需置疑的。
<br/>　　但是曾经武风云取得这人榜第十一位的时候，他的武神真身，才只是有着四条手臂。
<br/>　　而现在，武风云再一次换出来的武神真身，却是有着六条手臂，这也就意味着。
<br/>　　现在武风云的实力，比起他曾经取得这人榜第十一位的时候，还要强大许多。
<br/>　　现在武风云的实力，可以说是真正踏入到了人榜前十的地步。
<br/>　　这样的人，在外门中，绝对都是大人物，可以随时晋升内门的存在，放眼外门，近乎无敌。
<br/>　　就连楚风眠的眼神，在看到这武神真身的一刻，也都是微微有些惊讶。
<br/>　　这武风云，的确是有着几分门道，凭借这武神真身，他的实力只怕是跟拓跋鼎都相差不远了，的确是有着内门弟子的实力。
<br/>　　看来这北邙学院，的确是天才众多，卧虎藏龙，武风云不过是一个外门弟子，人榜前十都没有踏入的人，居然都有着如此实力。
<br/>　　那人榜前十的人，又要有多么厉害，更何况上面还有着地榜，天榜的弟子。
<br/>　　“小子，现在你后悔也晚了，敢跟我们风云会为敌，今日我就将你镇压，在你的心神中下心魔，让你将来每时每刻都会记住今天的噩梦！”
<br/>　　武风云看着楚风眠的表情，还以为楚风眠是怕了，大声的笑了起来。
<br/>　　武风云的六条手臂，突然握拳，猛然打了出去，这一次，便是像是如同刚刚那一样的镇邪白虎，居然是足足有着六头之多。
<br/>　　这六头镇邪白虎一齐出手，却是仿佛来通灵，懂得配合，居然是直接布置出杀阵来，围困楚风眠凝聚出的那条金龙。
<br/>　　“这条金龙，耗费了你不少的力量凝聚而出的吧，我就先将这金龙分食了！让这金龙成为我的力量！”
<br/>　　武风云大笑一声道。
<br/>　　“给我吞了他！”
<br/>　　一边说着，那六头镇邪白虎，便是同时扑了上去，开始撕咬这楚风眠凝聚而出的那一条金龙来。
<br/>　　九域中，在远古流传之中，便是就有着龙争虎斗一词。
<br/>　　如今天空之上的景象，却是完美的权势了这个词。
<br/>　　武风云凝聚出的六头镇邪白虎，跟楚风眠凝聚出的金色真龙，在空中，直接缠斗起来。
<br/>　　这画面，令人叹为观止。
<br/>　　“好可怕的二人，同为外门弟子，这二人的实力，却是要远超与我，随手一招，都足以将我击败。”
<br/>　　一尊外门弟子，喃喃自语道。
<br/>　　他抬起头，看着空中的景象，近乎的面露呆滞，这简直是如同神仙打架。
<br/>　　他要是参与进去的话，只怕是随意一头镇邪白虎，都足以是将他彻底击败。
<br/>　　更何况是楚风眠凝聚而出的，更加可怕的金色真龙。
<br/>　　“武风云是外门中的风云人物，人榜排位接近前十的佼佼者，有着如此实力，倒是并不稀奇。”
<br/>　　“倒是这楚风眠，只是这一届的新人，一个新人，居然拥有如此实力，足以挑战人榜接近前十的人物，这样实力，还真是前所未见。”
<br/>　　“拥有这样的实力，一般来说早就被长老挑走了吧，不知道此人为何会来外门。”
<br/>　　“别管那么多了，来了这样一个人物，搅动风雨，倒是一件好事，好久没有如此热闹了。”
<br/>　　一尊尊武者，都极为仔细的观察这。
<br/>　　这其中交手的一举一动，都是有着无上玄通，以他们的实力，要是能够参悟其中几分，都会得到极大的好处。
<br/>　　更何况，他们也都想要看看，如今这新人王楚风眠，到底能达到什么地步。
<br/>　　要是楚风眠能击败了武风云，那外门可就热闹了，北邙学院的外门足足是有着百年时间，没有出现过了，一个新人，居然足以击败人榜前十的弟子。
<br/>　　武风云虽然是人榜第十一位，但是真正实力已经是人榜前十了，如今众人都在这里看热闹，看起来这两人，到底是谁强谁弱。
<br/>　　北邙学院，位于独特的一方空间之中，这其中的空间，被一些强者，以无上神通给凝聚，变的坚韧无比。
<br/>　　楚风眠跟武风云，虽然是在凌空交手，但是并不会打破空间，力量也不会四散出去。
<br/>　　要不然的话，这北邙学院，只怕是早就被无数弟子交手，给打的破烂了。

<br/>　　武风云看着楚风眠，冷声笑道。
<br/>　　“一个新人，敢来挑衅我们风云会，还真是前所未遇的，不过倒好，今日我就来一个杀鸡儆猴，看看以后谁还敢挑衅我们风云会的威严！”
<br/>　　武风云的声音中，蕴含有着极为自信的神色，仿佛是已经胜券在握。
<br/>　　“瞬息之间灭杀我？你要是有这个本事，就来吧。”
<br/>　　楚风眠瞥了一眼武风云，倒是脸色平静到了极点。

<br/>　　武风云眉头一皱，脸色变得通红，显然是动了真怒。

<br/>　　“他的神罗伏虎拳，已经达到小成了，凝聚出四十九头镇邪白虎，据说这神罗伏虎拳若是大成，便是足以凝聚出整整九九八十一头镇邪白虎来，那才是真正的可怕，生死境强者都要退避。”
