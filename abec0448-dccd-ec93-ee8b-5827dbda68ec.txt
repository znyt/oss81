　　比武台上的战斗，仍旧在继续。
<br/>　　萧逸看了眼云渊长老和讳执事。
<br/>　　他终于知晓为何四月前云渊长老会不远万里，特地前往金光险地给他送剑帖了。
<br/>　　对他来说，此次剑域盛事，是一次难得的机会。
<br/>　　念及此，萧逸看了两人一眼，会心一笑。
<br/>　　“笑什么？”云渊长老显然注意到萧逸的笑容，问道。
<br/>　　“没什么。”萧逸笑笑，看向比武台，道，“两脉之间的比拼，应该要不了半天就会结束了。”
<br/>　　“嗯。”云渊长老点点头，“两宗武者，该比拼的都比拼了。”
<br/>　　“胜负，就现在看来，应当在伯仲之间。”
<br/>　　“真要分出两宗高下，也是这半天内的事了。”
<br/>　　两脉间的比拼，已持续两天有余。
<br/>　　该上场的剑者，也都上场了。
<br/>　　不过，这比拼，显然也有着规则和一轮轮之分。
<br/>　　现今，是最后一轮了。
<br/>　　无论是气宗弟子，还是器宗弟子，大多上场过数次，交锋过数次。
<br/>　　不过，就萧逸看来，两宗高低，若真说谁强，却不好说。
<br/>　　或者说，气剑力与器剑力，到底谁厉害些，同样也无定性。
<br/>　　两种剑力，层次本就一样，只是效果不同罢了。
<br/>　　两宗武者的上台比拼，与其说是气剑力与器剑力的比拼；倒不如说仅仅是武者间对这些剑力的应用本事。
<br/>　　两种剑力，虽效果不同，但却是同一层次的力量。
<br/>　　那么，比的，不过是个人的本事和手段。
<br/>　　“这种无休止的争论，怕是两宗之间再斗个一万年都不会有结果。”萧逸笑笑。
<br/>　　“确实。”云渊长老点了点头。
<br/>　　“争论气剑力与器剑力那个更强。”
<br/>　　“与争论魂师和阵法师哪个更强，有何区别，不过是职业不同，所修方向不同罢了。”
<br/>　　“真要论高低的，是个人本事。”
<br/>　　“不过，这也是剑域剑者有趣的地方，那是他们的信仰。”
<br/>　　“一个纯粹的剑修，若是连信仰都没了，那会是很可悲的事。”
<br/>　　萧逸点了点头，“就如同一个武者，不相信这武道之路有着尽头，不相信自己能走到武道尽头，少了那份斗志，将会很可悲。”
<br/>　　身旁讳执事闻言，霎时眼前一亮。
<br/>　　云渊长老满意一笑，点了点头。
<br/>　　或许，萧逸有些明白两宗争斗无数岁月的原因了。
<br/>　　剑修，最是纯粹，却也最容易产生固执。
<br/>　　当然，这份固执，若放到修炼之上，也并非坏事。
<br/>　　一个武者，就算不是剑修，若连这一份对自己所修一道的固执都没有，那岂非可笑？
<br/>　　数个时辰后。
<br/>　　比武台上，两道身影，同时跃上。
<br/>　　“这是最后一场了。”云渊长老沉声道。
<br/>　　“之前的胜负，各执一半。”
<br/>　　“两宗此次比拼的胜负，便看这最后一场了。”
<br/>　　萧逸看了台上的两人，皱了皱眉。
<br/>　　器宗方面，上台的，竟然是丁秋月这丫头。
<br/>　　而气宗方面，上台的，则是一个约莫25岁左右的年轻男子。
<br/>　　男子一身气势，单凭气息就远远超过了丁秋月。
<br/>　　“圣皇境八重。”萧逸眉头一皱。
<br/>　　“这家伙，已经第二次觉醒武魂了吧。”
<br/>　　“丁秋月不过是个圣王境六重，如何能敌？”
<br/>　　而且，丁秋月不过16、7岁。
<br/>　　两人间的年龄差距，差了差不多10年。
<br/>　　“没办法。”云渊长老摇了摇头，道，“这二人，便是如今两宗的接班人。”
<br/>　　“气宗的那个小子，名唤桌万宗，可不是个等闲之辈。”
<br/>　　“听闻此次盛事之后，气宗宗主之位便会传于他。”
<br/>　　“而器宗的那个小丫头，年纪虽少，却是器宗老宗主金口断言万年难遇的绝世妖孽。”
<br/>　　“听闻6岁之时，早便被定为器宗老宗主日后的接班人。”
<br/>　　萧逸点了点头。
<br/>　　他算是明白过来，为何丁家庄之内，会有‘通明器谱’这等品阶极高的剑力法诀了。
<br/>　　想必就是器宗赠予的。
<br/>　　而且，丁秋月被寒光府所掳，当即便是器宗六剑侍亲自出手相救。
<br/>　　此时，比武台上的比拼还未打响。
<br/>　　裁判在对二人说着些循例的规则。
<br/>　　萧逸皱了皱眉。
<br/>　　一旁，讳执事却语气淡漠，“桌万宗那小子，打算下杀手。”
<br/>　　“哦？”萧逸愣了愣，随即轻笑道，“这还没开始开始打呢。”
<br/>　　“再者，众目睽睽下，桌万宗应该…”
<br/>　　“笨蛋。”云渊长老轻笑道，“你这讳前辈，杀过的人比你吃过的饭还多。”
<br/>　　“桌万宗那小子有没有下杀手的心思，他一眼就能看穿。”
<br/>　　萧逸脸色一变。
<br/>　　他想起来，讳执事可是曾经中域之内人人闻之色变的中域杀神。
<br/>　　而且，还是曾经的天冥府总教导。
<br/>　　这等人物，确实一眼便能断人心思。
<br/>　　“果然。”萧逸忽然收敛了心思，看向比武台。
<br/>　　此时，比武台上的交手，已然打响。
<br/>　　而也几乎是这一瞬间，萧逸看到桌万宗出手的第一个动作，便已确定了讳执事之言没有错。
<br/>　　“果然？你也看出来了？”讳执事惊疑地看了眼萧逸，“看来你杀过的人，并不比我少多少。”
<br/>　　萧逸并未回答。
<br/>　　因为，比武台上的状况，忽然变了。
<br/>　　丁秋月忽然被震得节节败退。
<br/>　　而在她身后，数道破碎的剑刃，不知何时起便伫立于地面。
<br/>　　照这般情况下去，丁秋月将会自己在后退跌倒的过程中，被这些破碎的剑刃所穿身而亡。
<br/>　　“好个桌万宗，想来个意外杀人吗？”云渊长老这时也看出了端倪。
<br/>　　嗖…
<br/>　　恰在此时，萧逸一个闪身，跃上了比武台。
<br/>　　“喂，小子。”云渊长老一惊。
<br/>　　萧逸，已然来到了比武台上，一把扶住了丁秋月。
<br/>　　“大哥哥。”丁秋月见了忽然现身的萧逸，脸上尽是惊喜。
<br/>　　但还未来得及多说什么，已一口腥血喷出。
<br/>　　萧逸看了一眼，心头暗道一声果然。
<br/>　　丁秋月此时被封了一身元力，刚才倒退之时，若他不及时现身，丁秋月绝对会就此倒地，也躲不开地面的锋利剑刃。
<br/>　　“哪来的小子？胆敢扰乱两宗大比？”一声苍老大喝，忽然响起。
<br/>　　比武台外，云渊长老皱了皱眉，“我去拉这小子下来。”
<br/>　　“不必。”讳执事笑笑，“年轻人嘛，让他出出风头也好。”
<br/>　　锵…
<br/>　　比武台上，桌万宗竟未收剑，一剑袭来。
<br/>　　萧逸指尖一扬，一道剑气挥出。
<br/>　　嘭…一声爆响。
<br/>　　桌万宗仗剑挡之，竟是被剑气震得节节后退。
