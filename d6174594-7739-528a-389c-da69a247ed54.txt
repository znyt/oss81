　　巫人王已经等不及了，要尽快拿下苏莫，以免迟则生变。
<br/>　　而且，第二关马上就要开始了，必须尽快行动起来。
<br/>　　所以，苏莫刚一进入圣门之内，他便立刻出手了。
<br/>　　大手狠狠一探，一只硕大的玄力手掌快如闪电一般，向苏莫当头抓去。
<br/>　　风声呼啸，玄力手掌碾碎虚空，瞬息而至，强大的威势铺天盖地。
<br/>　　苏莫见此眸中杀机爆闪，刚准备还击，又立刻压下了这个想法。
<br/>　　刚才一大帮人冲击圣门之内，人都没有散开，旁边任云都等不少神武学府的弟子都在，这巫人王一击很强，不管不顾，他若强力击溃，很可能要伤到很多人。
<br/>　　唰！
<br/>　　苏莫立刻爆发出最快的速度，身形向侧方一闪，如同一道电光一般，急速闪避了过去。
<br/>　　轰！
<br/>　　硕大的玄力手掌没有击中苏莫，直接击在了后方的圣门之上，顿时发出了一声惊天的爆响。
<br/>　　圣门没有任何反应，玄力手掌则立刻崩碎开来，气浪汹涌。
<br/>　　其他人见此，立刻急速退开，免得遭到波及。
<br/>　　众人心惊，看来巫人王和苏莫的仇怨不小啊，这才刚进入圣门，就迫不及待的动手了。
<br/>　　而巨摩等一些后来的人，却是不知道巫族和苏莫有仇，见此顿时好奇了起来。
<br/>　　巫族主脉天才巫殇，见此暗暗摇头，巫人王真是太废了，修为和他相同，高达武帝境九重境界，一招之下居然还能让苏莫逃了！
<br/>　　“苏莫，你只敢当缩头乌龟吗？”巫人王见苏莫躲避，顿时一声怒喝。
<br/>　　“简直不知死活，放马过来吧！”苏莫闪避到远处停了下来，冷声说道，既然对方找死，他也无需顾忌什么。
<br/>　　这次真圣界内的考核，巫族必然会置他于死地，无论他怎么样，大祭司都不可能放过他。
<br/>　　所以，与其让巫族之人不停的挑衅，还不如大开杀戒，将他们全部杀光。
<br/>　　“哈哈！那就接招吧！”巫人王见此顿时狂笑一声，就准备出手了。
<br/>　　而就在此刻，第二关要开始了，只见真圣碑上发出一阵朦胧的青光，而后在圣碑周围万里之内，形成了一个个方圆千丈的平台。
<br/>　　这些平台仿若是青色的玉盘，数量足有八百个之多，所有玉盘均是分布在圣杯的周围，形成了一个巨大的圆形。
<br/>　　而后，青光笼罩在众人的身上，众人的身体不受控制的向一个个玉盘飘飞而去。
<br/>　　“苏莫，下一关就是你的死期！”巫人王见此，顿时厉喝了一声，既然第二关要开始了，他也只能放弃擒拿苏莫了。
<br/>　　到第三关之时，苏莫将逃无可逃。
<br/>　　外界。
<br/>　　各大势力带队的强者，身前都有虚幻的影像播放，真是真圣界内的场景。
<br/>　　这并不是多么玄妙的能力，只是其内的弟子，用特制的玉符，将其内的场景传递出来了而已。
<br/>　　大祭司看着眼前的影像，巫人王向苏莫出手的画面，他看的清清楚楚。
<br/>　　见到巫人王一击没有拿下苏莫，他不禁眉头微皱，但很快便施展了开来。
<br/>　　第三关可以自由挑战，到时候苏莫便没有活路了，就算苏莫实力再强，也不可能是武帝境九重的巫人王的对手。
<br/>　　就算巫人王真的杀不了苏莫，还有巫殇呢！
<br/>　　他来之前，已经叮嘱了巫殇，绝对不能让苏莫活着走出真圣界。
<br/>　　让族中子弟杀死苏莫，是在好不过的事情了，他就不用出手了。
<br/>　　不然，他堂堂巫族支脉大祭司，武圣境中期强者，出手抹杀武帝境的人族后辈，名声也不好听。
<br/>　　而且，到时候他失信于火元圣师和杜府主，难免让两人心生怨恨。
<br/>　　虽然他根本无惧火元圣师和杜府主，但那两人也不是一般人物，暗中给巫族报复，也是麻烦。
<br/>　　此时，吴晋导师、唐婉导师、木璃等导师，望着身前的影像，均是脸色难看。
<br/>　　巫族要将苏莫置于死地，怕是苏莫根本走不出真圣界了。
<br/>　　可是，他们没有丝毫的办法，只能希望苏莫足够强大，能保住一条命。
<br/>　　“有趣！”
<br/>　　逍遥剑圣见到影像中的情景，微微一笑，他自然也听过苏莫的名头，真圣榜第一。
<br/>　　只是他没想到，巫族子弟居然如此光明正大的要杀苏莫。
<br/>　　……
<br/>　　嗖嗖嗖！！
<br/>　　一名名进入圣门的天才，四散而开，身形飘飞到了一座座青色的平台上。
<br/>　　这是所有人都无法控制的事情，自己会被分散在哪个平台，完全看运气。
<br/>　　这一刻，很多天才都暗暗祈祷，自己所在的平台上，不要有太过强大的人。
<br/>　　因为这一关，所有人都要接受圣雨的攻击，每一座平台上只能有一人过关，谁坚持的越久，谁能坚持到最后，谁就能过关。
<br/>　　若是和巨摩，或者是姜神夜、飞长云等强大人物分在了一个平台上，天赋不如别人，实力不如别人，自然不可能比别人坚持的久。
<br/>　　嗖嗖嗖！！
<br/>　　人影重重，四散而开，仅仅是三个呼吸的时间，二十余万人，便分别落在了八百个平台上，每个平台二百五、六十人，较为均衡。
<br/>　　“啊！完了！”
<br/>　　“天啊！我怎么这么倒霉！”
<br/>　　“真是晦气！”
<br/>　　下一刻，惊呼声、抱怨声四起，很多人欲哭无泪，因为他们所在的平台，都有真圣榜上的强大人物。
<br/>　　真圣榜上也就二百人，平台有八百个之多，和真圣般上的天才分在一个平台，可见运气并不好。
<br/>　　比如任云都就非常郁闷，因为他所在的平台上，也有真圣榜上的天才，而且还是真圣榜排名第二的巨摩，这运气简直背到了极点。
<br/>　　他的实力不比真圣榜排名靠后的天才弱，甚至比之尤天韩都不弱多少，正常情况下进入第三关是没有问题的，现在则是直接完蛋了。
<br/>　　苏莫身形落在一座青色平台上，立刻打量了周围和他同平台之人，二百五六、十人之中，有大半人族天才，还有少量的巫族、巨灵族、飞灵族等异族的天才，均是修为强大之辈，他的修为算是最低的一人了。
<br/>　　不过，他所在的平台上，也没有什么值得注意的高手，根本无需担心。
<br/>　　当然，那些真圣榜上的天才，他大多都不认识，就算有真圣榜上的天才和他同平台，只要不是排名前几位那几人，对他没有丝毫的威胁。
<br/>　　不过，苏莫不在乎别人，却是有不少人将他当成了威胁！
