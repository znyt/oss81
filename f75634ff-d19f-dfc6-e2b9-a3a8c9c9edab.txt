　　佛寺的大门，依然封闭，因为还没有到真正开启的时候。
<br/>　　但是随着天空之上的七彩霞光，越发的绚烂，楚风眠也清楚等到这七彩霞光达到极致的一刻，就是这佛寺开启的一刻。
<br/>　　在场的无数强者，都在等待着佛寺的开启，没有人愿意在这个时候发生任何的冲突。
<br/>　　在这时发生冲突，便是等于放弃对佛寺之中宝藏的争夺，纵然是许多之间有着血海深仇的古帝，现在都只能是心平气和的等待着。
<br/>　　楚风眠跟燕皇，正在寻找青梦。
<br/>　　突然之间，楚风眠的脑海之中多出了一道声音，有人突然传音而来。
<br/>　　这是一种传音秘术，楚风眠都未曾见到过的，听其中的声音一尊十分年轻的女子，而且这声音，楚风眠听出了几分熟悉。
<br/>　　声音只有一句话。
<br/>　　“速来相见。”
<br/>　　旋即楚风眠的脑海之中，便是显现出了百里之外的一个地方。
<br/>　　“恩？”
<br/>　　楚风眠眼神之中露出了几分诧异，似乎是没有想到这突然发生的事，他想了一下，随即开口道。
<br/>　　“燕皇，你在这里等我一会，我先去一个地方。”
<br/>　　“好，小心一点。”
<br/>　　燕皇点了点头，也不多问。
<br/>　　楚风眠立刻化为了一道遁光，悄然之间隐藏了身形，来到了百里之外。
<br/>　　一眼望去，如今这百里之外的地点，只是一处荒漠，但是随着楚风眠靠近过去，突然之间天旋地转。
<br/>　　幻境！
<br/>　　有人在这荒漠之中，布置出了这样一道幻境，外人无法勘察其中，只有得到允许的人，才有资格进入其中。
<br/>　　踏入这幻境之中，一眼望去便是一座碧蓝的海洋，无边无际，没有尽头，在这海洋的中心，有着一座小岛。
<br/>　　在这岛上，有着一座石亭，石亭之中，一尊少女的身影，立于其中。
<br/>　　楚风眠打量了周围一眼，身形一闪，一步穿越海洋，踏入到了石亭之中。
<br/>　　“不知海公主为何找我？”
<br/>　　楚风眠来到了石亭之中，率先开口道。
<br/>　　这突然来找楚风眠的人，不是其他人，而是之前在天都大陆的探索商队之中，跟楚风眠和洛汐一同同行，后来被楚风眠所救的海公主。
<br/>　　那个时候，三大势力还没有彻底跟楚风眠撕破脸皮，楚风眠也不愿意滥杀无辜。
<br/>　　可现在，三大势力已经是将楚风眠视为眼中钉肉中刺，恨不得当场将楚风眠斩杀，这个时候海公主居然是来找他，的确是超出了楚风眠的预料。
<br/>　　“你快离开九域，三大势力的人，已经要下手杀你。”
<br/>　　海公主看见楚风眠到来，直接开口道，开门见山的说道。
<br/>　　“现在三大势力之间，已经是下了格杀令，圣龙秘境的圣龙宗所有弟子，都列在了必杀名单之中，你现在走还来得及。”
<br/>　　“这件事，我知道。”
<br/>　　楚风眠听到海公主的话一愣，旋即开口道。
<br/>　　三大势力想要杀他的事，楚风眠当然知道，他已经是遭到了无数次的暗杀。
<br/>　　只是楚风眠没有想到，海公主居然是为了告诉楚风眠这件事而来。
<br/>　　“那你可知道，三大势力已经是请出了半步天人境的强者，要来杀你？”
<br/>　　海公主听到楚风眠的话，又是开口道。
<br/>　　“半步天人境？”
<br/>　　楚风眠听到海公主的话，脸色猛然变化着。
<br/>　　九劫古帝，以现在楚风眠的实力，倒是不怕，可是半步天人境。
<br/>　　除非是楚风眠踏入了九劫古帝，不然半步天人境，远远不是楚风眠足以抗衡的。
<br/>　　大罗天星域的四大势力之中，传闻之中都有半步天人境的强者存在。
<br/>　　可这个级别的强者，千万年来，都未必会现身出世。
<br/>　　没想到，三大势力居然是要请出半步天人境的强者，要来杀楚风眠。
<br/>　　看来三大势力，也无法忍受楚风眠的实力在成长下去，打算彻底将楚风眠所扼杀。
<br/>　　“是谁？”
<br/>　　楚风眠不由的问道。
<br/>　　“是五行天的五行刀圣，你杀的五行刀祖，是他的弟子。”
<br/>　　海公主缓缓的开口道。
<br/>　　楚风眠杀了五行刀祖，碧螺老祖等人的事，她知道的一清二楚。
<br/>　　以三大势力的手段，轻易的足以查清楚查清楚这一切。
<br/>　　五行刀圣……
<br/>　　这尊人物，楚风眠在圣龙秘境的一些记载之中，曾经听闻过，据说是五行天最为古老的老祖之一。
<br/>　　足足是数万年的岁月，都没有在大罗天星域之中现身了。
<br/>　　在许多的记载之中，都有猜测这五行刀圣，已经是陨落了。
<br/>　　可现在看来，五行刀圣非但是没有陨落，反而是实力再进一步，达到了半杯天人境，冲击天人大道。
<br/>　　如果真的是五行刀圣来杀楚风眠，那可真的是无比棘手。
<br/>　　“你为何要告诉我这件事？”
<br/>　　楚风眠突然间看向海公主，不由的问道。
<br/>　　他跟海公主，应该是敌对关系才对，毕竟楚风眠可是也亲手杀了不少沧海都的弟子。
<br/>　　那葵海王，虽然不是楚风眠亲手所杀，是死在了燕皇的手中，可也算是楚风眠出了一份力。
<br/>　　“我蓝菱不会欠人恩情不报，你上次救我一命，这一次算是我还给你了。”
<br/>　　海公主说完，旋即身形闪烁，准备离开。
<br/>　　“多谢了。”
<br/>　　楚风眠点了点头。
<br/>　　五行刀圣即将出手对付楚风眠的消息，在三大势力之间只怕都是极大的隐秘，古帝都未必有资格知道。
<br/>　　这一次海公主前来告诉楚风眠，也是冒了不小的风险。
<br/>　　“你告诉你父亲一系的人，日后不与我为敌，等我踏灭了沧海都的时候，可以留他们一命。”
<br/>　　楚风眠突然出声道，旋即便是身形一闪，冲天而起，准备离开。
<br/>　　海公主的声音回头听到楚风眠的话，眉头一皱。
<br/>　　“自身难保，还说大话。”
<br/>　　当看到楚风眠准备离开的一刻，突然间又是一道声音传递了出来。
<br/>　　“小心佛寺开启之时。”
<br/>　　海公主在说完这一句话后，也是身形一闪，离开了这一片幻境之中，消失的无影无踪。
<br/>　　同时楚风眠身形闪烁，也从幻境之中离开，他的心中在琢磨这最后这海公主的这最后一句话。
