<br/>    陈明轻轻一挥手中的万道天竹，万剑出鞘，顿时就把这些野猪吓尿了，他们什么时候见过这般阵仗。
<br/>    
<br/>        万剑飞舞，大阵威势之下，所有的野猪都被陈明镇压在地，动弹不得，陈明手握万道天竹，悠哉悠哉来到高台上的野猪面前，问道:“听说，你要骑人？”
<br/>    
<br/>        高台上的野猪顿时就吓趴下了，“不敢不敢！”
<br/>    
<br/>        陈明伸手拍了拍身前的野猪，别说，还洗的挺干净，“转一圈我看看。”

<br/>        野猪首领只好照做，它还以为陈明觉得它不错，准备让它当坐骑，在陈明的身前转了一圈，特别展示了它强壮的后腿，然后陈明伸手摸了摸下巴，点头道:“嗯，这条大腿很不错，肥美细嫩，甚是劲道，去洗干净了烤好了送过来！”
<br/>    
<br/>        野猪首领愣了愣，两只眼睛看向陈明，“你是魔鬼吗？”
<br/>    
<br/>        陈明似乎是想起了什么，然后道:“对了，没有阉割过的猪，味道太重，你还是先把自己给阉割了，然后养一段时间再吃。”
<br/>    
<br/>        野猪首领直接五体投地趴地上了，“大仙饶命啊！”
<br/>    
<br/>        陈明忽然道:“对了，你耳朵里那个绿色的东西，从哪儿弄来的？”
<br/>    
<br/>        野猪首领支支吾吾，最后还是道:“大仙，那个地方，你不会想去的。”
<br/>    
<br/>        陈明道:“那我觉得还是阉割一下比较好”
<br/>    
<br/>        野猪首领终于是在阉割之下点头，“大仙，我带你去还不成吗？”
<br/>    
<br/>        化身成乌鸦，站在野猪的头顶，陈明跟着野猪来到一条黑色的大河前，野猪道:“大仙，游过这条河，就到了。那里有一座小岛，在小岛上便有着一棵树，结出这种果子。”
<br/>    
<br/>        陈明怒道:“那你还不赶快游，在这儿等着干嘛？”
<br/>    
<br/>        野猪道:“这是弱水，其他人若是触碰，都会被这弱水淹死，弱水之上，鸿毛不浮。”
<br/>    
<br/>        陈明道:“那你当初是怎么过去的。”
<br/>    
<br/>        野猪道:“弱水消灭的是意念灵魂强大的人，越是意念灵魂强大，就越沉，我老猪当初过去的时候，压根没有智慧啊，回来的时候，智慧还不明显，所以能够游过去。”
<br/>    
<br/>        陈明点点头，然后道:“你的意思是，你现在太聪明了，所以你也游不过去了？”
<br/>    
<br/>        野猪很是自信地点点头，然后陈明二话不说，上来就是一脚踢在野猪的屁股上，直接把野猪踢下了水，野猪在水里扑腾了一番，“不行了，不行了，我要死了.大仙，救命啊”
<br/>    
<br/>        陈明扫了一眼，这家伙还浮在水面上，脑袋全部在水面上啊。
<br/>    
<br/>        过了一会儿，野猪忽然发现自己竟然没死，好端端的浮在谁买了上，当即大喜道:“还好还好，我还是很蠢的！”
<br/>    
<br/>        陈明还是第一次看到有生灵因为自己蠢而感到高兴的，陈明振翅落在野猪的头顶上，野猪只觉得有一座大山压了下来，四蹄全力地划水，还是要沉下去。
<br/>    
<br/>        陈明也是感受到了，自己落上去之后就飞不起来了，太重了。
<br/>    
<br/>        直接一脚揣在野猪的身上，陈明再次上岸，野猪爬上岸来，很是愁眉苦脸地道:“不行啊仙师，你太聪明了，背不动啊！”
<br/>    
<br/>        陈明还是第一次遇到这种诡异的事，自己居然因为太聪明过不了河？
<br/>    
<br/>        陈明的目光在野猪的身上扫了一圈，一头野猪不够，多弄上几十头不就够了！
<br/>    
<br/>        这清心菩提树，我陈明要定了！
<br/>    
<br/>        不过这条河还真是诡异啊，这么诡异的东西，让陈明碰都不想碰，陈明没有半点想把这弱水当做武器的想法，若是这玩意儿能够被当做武器，那还不弱水满天飞了。
<br/>    
<br/>        估计连碰都不能碰。
<br/>    
<br/>        陈明想收一点到储物戒之中，却是发现，也没有任何效果，根本收不进来。
<br/>    
<br/>        陈明思考片刻，终于是道:“去，把你全家都叫来！”

<br/>        野猪首领现在哪里还敢反抗陈明，直接去叫了他的全家足足上百头野猪过来。
<br/>    
<br/>        陈明将一头头野猪踢下水，又将绳子在一头头野猪身上绑好，然后直接将一个硕大的木排丢到了上百头野猪的身上，反正没有智慧在这弱水上就没有智慧，木排应该没重量。
<br/>    
<br/>        木排落上去，果然没有什么反应，陈明这才一跃落在木排之上，只见一阵晃悠，但是还是稳住了。
<br/>    
<br/>        看来，上百头野猪还是有用的嘛！
<br/>    
<br/>        看了一眼前面的野猪首领，陈明道:“出发，前面带路！”

<br/>        陈明左右扫了一眼，还好没人发现我现在骑着一百头猪的样子，否则，身败名裂！
<br/>    
<br/>        哎，师者父母心，还不是为了门下这些败家玩意儿。

<br/>        就此，陈明踏着以猪为动力的木排，朝着那一座那传说中的岛前进，一路上，陈明看向水底之下，可以看得到一个个虚幻的人影飘过。

<br/>        其中不少都是披肩长发，脸色苍白，这分明就是鬼啊！

<br/>        这到底是什么鬼地方，别说，还真是鬼地方。

<br/>        陈明看向木排下的野猪，话说，这些野猪看到鬼，不怕的吗？

<br/>        我都觉得有点瘆得慌。
<br/>    
<br/>        然后陈明就听到了这些野猪的窃窃私语，“哇，快看，那个女的好长的头发，看起来好漂亮的样子！”

<br/>        “这边这个女的更奇怪，她没脸！”

<br/>        “原来人类的种类有这么多，好多不一样的啊！”

<br/>        果然，不知者无畏。
