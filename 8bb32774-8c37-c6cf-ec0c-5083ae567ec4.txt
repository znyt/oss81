        &lt;h3&gt;第三章第四十六章  机灵的闫欢&lt;/h3&gt;
<br/>
<br/>        陈子悦和闫欢正在西城，她们在西城已经开了五个连锁店了，这个是第六个，而且是最大的一个，她们将西城人流量多的庆户区的一栋大楼整体包了下来，一共10层，一次性签了十年的租赁合同。她们把这个打造成了一个具有规模的大型商场，停车场、日用百货，名牌服装，各种小吃等等。
<br/>
<br/>        从陈子昂入股她们公司后，她们开的超市是行云流水，分店遍地开花，日进斗金，因为她们做的都是民生方面的，和人们的生活息息相关，保质保量，服务又好，附近的居民肯定都是选择她们，只要进店都会衍生各种别的需求。
<br/>
<br/>        陈子昂是最大的股东，她和闫欢各占一小股，此时她和闫欢站在商场门口，请的是庆户区的韩老总来剪彩，她们也算是引进的投资商，政策上是会寄予扶持和鼓励的，闫欢的脑袋聪明，为人处世比她好，整个外围工作都是闫欢来做的。
<br/>
<br/>        原本剪彩后，晚上应该是要请人家吃饭的，但是陈子悦听说陈子昂回来了，也就没有心思和剪彩的老总吃饭了。
<br/>
<br/>        她对闫欢说：“欢，子昂回来了，我晚上要和他们一起吃饭，你就陪着韩老总一起吃饭，就说我有点不舒服。”
<br/>
<br/>        “子昂回来了？我也想见见她呢。咱晚上就不和他们一起吃饭了。”
<br/>
<br/>        “那怎么成？都已经说好了。”
<br/>
<br/>        “怕什么，等会听我的。等会我给你眼色，你就装病。”
<br/>
<br/>        剪彩结束之后，韩老总发表了简短的讲话，他们陪着韩总转了一圈，韩总大加赞赏，并表示感谢，感谢他们来投资，带动了庆户区的繁荣发展。
<br/>
<br/>        邀请韩总到办公室坐了一会，看看时间差不多了，闫欢给她使了个眼色，陈子悦皱了一下眉，她感觉这样不好，但是闫欢看着她说：“陈总，你怎么了？你哪里不舒服？”
<br/>
<br/>        “我没事。”陈子悦汗颜，这个闫欢就是一个机灵鬼。
<br/>
<br/>        “哎呀，陈总，你这几天本来就不舒服，你可别累倒下了。”闫欢站起来直接就走到她的身边，挡住了韩总他们的视线，使劲的掐了一下她。
<br/>
<br/>        “哎哟！”陈子悦吃疼，叫出了声。
<br/>
<br/>        “陈总，要不我送你去医院。”闫欢着急的说。
<br/>
<br/>        “那个，韩总，不好意思，我们陈总有点不适。”
<br/>
<br/>        “生病了，那赶紧送医院去看看吧，我就看陈总脸色好像不对，身体重要，身体重要。”
<br/>
<br/>        “韩总，我已经安排了酒店。”闫欢
<br/>
<br/>        “走了，走了，改天坐一坐。你赶快送陈总去医院。”韩总他们起身告辞。
<br/>
<br/>        闫欢开车急匆匆的送陈子悦去医院。
<br/>
<br/>        半路上
<br/>
<br/>        “闫欢，你太坏了，你掐疼我了。”
<br/>
<br/>        “嘿嘿，不使劲掐你，你会疼嘛？”
<br/>
<br/>        “你这是公报私仇。”
<br/>
<br/>        “没文化，真可怕，平时让我忙前忙后的，我这是私报公仇。我可是替你去见你的妹妹。”
<br/>
<br/>        “我又没有让你见我的妹妹。那是我的妹妹你跟着凑什么热闹？”
<br/>
<br/>        “我是去见大老板汇报工作。”
<br/>
<br/>        “都是你的理由。”
<br/>
<br/>        “说，去哪？不可能让我真的送你去医院吧？”
<br/>
<br/>        俩个人说说笑笑的赶往吃饭的地方。
<br/>
<br/>        “对了，给保姆打声招呼，我们不回去吃饭了。”闫欢对陈子悦说。
<br/>
<br/>        陈子悦和王浩楠生了一个女儿，家里人很是希望陈子悦生一个儿子，谁知道陈子悦就生了个女孩子。
<br/>
<br/>        生下孩子之时，因为是女孩子，当时王浩楠睡了两天，都不照顾陈子悦，而且王浩楠的母亲还唠唠叨叨的，说怎么又生了一个赔钱货，整个月子期间都是闫欢照顾着陈子悦。
<br/>
<br/>        闫欢是要管超市要照顾陈子悦，整个人累得瘦了一圈，也是因为如此，俩个女人的感情却比亲姐妹还要亲。
<br/>
<br/>        王浩楠一家人的费用都是陈子悦在管着，王浩楠的工资就那么点，还要开车，都不够他生活费的，好的是陈子悦的生意越做越好，也不怕给他们生活费。
<br/>
<br/>        陈子悦在陵县买了房子，将王浩楠的父母亲接到陵县住，没有生小的之前，王浩楠的父母亲还照顾着蔷悦，后来生了小楠楠，王浩楠的父母亲就不高兴了，再也不帮着看孩子，都是顾的人带孩子。
<br/>
<br/>        后来也不知道是谁给俩个老人说了陈子悦的过去，王浩楠的母亲就更是不待见陈子悦了。
<br/>
<br/>        王浩楠的母亲对王浩楠说：“她出身那样的一个环境，谁知道哪个孩子是不是你的？你还骗我们说，那个野种是你的孩子？我们这几年的感情白瞎了。”
<br/>
<br/>        王浩楠被他母亲说的心里更是烦躁。
<br/>
<br/>        从生下孩子后，俩个人的关系似乎就起了变化。
<br/>
<br/>        但是陈子悦无暇顾及家庭的矛盾，婆婆不帮她带娃，她就雇人带，她走到哪里孩子就跟到哪里。她和闫欢俩个人的生意越做越大，在陵县呆的时间也不多。
<br/>
<br/>        在西城开了超市之后，陈子悦将蔷悦和楠楠都带到了西城，俩个孩子都在西城上学。
<br/>
<br/>        王浩楠也会来西城，只是没有原来那么勤快。
<br/>
<br/>        闫欢不止一次的碰见王浩楠和她的那个女同事在一起，一次安慰女同事？次次都安慰？
<br/>
<br/>        她也不知道怎么给陈子悦开口说这种事情。
<br/>
<br/>        看着陈子悦干劲十足，忙忙碌碌的样子，闫欢也就三缄其口，装作什么都不知道，但是，纸是包不住火的。
<br/>
<br/>        不是王浩楠不想隐瞒，也不是陈子悦想要去打听。
<br/>
<br/>        是别人找上门了。
<br/>
<br/>        闫欢看了一眼陈子悦，她依然是一副恬静柔和的样子，自从知道王浩楠外面有人之后，陈子悦更是奋发图强，对王家俩个老人一如既往，只是基本不和王浩楠在一起了。
<br/>
<br/>        而且不吵不闹，不闻不问，似乎心如止水。
<br/>
<br/>        陈子悦看着闫欢一会看她一会看她：“你之前没有见过我？我脸上有花？看路。”
<br/>
<br/>        “你当然脸上有花，要不然我会这么死心塌地的为你卖命？”
<br/>
<br/>        “闫欢，别给自己脸上贴金啊，你也就是我要你。还卖命呢。”
<br/>
<br/>        “你这张嘴啊，也就是在我面前毒。”
<br/>
<br/>        “不毒你我毒谁啊？况且你毒不死。千年老妖怪。”
<br/>
<br/>        “彼此彼此！”
<br/>
<br/>        其实在她们心中对张函都充满了感激，她们俩个人都是因为张函的帮助才重新生活，而且也是因为张函俩个女人才在一起携手共进，展开了她们轰轰烈烈的事业。
<br/>
<br/>        要不是张函，她们俩人一个在狼窝，一个堕落红尘，怕是再无今生！
<br/>
<br/>        她们更好的生活，既是展现自我，也是回报张函的那一份心。
<br/>
<br/>        他们虽然没有太多的知识，好在她们都有一颗向往美好的心，她们足够努力、足够勤奋，她们仗着政策利好，仗着国富民强、踏着时代的步伐昂首前进。
<br/>
<br/>        这个世界上不乏困难，不乏黑暗，只要你向往光明，拥有一颗温柔的心，努力奋斗，所有的一切都会为你开路。
