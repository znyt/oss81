　　　　空中的飞行丧尸也有等阶之分，狼牙空军能够有时间瞄准击毙的，大多数都是等阶不是太高的飞行丧尸，而那些等阶高的飞行丧尸，除却体型巨大到恐怖之外，还有一点就是它们飞的很高，根本不是跟着大部队一起前来，而是忽然就从上方急速俯冲而下。
<br/>　　数量仅有一百的狼牙战斗直升机，在经过了这样的俯冲之后，就已经有两架直升机坠毁。
<br/>　　也就是说狼牙一下子就损失了两名军法处成员，及两名射击队最优秀的射手！
<br/>　　所有人都来不及反应，唯有立即知晓收到冲撞的那两名飞行员，在第一时间就用尽全力改变坠落方向，控制着机身朝着前方有着大·片丧尸聚集的范围而去。
<br/>　　两名在机舱内的射击队成员，更是在最后一刻猛开火力，不管是飞行丧尸还是陆地上成片的丧尸海洋，能杀多少是多少！
<br/>　　其余的狼牙空军成员望着那两架直升机坠落，在远处发出一声巨响，紧接着就是一阵火光冲天，将一大·片的丧尸都炸开成碎屑。
<br/>　　无论是军法处还是射击队的成员，眼见这一幕的空军全部心脏剧烈颤动起来，如果坠机就炸死更多的丧尸，这是他们起飞前就一致同意的共同决定。
<br/>　　“继续战斗。”
<br/>　　不知谁在沉寂的对讲机内开口，下一刻所有人都不在望向那四人牺牲的地方，目光再次紧盯高空。
<br/>　　他们没有时间悲伤，因为那会让更多的人死亡。
<br/>　　当那头高阶飞行丧尸冲出来的第一时间，不远处的高少辉就立即注意到，他连忙驾驶着天照3000急速飞去，只是依然来不及救下那两架坠毁的直升机。
<br/>　　但那头飞行丧尸却也只能撞毁两架，因为当天照3000到达的第一时间，这头飞行丧尸就被整个从上到下以一种恐怖无比的加速度撞向下方地面。
<br/>　　轰！
<br/>　　一声爆裂般的巨响，体型巨大的飞行丧尸被天照3000整个轰在地表上，压死了大·片的丧尸不说，更是让其一下子双翅骤断。
<br/>　　但只是这样还不够，于是天照3000下一秒又忽然消失不见，并不是真正的消失，而是在肉·眼无法触及到的速度中来到了高空，紧接着——
<br/>　　刷！
<br/>　　天照3000整个机身都朝着下方那倒在地上的飞行丧尸脑袋撞了过去！
<br/>　　轰！
<br/>　　又是一声巨响，地面大范围的深陷出一个凹坑，那高阶飞行丧尸的脑袋被撞的爆开，黑血像是瀑布一般哗啦啦的流淌而出，将周围被波及到而死伤一片的丧尸残骸都浸没。
<br/>　　“混蛋楚涵猜错了！这天照3000没有攻击设定，我只能用撞的！”天照3000内的高少辉一边骂着，一边将高度提升到空军作战之上，开始专门负责高阶丧尸。
<br/>　　被尸海包围的南署基地外墙，楚涵看着这一幕神色变了变，紧接着便抽空转身几次跳跃后，来到了内墙附近的指挥高台。
<br/>　　何峰正在不断听着侦查队汇报的战况，布满血丝的双目也一刻不停的盯着地图，他还在尽可能的将战术进行最合理的细微调整。
<br/>　　楚涵给出的战略一向都是大方向，细节之处都是何峰来处理，而这些细节从战前开始一直到战役结束，都有可能需要改动，因为哪怕一处微不可查的地方，都有可能会影响到人命。
<br/>　　“何峰，高少辉能不能杀丧尸？我是说人形丧尸。”楚涵一进来就问出一句没头没脑的话。
<br/>　　屋内忙的不可开交的众人都莫名其妙的看了楚涵一眼，但很快就将注意力放在了手中的工作上，来者是他们的最高长官楚涵，而楚涵的脑子里一向都装着他们搞不懂的东西，与其白费力气好奇，还不如专注于眼下战役。
<br/>　　何峰也正处忙碌之中，但当楚涵的声音一出，他就刷的回头，眼神充满了异样色彩。
<br/>　　“出来说？”楚涵看了眼屋内人多口杂的情况，转身就走向指挥间外的空地。
<br/>　　何峰二话不说的跟上，双目自然而然的看向了在此处轻而易举能观察到的高空战斗，他没有任何停留，走到楚涵身旁便开口：“丧尸不属于禁制，异种是。神秘家族一直不敢尝试，也是因为那个不属实的传闻。”
<br/>　　两人在这个紧张无比的大战之中，还有闲工夫聊起与之毫不相关的话题，要是被上官荣那些上将们知道，恐怕都得一个个吐血而亡。
<br/>　　听到何峰亲口所说的话，楚涵这才放下了心，笑道：“刚才高少辉杀高阶飞行丧尸的时候，一不小心恐怕还干掉了几百头丧尸群。”
<br/>　　何峰点头：“无关紧要，但我还真没想到高少辉会帮你到这个地步，有关丧尸到底能不能杀，神秘家族的人愣是纠结了两年多都不敢尝试。”
<br/>　　“生怕被你抓起来就地正法？那个不属实的传闻，不会就是肖家灭门的事吧？听高少辉说就是与尸潮有关？”楚涵心情不错，也就半开起了玩笑。
<br/>　　何峰摇头，道：“肖家的灭门是在末世爆发之前，而且也不是我动的手，是上一代……”
<br/>　　“哦，何培源。”楚涵若有所思，紧接着忽然道：“但我很奇怪，你们是怎么知道神秘家族的成员有无触动禁制？”
<br/>　　何峰看了楚涵一眼：“我们当然不知道，是阴阳谷知道，我们只是依照了制裁之地的指示，将人抓过去而已。”
<br/>　　这还是楚涵第一次听说内情，不禁想起制裁之地那口巨大深井，诸多神秘的感觉便油然而生，但同样也对之前高少辉见到丧尸就跑的可笑场景颇为无语。
<br/>　　压根就没有这一重限制，鬼知道那家伙在怕什么！
<br/>　　“还有一点，高少辉这个家伙知道我在这里。”何峰忽然眯起了眼睛，道：“高家的人一个都不简单啊！”
<br/>　　楚涵一愣，而后明悟：“制裁者就在这里对付丧尸群，他高少辉做了跟制裁者一样的事，制裁者怎么能抓他？或许他本身就在利用刚才的那头飞行丧尸试探，毕竟对于神秘家族来说行走在这丧尸遍野的世界，战力超绝却无法出手，的确很不爽。”
<br/>　　何峰看了楚涵一眼，便算是默认，他转身朝着指挥间走去。
<br/>　　楚涵也紧了紧修罗战斧，再次回到高墙之上。
<br/>　　两人的对话在这场战役之中微不足道，但无论是楚涵还是何峰都很清楚，这却是长远行程中的必要一环。
