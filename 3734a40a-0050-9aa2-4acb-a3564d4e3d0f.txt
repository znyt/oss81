        &lt;h3&gt;第三卷第九十七章  西城视察（感谢小九儿盟主）&lt;/h3&gt;
<br/>
<br/>        江俞琪不知道弟弟昨天为何急匆匆的去了北京，而且走的时候连招呼都没有给她打，只是给她留了言，说北京有事情，下午就会返回。
<br/>
<br/>        他们目前参与的南郊新区的改造项目中，加上原来的项目现在他们手中有了八个项目。
<br/>
<br/>        每天要和各种人打交道，如果光是买一块地皮盖个小区那倒是好说，可是现在是参与到新区规划中来，上面是非常重视的。
<br/>
<br/>        这一片还好都是原来的城中村和耕地，再加上是规划之类的项目，他们进行起来是非常方便的，但是这么大的项目同时开工，而且都被人盯着，他们要应付各自事情。
<br/>
<br/>        早上有老总要来视察，反正他们都是今天来视察，明天来视察，谁让这个项目是引人瞩目的呢。
<br/>
<br/>        陪同上面的老总来的依然是杨总，杨总工作态度严谨，平时都是他们姐弟俩个一起参加，而且杨总也多是和江俞轩沟通。
<br/>
<br/>        今天没有见到江俞轩有些意外。
<br/>
<br/>        江俞琪带着几个老总视察了各个工地，并且讲解了整个规划布局，以及预期交房的时间。
<br/>
<br/>        江俞琪指着在建的学校说：“这个是目前新区在建的学校，学校的项目要在业主入住之前完工，这个学校将会解决附近业主孩子上学难的问题。”
<br/>
<br/>        “小学、初中、高中在一起？”
<br/>
<br/>        “是的，小学和初中就隔了一条过道。”江俞琪指着在建的工地说。
<br/>
<br/>        正说着，突然旁边传来了喧哗声。
<br/>
<br/>        “怎么回事？”杨总问江俞琪。
<br/>
<br/>        “王越，你去看看怎么回事。”江俞琪吩咐身边的人。
<br/>
<br/>        大家本来都准备走了，然后停下来等着看怎么回事。
<br/>
<br/>        过了一会，保安扭着一个人过来了。
<br/>
<br/>        “江总，这个人到工地捣乱呢，被我们抓住了。”
<br/>
<br/>        “怎么回事？”江俞琪皱着眉头，怎么偏偏今天来捣乱？
<br/>
<br/>        “我们在工地旁边巡逻，看见这个人鬼鬼祟祟的，问他干什么，他扭头就跑。”保安回答。
<br/>
<br/>        工地施工一般人都很多，如果混进去人捣乱其实也很麻烦的，所以每个工地都配备着安保人员，还有监控设备。
<br/>
<br/>        “带下去问问怎么回事吧，附近都查一查看看有没有什么异样的？”
<br/>
<br/>        “工地的安全一定要保证，千万不能出任何差错。”某老总对着杨总说。
<br/>
<br/>        “是，江氏集团在这方面做的不错，整个区域都加固了防护网，配有安保人员。”杨东解释。
<br/>
<br/>        “嗯，大企业还是不一样啊，不过，江总这里就你一个人负责吗？”某老总问江俞琪，男人的骨子里多多少少还是会质疑女人的能力的。
<br/>
<br/>        “不，这里负责的是江俞轩，她的弟弟。”杨总说。
<br/>
<br/>        “哦，今天不在？”
<br/>
<br/>        “他去了北京，北京那边有事情。”
<br/>
<br/>        “什么事情？”某老总有些不开心，他们要来视察的消息可是在几天前都说了的。
<br/>
<br/>        “讨论婚期。”江俞琪无奈了。
<br/>
<br/>        “讨论婚期？”某老总明显不高兴了。
<br/>
<br/>        江俞琪感觉要阴转多云了。
<br/>
<br/>        “弟弟和张倩楠谈了好几年了，原本十一要结婚的，可是因为太忙，没有顾上，就拖下来了，这不，眼看着俩个人年龄越来越大了，两方家长着急了。”
<br/>
<br/>        “京城四美，其中一个名字就叫张倩楠。”另外一个说。
<br/>
<br/>        “正是我未来的兄弟媳妇。”江俞琪微微的笑着。
<br/>
<br/>        “哈哈哈，郎才女貌啊。”某老总笑着说。
<br/>
<br/>        “以后可得请我们吃喜糖啊。”
<br/>
<br/>        “那是当然，到时候一定请各位老总吃喜糖，望各位老总赏光。”江俞琪都没有想到她竟然拿着未来的兄弟媳妇做了一回挡箭牌。
<br/>
<br/>        人啊！
<br/>
<br/>        大家都很满意，对今天的工作做了一番指导和总结。
<br/>
<br/>        江俞琪有些生气弟弟，这也不知道谁什么天大的事情呢，说走就走了？
<br/>
<br/>        送走了各位老总，江俞琪叫来王越，她要问一下今天中午的情况。
<br/>
<br/>        “今天是怎么回事？”江俞琪脱掉外面的羊绒大衣，里面是得体的工作服，她虽然是工作服，但是都是定制的名牌，尖头小黑皮靴，身材苗条，显得精致而干练。
<br/>
<br/>        “是这样的，那个人姓冯，原本死也不开口，后来我们说要将他送到警，察局，他才说他是某单位派过来让他看看工地的情况。”
<br/>
<br/>        “为什么要看我们工地的情况？”
<br/>
<br/>        “说是想学习我们先进的经验。”
<br/>
<br/>        “怕不会是这么简单吧？既然是学习经验的，为何不光明正大的学习？还有他跑什么？”江俞琪问。
<br/>
<br/>        “再问，他就死活不说，我们也不能拿他怎么样，只有交到警，察局了。”
<br/>
<br/>        “你去警局看看情况，一定要多了解一些情况。”
<br/>
<br/>        “好的。”
<br/>
<br/>        江俞琪也是见过大风大浪的人，这一点小猫腻是瞒不了她的，她和某单位其实也没有太大的过节，生意场上的事情，抬头不见低头见的，你来我往都是正常的，只是这事情有点蹊跷。
<br/>
<br/>        “我们这些个工地，安全问题现在是第一，且不可大意，每一个进入工地的人必须持证上岗，像今天这人的行为一定要注意。”江俞琪再次叮咛。
<br/>
<br/>        “明白，我现在就去开会。”王越也能意识到安全的重要性。
<br/>
<br/>        过了两个多小时，王越回来了。
<br/>
<br/>        “那个姓冯的信息查到了，他以前是某公司的经理，现在不在某公司任职了，但是他对我们心存怨恨，其实是想搞破坏，没有想到还没有实施就被安保人员发现了。”
<br/>
<br/>        “他说是因为我们，他才失业的。”
<br/>
<br/>        “他是陈子建的妻弟，他们公司让他去找陈子建，让陈子建带人给我们勘查的时候说这块地皮不能开发，但是陈子建没有听，他们公司觉得他没有用处，就找理由将他开了。”
<br/>
<br/>        江俞琪觉得现在有些人的脑子真的是让人感觉不可思议。
<br/>
<br/>        “给陈子建一个面子，这件事情就过去了吧。”江俞琪说“这样吧，我给你一个电话，你打电话给这个人，就说她弟弟在警局，让她去领人。”
<br/>
<br/>        江俞琪将冯睿的电话给了王越。
<br/>
<br/>        一个小时之后，冯睿的电话就打过来了。
<br/>
<br/>        “江总，听说我弟弟因为你们工地的事情，被警局的人带走了，我给子昂打电话，子昂正忙着，我就直接给你打电话了。好歹我们两家也是亲戚，你怎么能让警局将我弟弟带走呢？”
<br/>
<br/>        江俞琪直接逻辑混乱了，她本来是想提醒冯睿，给陈子建还一个人情。没有想到人家直接兴师问罪来了。
<br/>
<br/>        “你说我们是亲戚？”江俞琪不知道冯睿哪里来的自信。
<br/>
<br/>        “怎么不是呢，他们孩子都有了。”
<br/>
<br/>        “呵呵，冯总，我弟弟开玩笑你也信？”
<br/>
<br/>        “我就说呢，陈子昂怎么可能攀上你们家。”冯睿这一段时间找不到陈子建，心里火大得很。
<br/>
<br/>        但是江俞琪说的话她也不信，她认为江俞琪在敷衍她。
<br/>
<br/>        江俞琪感觉她的三观碎了一地。
<br/>
<br/>        这是陈子昂的嫂子吗？
<br/>
<br/>        “江总，您看啊，我们子建好歹也帮过您吧，你就大人有大量，放过我弟弟吧。”
<br/>
<br/>        “你直接去警局带他回家不就得了，给我打什么电话呢，不好意思啊，冯总，我这边还有事情，回头聊，你直接去警局接他回家就是。”
<br/>
<br/>        江俞琪懒得听冯睿的话，直接挂了电话。
