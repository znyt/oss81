　　楚风眠是一尊剑修，所谓剑修，最为重要的，便是剑。
<br/>　　楚风眠手中的赤炎剑，虽然被他炼制过，威力已经是强大了许多，但是以现在楚风眠的实力。
<br/>　　赤炎剑已经无法完全发挥出楚风眠的剑术。
<br/>　　只有天级灵器，有着灵性的天级灵器，才足以将楚风眠的剑术，发挥到大成。
<br/>　　若是可以得到一把天级灵器的长剑，楚风眠的实力，还可以再进一步！增强一倍，都有可能。
<br/>　　这把天级灵器的长剑，似乎一开始并未出现在密室之中，等待众人收取了无数宝物之后，才突然出现的。
<br/>　　以至于楚风眠一开始都没有察觉到。
<br/>　　“这把剑，我必须要得到！”
<br/>　　楚风眠心中自言自语一句，他的眼神，坚定无比。
<br/>　　天级灵器，本就是百年难遇，更何况是一把天级灵器的长剑。
<br/>　　楚风眠的身形连连闪动，一下子也靠近了这一把天级灵器附近。
<br/>　　“楚风眠，你的收获怎么样？”
<br/>　　楚风眠刚刚来到附近，白萱便是飞了过来，出声询问道。
<br/>　　她的脸上，也露出了几分喜色，在争夺之中，她也得到了其中的三件宝物。
<br/>　　虽然说远远比不上楚风眠，但是在如此之多的众人混战之中，能够得到三件宝物，也是不错的机缘了。
<br/>　　“收获不错。”
<br/>　　楚风眠轻笑了一声，并没有细说，反倒是开口询问道。
<br/>　　“这天级灵器！是什么时候出现的？”
<br/>　　楚风眠一直是在密室内部，收取宝物，所以并没有注意到，这天级灵器，是什么时候出现的。
<br/>　　“据说是有个人，突然引动了机关，才突然出现的，不过眼前这天级灵器，想要拿到，却并不容易。”
<br/>　　白萱指着那天级灵器，开口道。
<br/>　　“你看上面的禁制。”
<br/>　　顺着白萱所指，楚风眠看了过去。
<br/>　　这天级灵器，跟颜古龙帝洞府中的无数宝物一样，其中上面设置了禁制。
<br/>　　七彩霞光，遍布这一把天级灵器周围。
<br/>　　但是这天级灵器上的禁制，却是有着一点不同，那就是力量的差距。
<br/>　　在这把天级灵器上的禁制，只怕是比起这密室大门上的禁制，都要坚固的多，虽然经历了五万年的岁月冲刷。
<br/>　　其中的力量，还是让楚风眠都感觉到，以他现在的全力，在这禁制面前，都是渺小的蝼蚁。
<br/>　　“这禁制，莫非是颜古龙帝亲手布置的？”
<br/>　　楚风眠看到这天级灵器上面的七彩霞光，也是吓了一跳。
<br/>　　岁月之力，才是世间最为恐怖的力量，无数远古强者，都无法抵抗岁月的冲刷，直接陨落。
<br/>　　这禁制，也同样如此，强者的禁制，可以维持一百年，一千年，一万年。
<br/>　　但是五万年的时间，足以将其中的大部分力量，都给消耗殆尽，可眼前这七彩霞光的力量，却是在五万年后，还拥有这如此恐怖的力量。
<br/>　　能够做到这一点的，只怕也就只有曾经踏上九域巅峰的颜古龙帝，亲手布置下来的禁制才有可能。
<br/>　　“利用血脉之力，我可以化解这上面的禁制，但是在如今众目睽睽之下，若是化解了上面的禁制，只怕是会被认群起攻之！”
<br/>　　楚风眠看着那天级灵器，也不由的眉头一皱起来。
<br/>　　想要得到这天级灵器，并不容易，天级灵器，人人都想要得到。
<br/>　　在场的众人，每个人都是在虎视眈眈，要是现在楚风眠冲上去，绝对会引起众人的愤怒。
<br/>　　以现在楚风眠的实力，要是众人一起围杀的话，他想要逃出去，也都是极难。
<br/>　　楚风眠低声自语道。
<br/>　　“要等一个机会才行。”
<br/>　　楚风眠一边等待着机会，一边便是观察的眼前这把天级灵器。
<br/>　　在这把长剑的剑身之上，雕刻这一条栩栩如生的龙。
<br/>　　这条龙，并非是寻常真龙，就算是一个雕像，楚风眠也可以感觉到上面的恐怖龙威，跟古老之际的气息。
<br/>　　这气息，比起远古战龙都要古老，这是万龙之祖的气息。
<br/>　　“莫非这把剑，就是传说中龙族的至宝之一，祖龙至尊剑？”
<br/>　　楚风眠脑海一动，刹那之间便是想到了这把剑的来历。
<br/>　　龙族，虽然天生神力，但是却也不是没有龙族，愿意修炼剑道。
<br/>　　起码曾经的颜古龙帝，便是一尊强大无比的剑修，他的剑道造诣，也达到了登峰造极的地步。
<br/>　　一尊剑修扬名，手中的剑，一样重要。
<br/>　　颜古龙帝的剑，便是龙族至宝之一的祖龙至尊剑，号称是曾经万龙之祖亲手缔造而出的，拥有万龙之祖的力量。
<br/>　　“若真是祖龙至尊剑的话，我要是得到了，只怕是好处更多！我的血脉毕竟是远古战龙的血脉，虽然不纯，但是也可以勉强催动其中的万龙之祖的力量！”
<br/>　　楚风眠心中暗自想到，眼前这把祖龙至尊剑，他是必须要得到才行。
<br/>　　一个时辰，两个时辰。
<br/>　　楚风眠一直是在等待着机会，那落长恨等人，也都是在这祖龙至尊剑面前，等待着。
<br/>　　在这段时间中，有着几尊武者，曾经是去试探这其中七彩霞光的威力。
<br/>　　但是近乎是瞬息之间，便是就被七彩霞光，直接灭杀了。
<br/>　　没有一丝一毫的反抗，瞬间被灭杀，这也让在场众人的脸色，都变极为惊恐。
<br/>　　可是却没有任何一人，想要离开，因为这天级灵器，实在是诱惑太大，没有人会愿意放弃这个机会。
<br/>　　时间越来越久，在这密室之中，聚集的人也越来越多。
<br/>　　“轰隆！”
<br/>　　从外，又是五六尊身影，突然出现。
<br/>　　但是这一次，这五六人身上的气息，却是强大到了极点。
<br/>　　这五六人中最为弱小的几人，都丝毫不比落长恨要差，其中最强的一人，隐约之间，更是压过了在场的无数人。
<br/>　　“这些人是？”
<br/>　　在场众人，无数目光直接看了过去，当看到这五六人身上衣服胸口上，一个大大的荒字的一刻，一个个皆是脸色大变起来。
<br/>　　这个荒字，直接便是代表了他们的身份。
<br/>　　“中州！荒古门的人！”
<br/>　　白萱都是长大的嘴巴，开口道。
<br/>　　“荒古门的弟子，怎么会来到我们北域？”
