　　对于苏莫排进真圣榜第一，众人基本没有异议了，但下面的那些排名，多多少少还是有些异议。
<br/>　　只不过，就算有异议，也没有人再提出来了。
<br/>　　毕竟，就算有异议的话，谁也拿不出什么实质的证明。
<br/>　　比如尤天韩，他排在了真圣榜第一百六十七名，但他认为自己能排进前一百名，甚至是前五十名，但他只是自己认为，并没有实质的证据。
<br/>　　除非他能打败排在前面的人，证明自己。
<br/>　　上了榜单的人都无法反驳，其他没有上榜的人，更加不会多说什么了。
<br/>　　“玄净前辈，那就开始拍卖会吧！”
<br/>　　“是啊，已经浪费了很长的时间！”
<br/>　　“开始吧！”
<br/>　　众人纷纷开口，不过很多人心中都憋了一口气。
<br/>　　有些没有上榜的人，则是思索着，在真圣界开启的时候，一鸣惊人，挫败那些荣登真圣榜的人。
<br/>　　而少量登上了榜单的人，则也是准备到时候，挫败排名在自己前面的人，证明自己的排名应该更高。
<br/>　　“呵呵！那好，现在拍卖会正式开始！”
<br/>　　玄净尊者朗笑一声，也不浪费时间，立刻轻轻拍了拍手。
<br/>　　而后，后方的宫殿之中，便有一名美貌的侍女，端着一个长宽约三尺的紫色托盘走上了拍卖台。
<br/>　　托盘上盖着艳丽的大红色丝绸，丝绸掀开，露出了一柄银色的短刀。
<br/>　　“碧蛇刀，尊级下品兵器，品质上佳，威力强大，起拍价五百块下品灵晶，每次加价不得少于五十块下品灵晶，现在开始竞拍！”
<br/>　　玄净尊者朗声喝道，其话音一落，出价声便不绝于耳。
<br/>　　“五百五十块下品灵晶！”
<br/>　　“六百五十块！”
<br/>　　“七百块！”
<br/>　　“……！”
<br/>　　随着不断的竞价，整个拍卖场渐渐的火热了起来，声浪不断。
<br/>　　但是，此时的苏莫，已经没有了留下来竞拍的心思了，他必须尽快行动起来。
<br/>　　“谢兄，这拍卖会能否中途离开？”苏莫低声向谢彬问道，他见拍卖场的大门紧闭，所以有此一问。
<br/>　　“中途离开？”谢彬闻言一怔，不明白苏莫为何要离开，但还是点头道：“当然可以！”
<br/>　　“那就好！”苏莫点头，能提前离开就好，不过现在整个拍卖场都没有人离开，他暂时也不好离开，免得引人注目。
<br/>　　远处，尤天韩一直在暗中留意着苏莫，听到苏莫的话语之后，嘴角勾起了一丝冷笑。
<br/>　　拍卖会持续进行之中，一件又一件的宝物不断的被拍卖。
<br/>　　这拍卖会虽然是武帝境的拍卖会，但尊级的宝物非常多，因为来参加拍卖会的人，大多都不是一般的武帝境武者。
<br/>　　拍卖几乎是不分白天和黑夜，即便到了晚上，拍卖会依旧持续进行，拍卖的各种宝物，已经多达数百件。
<br/>　　黑夜对于武帝境的武者来说，没有丝毫的影响。
<br/>　　经过了长时间的考虑，苏莫心中已经有所决定了，放弃了逃离天冥星的打算，而是躲进神武学府之中。
<br/>　　巫族虽然强大，但天冥星支脉却并没有多强，神武学府应该能够庇护他。
<br/>　　当然，这也不是长久之计，躲在神武学府只能是短期的打算。
<br/>　　“下一件拍卖品，是一门火属性功法，等级为尊级中品，起拍价一万下品灵晶，每次加价不得少于一千下品灵晶！”
<br/>　　玄净尊者的话语再次响起，随即一枚玉简呈现在了众人的眼中。
<br/>　　尊级中品的功法，比之尊级下品的宝物，价格昂贵十倍都不止。
<br/>　　苏莫眼眸一亮，火属性的功法正是他需要的，虽然只是尊级中品功法，但也相当不凡了。
<br/>　　而且，拍卖会进行到现在，根本没有尊级上品的宝物出现，一件都没有，想必只有武尊境的拍卖场，才有尊级上品的宝物。
<br/>　　“一万二千块下品灵晶！”
<br/>　　“一万三千块！”
<br/>　　“一万五千块！”
<br/>　　“……”
<br/>　　尊级中品的功法出现，顿时引起了很多人的竞价，出价声不绝于耳。
<br/>　　没过多久，这门尊级中品功法，价格就被顶到了四万下品灵晶的地步。
<br/>　　最后，苏莫出价四万五千下品灵晶，成功的将此门功法竞拍到手。
<br/>　　很快，便有侍女将玉简送到了苏莫的面前，他直接支付了灵晶。
<br/>　　此拍卖会，由于规模太大，每一件拍卖品拍卖成功之后，均是立刻支付灵晶，奉上拍卖品。
<br/>　　接过玉简，苏莫稍稍查看了一番，便立刻收了起来，的确是尊级中品功法，玄妙无比。
<br/>　　此门功法，名为《三千炎焱诀》，比他修炼的帝级上品火属性功法，强大数倍不止。
<br/>　　拍卖继续，过了一个时辰以后，又出现了一门金属性的尊级中品功法。
<br/>　　毫无疑问，这门功法再次被苏莫收入囊中，花费了四万八千下品灵晶。
<br/>　　没有人争的过他，因为身上财富巨大的人，不缺尊级中品功法，那些需要尊级中品功法的人，身上财富都很有限。
<br/>　　四万多下品灵晶，一般的武帝境武者，根本就不可能拿出来。
<br/>　　拍卖会一刻都不停，一件又一件宝物被拍卖，苏莫只是静静的看着，除了需要的功法之外，其它一概不参与竞拍。
<br/>　　谢彬倒是参与了几次竞拍，但是只拍得了一件尊级下品的内甲。
<br/>　　这拍卖会果然是宝物居多，两天之内，五行属性的尊级中品功法便全部出现了。
<br/>　　毫无疑问，全部都被苏莫拿下了，其中最贵的一门土属性功法，价格高达五万下品灵晶。
<br/>　　“苏兄，你还真是财富惊人啊！”谢彬看着苏莫又收起了一枚玉简，眸中露出震惊之色，因为苏莫拍得了五门尊级中品功法，花费的灵晶接近二十万的恐怖地步。
<br/>　　他完全没有想到，苏莫居然如此的富有。
<br/>　　不过，想到苏莫斩杀太史世家派系的人时，好像夺取了不少储物戒，便也释然了。
<br/>　　“长时间积累下来的！”
<br/>　　苏莫笑了笑，没有过多解释，因为连续拍得了五门尊级中品功法，他已经引起了不少人的关注。
<br/>　　毕竟，近二十万的下品灵晶，可不是小数目。
<br/>　　尤天韩望了一眼苏莫，眸中闪过疑惑之色，不明白苏莫为何购买这么多功法。
<br/>　　因为按理来说，一门功法便足够修炼数十年，甚至是数百年了，就算购买再多的功法也是无用。
<br/>　　接下来的拍卖之中，苏莫便彻底失去了兴趣，因为他需要购买的功法，已经全部凑齐了。
<br/>　　这个时候，他已经生出了离开之心，必须要尽早离开了。
