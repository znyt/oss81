        &lt;h3&gt;第三卷第二百七十四章  寒伯安见陆宇霆&lt;/h3&gt;
<br/>
<br/>        寒伯安没有时间在西城呆多久，公司的事情还需要他回去处理，李长卿先期回了公司，但李长卿只是一个助理，重大的事情还需要寒伯安出面。
<br/>
<br/>        寒伯安很想陪着安男在西城多玩几天，况且郝景文和江俞轩还在西城，可是看着邮箱里李长卿发过来的邮件，他不得不回上海。
<br/>
<br/>        “寒总，陆宇霆和吴润竹到访。看来是不见你不罢休。速决定。”
<br/>
<br/>        陆宇霆和吴润竹第一次到上海来的时候，寒伯安让李长卿接待的，这一次如果再不见，只怕是真的会撕破脸皮，毕竟吴家和陆家还是数一数二的人家。
<br/>
<br/>        这一次他们来会不会和甸城有关系？苍岚雪的靠山到底是哪一家？
<br/>
<br/>        陆家如此的找他，到底意欲何为？
<br/>
<br/>        寒伯安皱着眉头，看着李长卿发来的邮件陷入沉思。
<br/>
<br/>        安男轻轻的走到寒伯安的面前，手搭在寒伯安的肩膀，轻柔的给寒伯安捏着脖子，被寒伯安反手一拉跌入寒伯安的怀里。
<br/>
<br/>        “在想什么呢？眉头紧皱？”安男的手拂上寒伯安的脸，揉捏着寒伯安的眉头。
<br/>
<br/>        “想你。”
<br/>
<br/>        “你这回答能不能走点心，想我的时候不应该是甜甜的吗怎会眉头紧皱？”安男柔声说。
<br/>
<br/>        “我们和陆家并无往来，但是陆家最近这一段时间似乎是想和我们拉上关系，我不想参与到他们任何一方复杂的关系里。”
<br/>
<br/>        “逃避不是问题，该解决的还是要解决的，如果你不想和他们有所接触，就想办法委婉的拒绝掉。”安男安慰着寒伯安，她不懂生意上的事情，但她不想看到寒伯安苦恼。
<br/>
<br/>        “我得回上海。”寒伯安头抵着安男的头，低声说。
<br/>
<br/>        “那你先回去，我陪着爸爸妈妈，舅舅马上要做手术，我这个时候不能离开。”安男环着寒伯安的脖子。
<br/>
<br/>        “安男，以后也许我们会遇到很多不可预知的事情，你一定要记得用你的智慧保护好自己，我有时候都后悔和你公开我们的关系。”
<br/>
<br/>        “我们是夫妻，我们应该携手与共，我不在乎什么名誉和地位，我只想和你平平安安的过一生，可是我知道寒伯安的身份和地位，我已经做好了思想准备，时时刻刻和寒伯安在一起努力着战斗着。你放心，我不会拖你的后腿的。”
<br/>
<br/>        “我以为我能给你幸福，没有想到我给你的是担惊受怕。”
<br/>
<br/>        “所有的担惊受怕都抵不上我深入骨髓的爱着你。”安男站起来：“我给你收拾行李。”
<br/>
<br/>        “一起。”寒伯安要回上海留下安男在西城，安男要等陈元良的手术做完之后才能回去。
<br/>
<br/>        .......
<br/>
<br/>        李长卿在机场接机，看到寒伯安到了，走过去接过寒伯安的行李箱：“安男小姐姐没有回来？难道你们不是比翼双飞吗？”
<br/>
<br/>        “比你个头啊，快说，怎么回事？”寒伯安嫌弃的看了李长卿一眼。
<br/>
<br/>        “你这是有了安男小姐姐，立马就翻脸不认人了？合着我陪你演了几次戏都白演了？”李长卿表示不服。
<br/>
<br/>        “我如果问你怎么回事，你还没有回答！难道要我问第三遍，李长卿，你最近是不是嘚瑟的上头了？”寒伯安语气冰冷。
<br/>
<br/>        “我回来的第二天，陆宇霆和吴润竹就找上门了，非要见你，我已经拖了一天，如果你不见，怕说不过去，我就告诉他们，你出差了，我一定联系你让你早点回来。”李长卿立马换了一副面孔。
<br/>
<br/>        “知道他们想干什么吗？”
<br/>
<br/>        “似乎他们就是来拜访你的，大家族一般都需要强大的财力和支持。”
<br/>
<br/>        “他们和甸城有没有关系？”
<br/>
<br/>        “看不出来，陆宇霆这人，我对他的感觉还不错，好像没有太多的心机。”
<br/>
<br/>        “人不可貌相你不知道？你见了几次面？”寒伯安停下来看着李长卿，这个李长卿人是聪明也很能干，就是自我攻略很严重，这种人往往就会被表面所欺骗。
<br/>
<br/>        “感觉陆宇霆不错，比他的那个妻子要好很多，他的妻子心机特别沉。”
<br/>
<br/>        “吴家人！”
<br/>
<br/>        .......
<br/>
<br/>        寒伯安是第一次见陆玉霆，大概也就是三十多岁的年纪，温润如玉，风度翩翩，说话也是慢条斯理的，寒伯安突然想到了一句诗：眼明如碧波清潭，眉自如晓霜映日。
<br/>
<br/>        吴润竹一绺靓丽的黑发飞瀑般飘洒下来，弯弯的柳眉，一双明眸勾魂慑魄，秀挺的琼鼻，粉腮微微泛红，滴水樱桃般的樱唇，嫩滑的雪肌如冰似雪，身材绝美，但是眼神感觉深不可测，让人心生寒意。
<br/>
<br/>        难怪李长卿说他对陆宇霆的感觉不错，这是一对夫妻，俩个人都是俊美无俦，可是感觉俩个人的行事风格感觉格格不入。
<br/>
<br/>        “久闻寒总大名，今天能见到真是三生有幸。”陆宇霆见到寒伯安开心的说。
<br/>
<br/>        “真的抱歉，陆总来上海两次我都出差了，这次还是长卿给我说的，要不然我都不知道你来上海了，实在是有些慢待了。”
<br/>
<br/>        “能见到寒总我就已经很开心了。”陆宇霆确实看起来很开心。
<br/>
<br/>        “这位是？”
<br/>
<br/>        “吴润竹，我的老婆。”陆宇霆笑着说介绍。
<br/>
<br/>        “吴女士真不愧是京城四美，早就听说二位郎才女貌，是最幸福最美好的一对神仙伴侣，今天一见果然如此，真是令人心生羡慕啊。”寒伯安看了一眼陆宇霆夫妇，又看了一眼李长卿。
<br/>
<br/>        “你这见谁都羡慕，难道我在你身边你就感觉不到幸福吗？”李长卿不满的开口了。
<br/>
<br/>        吴润竹听得一愣。
<br/>
<br/>        “别闹。”寒伯安的语气宠溺。
<br/>
<br/>        陆宇霆和吴润竹对视一眼。
<br/>
<br/>        “不好意思，我的这个助理总是会有一些小脾气。”寒伯安解释着：“我们不用管他。”
<br/>
<br/>        “寒夫人没有在上海吗？”吴润竹问，按道理他们夫妇来访，寒伯安是应该带着夫人来接待他们的。
<br/>
<br/>        “吴夫人这是不满意我陪着你们了？第一次是我接待你们的，第二次就想过河拆桥？我哪里比不上他的那个劳什子夫人了？”李长卿看着自己修长的手指一反常态冷冰冰的说。
<br/>
<br/>        “不，不，你误会了，感谢李助理的盛情款待。”陆宇霆急急忙忙开口说。
<br/>
<br/>        “感谢李助理的盛情款待，来，我敬李助理一杯。”吴润竹也看出来寒伯安和李长卿的关系不一般，那么李长卿在他们的这次之行中是不是会起很重要的作用？
<br/>
<br/>        本来是想着找安男，从安男身上找缺口，现在看来这个李长卿的作用不可小觑。
