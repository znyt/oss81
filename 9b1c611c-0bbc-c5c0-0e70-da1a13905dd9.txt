<br/>    冰螭携带着无尽的寒气朝着陈明冲去，陈明瞬间就淹没在一片寒气之中。
<br/>    
<br/>        尸九泉展开感知，可以感觉到陈明还活着。
<br/>    
<br/>        无尽的寒气之中，陈明连连挥拳，搏仙术施展，恶蛟与穷奇当空飞舞，咆哮着将周围的寒气驱散。
<br/>    
<br/>        陈明立于一地的冰晶之中，看向远处眸光阴冷的尸九泉。
<br/>    
<br/>        尸九泉将白骨权杖一转，无尽的寒风从白骨权杖之中吹出，寒风之中有着细小的冰晶，化作一柄柄冰刀，朝着陈明刺去。
<br/>    
<br/>        陈明双臂展开，挥拳大开大合，如同起手撼动群山，拳威将一阵阵寒风打得停滞，陈明再次朝着尸九泉挥去。
<br/>    
<br/>        尸九泉趁着陈明驱散寒风的片刻，已经再次挥动白骨权杖，想要再次凝聚一条冰螭！
<br/>    
<br/>        陈明扫了尸九泉一眼，尸九泉最为强大的手段便是这冰螭，张口吐出一口寒气，陈明的意念落在地面上的春秋之上，不过这样操控春秋是不是太明显了。
<br/>    
<br/>        陈明的意念一动，万剑出鞘，从背后的万道天竹之中飞出，在陈明的周围瞬间布成一道剑网。
<br/>    
<br/>        趁着万剑飞舞之极，陈明操控着春秋，缓缓深入地面之下。
<br/>    
<br/>        尸九泉扫了周围的这些飞剑一眼，“你应该知道，你的这些飞剑，对于我来说，是没有用的，陈明呵呵一笑道:“至少，只要你一剑没有防住，那么我就有机会近身一次。”
<br/>    
<br/>        现在的局面，还真是的有些搞笑，陈明作为一个阵法师，竟然要靠着近战战胜敌人。
<br/>    
<br/>        不过也没办法，现在的周天星辰大阵只是一个残阵，根本没有办法施展星辰大阵的全部威力。
<br/>    
<br/>        陈明朝着尸九泉冲去，就在这时，尸九泉根本没有凝聚完整的冰螭，可是也只能迫不得已地朝着陈明放出。
<br/>    
<br/>        陈明再次被轰退三十步，挥动拳头打碎这一条冰螭，尸九泉的眼中也是露出一抹怒色，他还是第一次和这么强大的阵法师对战，他一直以为，自己最害怕的敌人就是雷系的修行者，没有想到，在陈明的面前，他的术法，更是没有任何用处，只是用人多去干扰一个阵法师，给自己争取念咒的时间，根本就是做梦！
<br/>    
<br/>        两人在百里平台之上战斗无数次，都是有些损伤，尸九泉没有办法，只能再次放出四具活尸，这也是他最后的四具活尸。
<br/>    
<br/>        陈明与四具活尸缠斗在了一处，尸九泉的眼中终于是露出了一抹喜色。
<br/>    
<br/>        就在冰螭即将凝聚而出的那一瞬间，尸九泉可以看得到，陈明已经解决了四具活尸，虽然没有凝聚完成，尸九泉还是只能将手中的冰螭打出。
<br/>    
<br/>        就在冰螭即将吞没陈明的那一瞬间，尸九泉忽然发现陈明消失了，就在自己的脚下，一阵剧烈的颤动传来，糟了，这个小子怎么会突然出现在这里！
<br/>    
<br/>        陈明从地下飞出，悍然一拳便是轰在了尸九泉的腰间，打得尸九泉的骨头咔嚓一声脆响！
<br/>    
<br/>        陈明见一击得手，怎么会放过这么好的机会！
<br/>    
<br/>        又是连续三拳落下，打得尸九泉连连咳血。
<br/>    
<br/>        陈明得势不饶人，左手伸手一把抓住尸九泉，右手如同雷鸣闪电一般，搏仙术连连施展，如同狂风暴雨一般在尸九泉的身上落拳！
<br/>    
<br/>        陈明扫了尸九泉一眼，想要将尸九泉直接灭杀，尸九泉却是道:“住手，我在这颗星辰之上浪费了一千年的时间，你就不想知道为什么我会如此吗？”
<br/>    
<br/>        陈明的拳头停住，想要听听看尸九泉说什么，尸九泉现在的危险指数，不过三千而已，他倒要看看，尸九泉到底能够玩出什么花来。
<br/>    
<br/>        尸九泉擦了擦嘴角的黑血，然后道:“在这座大陆之上，有着一条河流，这一条河流之中，有着无尽的灵魂，而这些灵魂的去向，便是一棵树，一棵英灵树！”
<br/>    
<br/>        陈明有些疑惑，“英灵树，什么玩意儿？”
<br/>    
<br/>        尸九泉道:“英灵树，有着很多种，比如星空之中有着仙王坐镇的瑶池，之中便是有着一棵蟠桃圣树，吃下这蟠桃，便可以延长自己的寿命，又比如天庭，有着一棵造化仙树，吃下这造化仙树的果子，便可以提升境界！诸如此类，还有很多种，而这座大陆之上，便是有着一棵英灵树的树苗！”
<br/>    
<br/>        陈明愣了愣，问道:“这怎么可能，这些可都是逆天的东西！寿元，可是限制修行者最大的障碍，而吃下一颗果子，便可以提升境界，开什么玩笑？境界可是需要感悟的，又不是像闻道境一般，只需要增强肉身！”
<br/>    
<br/>        尸九泉笑道:“我就知道你会吃惊无比，你要知道，这些树的养料到底是什么。”
<br/>    
<br/>        陈明道:“是什么？”
<br/>    
<br/>        尸九泉道:“是灵魂！只有灵魂才能供养这些树的成长！”
<br/>    
<br/>        陈明忽然想到了燕山的清心菩提树，这棵树上结出的果子，可是能够让人的悟性提升，让他的师德收获速度快上十倍！
<br/>    
<br/>        这棵树，又何尝不是夺天地造化？
<br/>    
<br/>        而且，在清心菩提树的周围，有着无数的灵魂，正在朝着菩提树游去，或者说，根本不是游去，而是在吸取这些灵魂！
<br/>    
<br/>        尸九泉继续道:“只要得到了这棵树，你就有机会问鼎天下，争夺整个星空，建立如同天庭，这种可以驾驭群仙的势力，天上地下，唯我独尊！你只要放了我，我必会臣服于你，教你如何得到这棵树，如何使用！”
<br/>    
<br/>        陈明道:“那按照你这么说，不是根本没有来生？”
<br/>    
<br/>        尸九泉呵呵一笑，“你是蠢吗？若是死去的灵魂无限增长，世间哪里有地方可以容纳？而且，只要掌控了这棵树，就可以在一定程度之上将自己需要的灵魂，不想死去的人的灵魂提取出来，这便是英灵树的名字由来，英灵，永不阵亡！而这棵树，一旦到了成熟期，可是可以重塑英灵之躯的！让英灵复生的！”
<br/>    
<br/>        尸九泉继续诱惑道:“意思就是说，只要得到了这棵树，你，将永生不死！”
<br/>    
<br/>        然后陈明就一把直接掐断了尸九泉的脖子，将尸九泉的脑袋握在手里，尸九泉并没有死，毕竟是仙人，而是看向陈明:“你难道就不想得到这一棵英灵树吗？”
