　　　　这个夜晚注定对于白镇人民是不眠之夜，没有人舍得早早的入睡。
<br/>　　涂小安开始随意的到处闲逛，他喜欢极了这种双脚行走而脚踏实地的感觉。
<br/>　　非常的真实。
<br/>　　悄然的涂小安隐隐一想，不知道老妈跟老姐在哪里。
<br/>　　白镇那么好的日子，没有人会缺席，只是涂小安今天也没顾的上她们，另外今天的人实在太多了，可谓茫茫人海。
<br/>　　不是刻意的寻找，根本发现不了。
<br/>　　罢了，就让老姐陪着老妈好好过节日吧。
<br/>　　幻化人形，涂小安甚至还没告诉自己的老姐，也没任何人知道。
<br/>　　因为这副面孔对她们只是一个陌生人，知道与不知道并不重要。
<br/>　　终于一天，涂小安会真正的化形，到时候已自己的真面目出现在自己亲人的面前，岂不是更好。
<br/>　　漫无目的的逛着，一张张陌生面孔，都挂着善意的脸，很多人的脖子上也都挂着一条毒蛇，别的城市夜晚有人会出来溜宠物，基本都是遛狗。
<br/>　　但白镇，却是溜蛇，而且非常常态，更别说今天是蛇王节。
<br/>　　一个小摊面前，涂小安微微的一顿，一对情侣迎入他的眼帘。
<br/>　　“老公，白镇的蛇王节也太壮观了吧，原来他们信仰的蛇王是真实存在的”
<br/>　　一位面容娇美的美女一脸激动的的挽着一位年轻男子的手，看着小摊的小物件。
<br/>　　男子笑了笑，：“我也不敢置信，今天也算大开眼界了，看完这里，我们也赶紧去蛇王庙拜拜，想必现在人应该少一点了”
<br/>　　蛇王雕像入庙，络绎不绝的人上香，那排队简直是无边无际的。
<br/>　　很多人只能先等着，先逛着，等一大波人先上完香，第二波人再进去。
<br/>　　因为你就算是挤，也完全是挤不进去的。
<br/>　　娇美女子重重的点头：“今天我们真的赚了，不但都喝到了蛇王水，还领了一个蛇王雕像，到时候放家里好好的供着”
<br/>　　她继续道：“老公，说来也奇怪，我以前都是很怕蛇的，但现在不知道为什么，看到这满街的蛇，我一点都不怕，反而还觉得有点亲切”
<br/>　　男子摸了一下她的秀发：“想必是见多了就不怕了吧”
<br/>　　“老公，更奇怪的是，我现在不但不怕，我甚至还想养一条宠物蛇”女子忽闪忽闪眼眸的说，这个想法不知道为何，愈演愈烈。
<br/>　　男子闻言，也微微诧异，到也没多想：“这里是白镇，你想养条蛇还不简单吗，到时候我们买条带回家”
<br/>　　“嘻嘻，好呀，我还要养毒蛇呢”
<br/>　　“你呀，胆子越来越大了，还毒蛇，小心被咬哦”
<br/>　　边说边聊，这对小情侣从涂小安的身边擦肩而过。
<br/>　　涂小安深深的看了她娇美的女子一眼，微微一笑，有种由衷的高兴。
<br/>　　这对情侣不是别人真是舒瑶跟他的男朋友陈响。
<br/>　　这位蛇胎女子的恶梦因为涂小安将他们删除了记忆而结束，他们幸福的小生活得以继续。
<br/>　　如果他们的记忆还在，恐怕难以如此轻松的生活。
<br/>　　不好的记忆没了，那段经历也就从来不曾发生过。
<br/>　　舒瑶说，自己看到蛇不怕了，陈响的回答是她看多了蛇自然不怕。
<br/>　　非也，唯有涂小安知道舒瑶为什么不怕蛇了。
<br/>　　一个怀上蛇胎的女人，又怎会还怕蛇呢，她这辈子注定跟蛇有了不解之缘。
<br/>　　而从她肚子里爬出来的那条蛇，超强的变异体，成为蛇精一点压力都没有。
<br/>　　若是变异双头蛇以后忽然想认母，涂小安还真有点不知道怎么办。
<br/>　　这个想法涂小安之前没想过，看到现在的舒瑶到诡异的冒出来。
<br/>　　双头蛇现在看似血脉很高，但毕竟出生不久，心智不高，很多东西它还想不到。
<br/>　　可如果双头蛇成为蛇精，那便不一样了，心智也会迅速的提高。
<br/>　　几个念头闪过，这对小情侣也消失在人海之中。
<br/>　　涂小安连忙抛去那些怪异的想法继续闲逛，蛇王节有很多有趣的节目，应接不暇，绝对不会让你觉得无聊。
<br/>　　尤其是斗蛇，更是白镇的一大特色，涂小安自然不会放过，当然他也只是去看一看，不会在跟别人斗蛇。
<br/>　　因为这简直就是王者虐青铜，他系统中随便召唤一条蛇，就可以秒杀一切了。
<br/>　　热闹的白镇，热闹的夜晚，随着时间的流逝，这座千年古镇的人流量反而越来越大，很多临近地方的人，都纷纷赶到了白镇，来参加蛇王节。
<br/>　　这一天白镇有太多的不可思议的事情发生，科技资讯发达，很多消息都飞蛾扑火的传出去，吸引了不少人。
<br/>　　不断有新人来到白镇，看着满街的人与蛇，那是又惊又怕，但一阵惊怕之后，却露出了浓浓的新鲜意味。
<br/>　　这座古镇，还真的与众不同。
<br/>　　此时人群中多了一位鹰钩鼻的中年男子，他的样子有点邪派，让人一看，就不会产生特别好的印象。
<br/>　　他看到满街的蛇到没有什么特别的反应，反而是这满街的人流量，让他为之一喜。
<br/>　　好像是找到了梦寐以求的好地方。
<br/>　　他的神情兴奋中却不尽然的流露出一丝淡漠的神态，好似人命为蝼蚁，不把众生放在眼中。
<br/>　　“就是这里了，最适合我偷偷的作案”
<br/>　　作案？
<br/>　　要是被旁边人听到，还以为这个鹰钩鼻的男子是一个小偷，专门在人流量大的地方投机取巧，伺机下手。
<br/>　　但实则，将这位当做小偷，还真是冤枉跟屈才了。
<br/>　　他周身隐隐散发若有既无的能量波动，赫然，这是一位人类觉醒者。
<br/>　　身材开窍，突破桎梏，不同普通凡人，甚至还会因为身体觉醒，得到一些特殊的“超能力”手段。
<br/>　　这时，鹰钩鼻的男子来到了一个僻静的昏暗小角落，他从身上掏出一个非常别致的小黄袋，显得格外的慎重。
<br/>　　小黄袋打来，既然一抹抹白点的小东西齐刷刷的飞了出来，旋即，悬浮在鹰钩鼻男子的面前。
<br/>　　“宝贝，今天可以让你们美餐一顿了”
<br/>　　男子笑了，笑意中带着冷冽，再往人山人海的街道看了一眼，笑意更浓了。
