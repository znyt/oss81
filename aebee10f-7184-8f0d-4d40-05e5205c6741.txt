　　一团紫色血肉漂浮在半空，不停蠕动的肉瘤密布其上。

<br/>　　接着另一个方向再次鼓起一个包，然后再次平静。如此反复。好似有一个怪物在肉团之内左冲右突。正不断向肉团束缚发出冲击。
<br/>　　轰！
<br/>　　突兀的，半空中一声闷雷般炸响。肉团毫无征兆的爆炸，大片紫色血雾弥漫半空。
<br/>　　苏重皱着眉头飘在半空，身上衣服破破烂烂。虽然没受伤，但却到处沾满紫色血液，污浊异常。
<br/>　　“竟然跑了？”苏重注视天边，这一会儿早就没了蚁王的踪迹。
<br/>　　低头看向地面大坑，本来躺在深坑内部的尼飞比多也消失不见。
<br/>　　“断臂都带走啦？该死！”
<br/>　　尼飞比多的能力很特殊，能够控制傀儡般控制自己战斗，发挥出本体数倍的战斗力。同时也能把别人制作成自己的傀儡。
<br/>　　最重要的是，他的能力具有一定的医疗效果。特别擅长拼接残肢！
<br/>　　嵌合蚁生命力本就强悍，这意味着，尼飞比多很快就能完全恢复！
<br/>　　跑了蚁王，竟还跑了两个护卫！苏重脸色难看。如果不是尤匹，已经在刚才的自爆中消失殆尽，苏重绝不介意把它碎尸万段！
<br/>　　“跑了蚁王，剩下的就全部去死吧！”右手五指狠狠攥紧。
<br/>　　一直在匀速收缩的线牢，毫无征兆的速度暴涨。森林中，惨叫声骤然高昂。突兀加速的线牢，让嵌合蚁措手不及。毫无防备下，瞬间被线牢切成碎片！
<br/>　　……
<br/>　　嵌合蚁巢穴，尼特罗三人站在洞**，抬头打量笼罩天地的线牢。
<br/>　　有诺布的空间能力，生物专家很快到达。在寇鲁多带领下，众人顺利进入嵌合蚁巢穴。
<br/>　　等他们到来时，这里已经基本成了空巢。只有寥寥几个嵌合蚁选择留下来守护蚁后，或者说是选择脱离蚁王，并倒向人类一方。
<br/>　　这些大多是觉醒部分人类记忆，受到曾经意识影响的嵌合蚁。
<br/>　　虽然他们及时赶到，但蚁后伤势太重，最后不治身亡。另外，众人还在蚁王肚子里发现了另一个生命，一个幼小女婴。他被寇鲁多认为是蚁后遗产，新的王。
<br/>　　尼特罗接受了一众嵌合蚁的投降，并答应为在对方不作乱的情况下，给对方提供庇护。
<br/>　　处理完嵌合蚁的事情，三人这才有时间观察天空的线牢。
<br/>　　“这个家伙是谁？”诺布眉头始终皱着，黑红线牢一直给他强烈的压迫感。
<br/>　　“不过也算做了件好事。”莫老五嘴角翘起笑道：“没了蚁后和蚁王钳制，大批嵌合蚁溃散。如果真被他们潜入人类社会，必然会造成巨大混乱。现在好了，全给关笼子里去了。”
<br/>　　轰轰轰……
<br/>　　远方天空骤然闪现巨大黑红刀芒。惊天刀芒，即便隔得那么远，三人依旧能感觉到其中的强烈威胁。
<br/>　　“他变得更强了。”尼特罗嘿嘿笑道。
<br/>　　“您见过他？”诺布眼中带着隐隐的惊惧。看着远方天空那不断闪现的黑红刀芒，这才知道，当初对方杀嵌合蚁的时候，根本就没用全力！
<br/>　　“这家伙难道是在和蚁王战斗？”莫老五脸色突然一变：“会长！”
<br/>　　嗖！
<br/>　　等莫老五转头看向尼特罗时，却发现不知何时，尼特罗早就消失在原地。
<br/>　　莫老五释然一笑，自己能想到，尼特罗会长怎么会想不到。
<br/>　　“诺布，那个牢笼正在收缩，我们必须尽快离开这里。”只是看远方的战斗余波，就知道对方厉害。这个不断收缩的牢笼，很可能存在着某种未知危险。
<br/>　　当务之急，是赶紧带着投降的嵌合蚁离开。可还没等众人走出巢穴，尼特罗就已经返回。
<br/>　　“会长？”诺布有种不好的预感。
<br/>　　“啊。三大护卫有一个自我爆炸，蚁王趁机跑了。”尼特罗不无遗憾道。
<br/>　　他速度惊人，众人还没跑到线牢边界，他已经打了一个来回。也远远的看到了气急败坏的苏重。
<br/>　　“牢笼收缩速度在加快！”莫老五抬头盯着骤然加速缩小的线牢。
<br/>　　“好快！”诺布已经听见了线牢割裂空气的尖锐鸣叫。
<br/>　　不到一分钟，庞大的牢笼竟已缩小到直径五百米大小！
<br/>　　放眼望去，线牢所过之处，树木全部倒伏割裂。一片茂密森林，只剩下嵌合蚁巢穴周围五百米的范围。其他地方，全都各种碎块混合在一起，厚厚的铺在地面上。
<br/>　　从天空看下去，整个ngl森林深处，就好似多了个数十公里的麦田怪圈。

<br/>　　看那些被割裂的树木岩石，在想想刚才甚嚣尘上的惨叫声。不用看，他就已经猜到了森林中的惨况。这根本就是一面倒的屠戮！
<br/>　　看着巢穴下方惊恐聚集的嵌合蚁，尼特罗叹了口气：“老了，心也软了。”

<br/>　　嘣嘣嘣……
