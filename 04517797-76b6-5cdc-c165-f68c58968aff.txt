夏夕绾就站在这里，澄亮的目光落在他挺括的身影上，看着他在茫茫世界里疯狂寻找着她。
<br/>
<br/>        所有的记忆又像是回到了几个月前，他离开的最后一天，那样小心翼翼又卑微讨好的去爬树，送了她三颗鸟蛋，要跟她一起孵出小鸟来。
<br/>
<br/>        白皙的眼眶突然变红了，里面覆上一层晶莹的水雾，夏夕绾想哭又想笑，将自己弄得像个傻子。
<br/>
<br/>        这么一刻她才懂现在的陆寒霆，现在的陆寒霆就像是一个傲娇的小公举，他的心里总是矛盾又别扭，他明明就是喜欢她，他喜欢她啊。
<br/>
<br/>        夏夕绾突然觉得陆先生从来没有离开过，陆寒霆就是陆先生，陆先生就是陆寒霆！
<br/>
<br/>        这种认知充盈了她整个胸腔，夏夕绾飞奔过去，从后面一把抱住了陆寒霆精硕的腰身。
<br/>
<br/>        女孩柔软的身体从后面贴上来的那一瞬间陆寒霆高大的身躯倏然一僵，同时他紧绷的神经也松懈了下来，垂在身侧的两只大手拽了一下拳，他转过身，阴冷训斥的盯着她看，“夏夕绾，谁让你乱跑的，就因为我说了你两句，你就耍小性子跑出来吗，外面很危险你知道不知道？”
<br/>
<br/>        刚才他正在气头上，换成任何一个男人得知自己喜欢的女孩儿结过婚一时都接受不了的，他刚想冷静一下，但一抬头发现她竟然不见了。
<br/>
<br/>        她总是这样让人不省心！
<br/>
<br/>        夏夕绾仰着巴掌大的绝色小脸看着他，笑意盈盈道，“你关心我？”
<br/>
<br/>        什么？
<br/>
<br/>        陆寒霆这才发现自己的反应有点过激了，他抿了一下薄唇，一张俊脸顿时很难看，“没有。”
<br/>
<br/>        夏夕绾踮起脚尖，两只小手搂抱住了他的脖子，将自己黑漉又扑闪的大眼睛凑到他的面前眨巴眨，“你就有！你关心我！”
<br/>
<br/>        她不是疑问句，而是肯定句了。
<br/>
<br/>        陆寒霆最受不了她主动勾.引他的样子，现在她又玩什么把戏？
<br/>
<br/>        他伸手将她的纤臂给扯下来，“夏夕绾，别碰我！”
<br/>
<br/>        “那我…亲你，可以吗？”
<br/>
<br/>        夏夕绾凑上去，亲了亲他的薄唇。
<br/>
<br/>        少女身上那股清甜的体香侵袭而来，萦绕在了他的鼻翼下，她嘴巴甜甜的，像吃了什么奶糖。
<br/>
<br/>        陆寒霆不喜欢甜食，但是喜欢甜甜的女孩儿。
<br/>
<br/>        他高大的身躯都僵住了。
<br/>
<br/>        夏夕绾自然察觉到了他的反应，本来寒风吹散了一点燥热，但是碰上他，她又“蹭”一声被点燃了。
<br/>
<br/>        夏夕绾捧住他的俊脸，加深了这个吻。
<br/>
<br/>        她跟他结过婚，亲密的事情都做了，但是一般情况下他都是主动的那一方，霸道的掌控着节奏，她跟着就好了。
<br/>
<br/>        现在算是她少有的主动，她学着他的样子亲吻他。
<br/>
<br/>        陆寒霆很想拒绝的，垂在身侧的大掌松开，捏紧，再松开，可是就算他用尽了全身的力气都无法将她推开，舍不得推开。
<br/>
<br/>        他就这么僵着身体给她吻。
<br/>
<br/>        这时夏夕绾松开了他，她一张绝色的小脸烧的酡红，看着活色生香，她嗓音小小而娇俏的问他，“要不要…再亲亲了？”
