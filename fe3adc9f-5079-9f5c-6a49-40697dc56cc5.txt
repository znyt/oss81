        &lt;h3&gt;第三卷第二百三十八章  夏颜升职&lt;/h3&gt;
<br/>
<br/>        夏颜和呼延临庆在陈子昂公司挖过去的三个人因为涉嫌公司机密被律师李年起诉，在他们签署的保密合同里就有好几条对他们起到了制约。
<br/>
<br/>        协议规定的保密期限为自本协议签署之日起至双方合同关系终止或解除后五年内有效，其中对商业秘密的保密期限为施工期间及竣工后的无限期期间。
<br/>
<br/>        在保密期限内，乙方无论何种原因必须承担保密义务，且甲方不支付任何保密费。
<br/>
<br/>        刘清影他们三个人本来就是陈子昂公司里个骨干，尤其是三个技术人员，他们签署这份协议的时候就间接的承认了他们在离开公司五年之内不得从事相同的掌握着公司绝大部分信息机密的工作，但是他们辞职连几个月都不到，就去了呼延临庆的公司，这违反了他们当初签署的协议，李年律师动用了法律手段，四个人不能在呼延临庆的公司干下去了。
<br/>
<br/>        刘清影他们一走，呼延临庆的工作就进展困难了，他开始广纳人才，准备进行二次开发，可是时间不等人。
<br/>
<br/>        每一个项目都是有交工期限的，尤其是现在的招标项目越来越严谨，从上到下一切都会有监管机制。
<br/>
<br/>        夏颜是西城信息化的顶尖人物，找人她也是能找到的，项目还得继续，夏颜秘密的找了几个人将整个方案研究了一遍，要重新开发，陈子昂能做到的事情，她夏颜也会找到，况且方案和思路还是现成的。
<br/>
<br/>        “我说你给我认真点，这两个项目下来也赚不少钱，我给你找到了几个顶尖的人才，你自己带领着大家重新研发，就顺着他们的思路走，多给他们一些钱。”夏颜真是恨铁不成钢，她怎么就认识了呼延临庆这样的猪队友？
<br/>
<br/>        “交给我吧，你放心，我一定做好的。”
<br/>
<br/>        “呵呵呵呵，你给我说过多少次这样的话，最后还不是我跟在你后面收拾乱摊子，我就觉得我在养儿子一样。我到是挺佩服陈子昂那个怪物的，没有多少文化知识，竟然能领导那么多人才。”夏颜叹了口气，如果呼延临庆是陈子昂的话，那她是不是就会高枕无忧了？
<br/>
<br/>        “陈子昂现在的成就还不是你给的，你们可是调换了身份的。”呼延临庆不怕死的顶了夏颜一句。
<br/>
<br/>        “你说什么？”夏颜大怒，顺手将手中的书砸向了呼延临庆的脸，呼延临庆一个躲闪不及，胖胖的脸上划出了一道血痕。
<br/>
<br/>        “颜颜，你竟然动手打我？”呼延临庆捂住脸。
<br/>
<br/>        “滚吧你，看见你就烦，一点本事都没有，就知道拍马屁。”夏颜最近心烦意乱，就洛城项目上的事情已经有人开始针对她了。
<br/>
<br/>        尤其是哪个乔小东，每个项目他都盯得死死的，让她无计可施，目前她的身份是动不了乔小东的。
<br/>
<br/>        但是夏颜没有难过很久，她的职业生涯又迎来了人生的高峰期。
<br/>
<br/>        陆家老三陆胤杰到西城视察了。
<br/>
<br/>        蔡少芬之前就给夏颜打过电话，让夏颜抽时间去见见他们家老三，在陆家老三到西城的时候，蔡少芬又给夏颜打电话了
<br/>
<br/>        “颜颜，这次是你三叔过去了，我已经将你的情况给你三叔说了，你抽时间去见见他，你三叔在那边还是说得上话的。”
<br/>
<br/>        “谢谢蔡妈，您看我什么时候去找三叔合适？”
<br/>
<br/>        “你自己打电话联系吧，颜颜啊，我只有一个儿子一个女儿，就是嫌孩子太少了，以后你们一定要多帮扶着，你好好干工作吧，信息化这几年可是很吃香的。”
<br/>
<br/>        “您放心，今后您让我去那我就去那，你让我干什么我就干什么，我也是您的女儿啊。”
<br/>
<br/>        “那就好，那就好，你现在年纪也不大，正是大好时光，好好努力，尤其是记得要培养自己的人手。”蔡少芬谆谆教导，点点叮咛。
<br/>
<br/>        他们陆家确实是需要培养年轻人，陆玉霆是他们陆家培养的接班人，但是陆宇霆似乎不善权谋，心性太过善良，既好也不好，好的是他能得到很多人的明面上的支持，不好的是，在这样的环境下，如果陆宇霆没有助力就会被人碾压。
<br/>
<br/>        陆家老三到西城视察，参观了西城的信息化项目，而夏颜作为西城信息化一把手，是要全程陪同和讲解的，讽刺的是，陆胤杰参观的都是陈子昂手中的项目，这些项目有些还是陆胤杰在西城的时候陈子昂他们做的，陆胤杰对西城的发展做出了肯定，特别表扬了夏颜的功劳。
<br/>
<br/>        陆胤杰也参观了西城南郊新区的开发，给予了杨东他们表扬和肯定，希望他们带领西城走向更高的台阶。
<br/>
<br/>        江俞轩作为开发商也会全程陪同。
<br/>
<br/>        陆胤杰：“经常听人提起过江家大少啊，一个传奇的人物，这个南郊新区建设得这么好，也有你们很大的功劳啊。”
<br/>
<br/>        “谢谢陆总夸奖，还是领导领导有方，要不然我们能做什么？我们就是一个搬运工。”
<br/>
<br/>        “哈哈哈哈，说得好，你们就是一个搬运工，将这些大楼都建起来了，不错啊，江少如此低调，难怪张家那姑娘看上你了。”
<br/>
<br/>        “张家姑娘和江大少情投意合，不知道我们什么时候能吃上你们的喜糖呢？”夏颜笑眯眯的问。
<br/>
<br/>        陈子昂和江俞轩的事情如果传到张倩楠的耳朵里，张倩楠要是不在乎，她夏颜就不姓夏。
<br/>
<br/>        江俞轩记起来张家二爷让他们秘密结婚的事情：“大家都说我和张美女有关系，可是谁又看见过我们在一起？一切不过是捕风捉影罢了。”
<br/>
<br/>        “哈哈哈哈，年轻人谈了恋爱还要遮遮掩掩，真是有意思。”
<br/>
<br/>        “谈什么恋爱，我和她在一起上过学，是同学关系。”
<br/>
<br/>        “江总这么着急的想撇清和张家的关系，莫非真的和陈子昂有瓜葛？”夏颜故作吃惊的问。
<br/>
<br/>        “唉，夏总这么关心我，莫非是看上我了？今天可是陆总来视察，而不是谈论我的个人问题。”江俞轩笑着将话题岔开了。
<br/>
<br/>        马总在旁边听着夏颜和陆胤杰的一唱一和，似乎陆胤杰和夏颜很熟悉？
<br/>
<br/>        晚上一行人吃过饭，陆总对马总说：“夏颜的能力不错，将信息化做得这么好，她应该有更高的发展，你这个当领导的不知道推荐推荐？”
<br/>
<br/>        经过马总的极力推荐和跑路，夏颜的工作很快调动了，她发展的空间更大了。
