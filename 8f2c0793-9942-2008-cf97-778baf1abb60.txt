夏夕绾能感觉到陆司爵是爱这两个儿子的，所以在海城时，他不能允许她的存在，但是现在得知她是林水瑶的女儿了，林水瑶跟柳璎珞当年可是冠盖满京华的两姝，他想通过她找到柳璎珞，所以他放下了两个儿子，允许了她的存在。
<br/>
<br/>        这也就是说，在陆司爵心里，柳璎珞比这两个儿子还要重要的多。
<br/>
<br/>        夏夕绾第一次真正的意识到，陆司爵深爱柳璎珞。
<br/>
<br/>        柳璎珞已经消失很多年了，她的儿子陆寒霆今年都已经28岁了，夏夕绾不知道究竟是怎样的深爱才能抵挡住岁月的侵蚀和力量，让这个帝王般的男人一直守在这里，或许是守在他们相遇最初的地方。
<br/>
<br/>        可是，既然深爱，为何又要伤害？
<br/>
<br/>        当年柳璎珞怀上二胎，自己拿着刀亲自剖开自己的肚子，取出了婴儿，是何等的惨烈？
<br/>
<br/>        如果不是被逼到绝境，柳璎珞又怎么舍得丢下自己的儿子陆寒霆跳入大海，一去不回头？
<br/>
<br/>        柳璎珞真的还活着吗？
<br/>
<br/>        难道，柳璎珞当年的消失跟妈咪有关？
<br/>
<br/>        既然柳璎珞还活着，为什么不回来找陆寒霆？
<br/>
<br/>        陆司爵这么爱柳璎珞，又为什么娶了柳招娣，和柳招娣生下了陆子羡？
<br/>
<br/>        任何女人，都无法接受婚姻的出轨，肉体的背叛。
<br/>
<br/>        夏夕绾以为今天来一趟陆家会有所收获，但是跟城府极深的陆司爵博弈，她毫无胜算，反而更加迷雾重重。
<br/>
<br/>        不过夏夕绾知道了，肯定了，她一切问题的答案都在这里。
<br/>
<br/>        她妈咪从哪里来？
<br/>
<br/>        她妈咪现在在哪里？
<br/>
<br/>        她亲生的爹地是谁？
<br/>
<br/>        这一切的一切的，所有的答案都在帝都城！
<br/>
<br/>        这时陆子羡走了过来，“绾绾，你怎么站在这里？”
<br/>
<br/>        夏夕绾回神，澄亮的翦瞳落在陆子羡俊美的脸上，莞尔一笑，“哦，刚才遇到了伯父，聊了两句。”
<br/>
<br/>        陆子羡清冷的黑眸看了一下陆司爵消失的方向，“我爸跟你说了什么？”
<br/>
<br/>        “没说什么，就随便聊聊，子羡，你送我回家吧，现在很晚了，夏爸蓝妈还在家里等我呢。”
<br/>
<br/>        陆子羡看着夏夕绾那双因为聪慧而闪烁着耀眼碎光的翦瞳，她总是这样，勇敢独立，可以自己想出很多办法来解决问题。
<br/>
<br/>        她独独没有想到依靠他。
<br/>
<br/>        什么时候她会将自己的小脑袋抵靠在他的肩上，寻找他的庇佑？
<br/>
<br/>        陆子羡温暖的勾起了薄唇，然后将自己白皙漂亮的大手递了过去，“走，绾绾，我送你回家。”
<br/>
<br/>        夏夕绾将自己纤白的小手递到了他的掌心里，陆子羡紧紧的握住，带着她回家。
<br/>
<br/>        这时陆寒霆从洗手间里出来了，他脸色阴鹜薄寒的伫立在二楼的雕花栏杆处，低眸俯视着下面的两个人。
<br/>
<br/>        他深如寒潭的狭眸很快就落在了陆子羡和夏夕绾紧紧交握的手上，他的目光像淬了毒的冰，恨不得现在就冲下去将这两个人给分开。
<br/>
<br/>        刚才她那么一顶，他现在还隐隐作痛，她一点都不关心他的身体，反而掉头就跟别的男人走了。
