　　现在楚风眠的实力，放在北域中，都足以称得上是一方大人物了。
<br/>　　要是楚风眠愿意，他回到北邙学院去展现实力，足以一跃之间晋升为种子弟子，甚至是晋升核心弟子，都有可能。
<br/>　　“七大宗门，距离报仇，更进一步了！”
<br/>　　楚风眠眼神中，露出几分杀意。
<br/>　　他的实力越强，楚风眠的心中杀意也就越是翻腾。
<br/>　　但是这实力，还不够。
<br/>　　七大宗门中，超越生死境的老古董，也有着许多。
<br/>　　就算是在北域中，各大势力也并非是没有超越生死境的老古董，这种老古董若是出手，楚风眠肯定必败无疑，甚至是都有可能有生命危险。
<br/>　　他的实力，必须要更强才行，现在楚风眠所缺少的，就是积累，等到楚风眠真正度过生死大劫，晋升生死境。
<br/>　　在这北域中，他也就可以真正足以肆意纵横了。
<br/>　　“不过以现在的实力，对付陆家，也不再是没有毫无还手之力了，欺压了我那么久，也该报仇了。”
<br/>　　楚风眠眼神中透出一丝阴冷。
<br/>　　他在北邙学院手，受到无数针对，之前外门任务考核的时候，都差一点因为陆家的原因，令他吃了一个大亏。
<br/>　　还是幸好有着白萱到来，才帮助楚风眠解围的。
<br/>　　但是这个仇怨，楚风眠可从未忘记过，等他这次回到北邙学院，就是去报仇的时候。
<br/>　　不止是陆家，还是夜天君，燕古等人，敢去打寒月漓主意的人，这些人，楚风眠迟早都要将他们，彻底毁灭。
<br/>　　回到北邙学院，便是楚风眠真正崭露头角的时刻。
<br/>　　“风千年，焚天绝，这两尊大人物，居然都死了。”
<br/>　　“此人到底多么强大，生死境高阶的人物，都能斩杀！”
<br/>　　“看来北邙学院，又一个夜天君崛起了。”
<br/>　　无数武者，一直到现在都没有缓过神来，风千年，焚天绝，沧澜逆。
<br/>　　这三尊可都是生死境高阶的大人物，他们也都有所听闻过的存在，今日就在他们的面前陨落，谁都想不到是这个结果。
<br/>　　就连白萱，都想不到，楚风眠的实力，居然会达到这一步。
<br/>　　在北邙学院的时候，楚风眠的实力，也就是跟她在伯仲之间，在楚风眠踏入到神力境后，便是彻底的超越了她。
<br/>　　但是如今，楚风眠的实力居然是再一次的突飞猛进，生死境高阶的人物，都可以直接轰杀。
<br/>　　“太恐怖了，这种实力，已经不是人了。”
<br/>　　白萱心有余悸的想到，心中也是庆幸，幸好她跟楚风眠关系极好，要是跟楚风眠这样的人作为敌人，才是真正的噩梦
<br/>　　这每个人的想法，楚风眠都懒得理会，他看向这颜古龙帝洞府处，布置出的禁制。
<br/>　　“破！”
<br/>　　楚风眠淡然吐出一个字，这上面布置的禁制，便是开始破碎，猛然崩塌。
<br/>　　这颜古龙帝洞府的出口，便是再一次被打开。
<br/>　　“走，我们快走！”
<br/>　　无数武者，纷纷向着这出口飞去。
<br/>　　在这颜古龙帝洞府中，他们都有所收获，谁也不想在留在这里面，纷纷遁光离开。
<br/>　　“我们也走。”
<br/>　　那些三大势力的弟子，看到这禁制被破，也都是对视了一眼，化为了一道遁光，想要离开。
<br/>　　但是就在他们快要走出洞府的一刻，一道身影，却是直接拦住了他们。
<br/>　　“你们，将所有的空戒，统统交出来，才能离开！”
<br/>　　楚风眠冷冷的开口道，看向那三大势力的弟子。
<br/>　　其他人可以离开。
<br/>　　但是这三大势力的人，可不能离开。
<br/>　　“什么？”
<br/>　　“让我们交出空戒？”
<br/>　　一些三大势力的弟子，看向楚风眠，都是含怒开口。
<br/>　　他们三大势力，在北域中，可都是有着不小的地位，起码就算是北邙学院的人，也不管太针对他们。
<br/>　　他们可以打劫别人，但是还没有人敢来打劫他们。
<br/>　　“小子，你未免太过分了吧……”
<br/>　　一尊不怕死的弟子，似乎也如同平日里的嚣张，对着楚风眠开口道。
<br/>　　但是他开口的下一刻，声音却是戛然而止。
<br/>　　一道灵力，已经是轰击了他的身上，顿时此人，便是当场被轰杀，一枚空戒，也是落入到了楚风眠的手中。
<br/>　　“违令者，死！”
<br/>　　看到此人被楚风眠轰杀，三大势力的不少人，才缓缓的醒悟过来。
<br/>　　楚风眠可不是他们平日里遇到的那种人，会被三大势力的名声，都给吓得不敢出手。
<br/>　　就连沧澜逆，风千里，焚天绝这样的大人物，都死在了楚风眠的手中。
<br/>　　他们这些人，又算得了什么。
<br/>　　“交。”
<br/>　　“我交。”
<br/>　　几位三大势力的弟子，犹豫了一下，都是将空戒，直接拿了出来，扔了过来。
<br/>　　楚风眠接过，便是点了点头，让这几个人离开了。
<br/>　　但是大部分的三大势力弟子，却都依然不愿意，将空戒交出来。
<br/>　　这空戒中，可是他们所有的收藏，没有任何人，愿意白白交出来。
<br/>　　“该死！我就不信他一个人可以将我们全部杀了！”
<br/>　　“逃！能逃出去一个是一个，他不过就是一个人罢了！”
<br/>　　“对，我们能够逃出去，为何要听他的话！”
<br/>　　几名三大势力的弟子，对视了一眼，猛然化为了无数遁光，直接便是向着外面冲了过去。
<br/>　　这一道道遁光，分散无比，似乎是要让楚风眠，根本无法阻拦。
<br/>　　“不知死活！”
<br/>　　楚风眠冷哼一声。
<br/>　　他已经给了这些三大势力弟子一个机会了。
<br/>　　既然他们不懂得珍惜。
<br/>　　那就是在自己找死了。
<br/>　　“都给我死！”
<br/>　　楚风眠催动灵力，大手一挥，这无数遁光，突然都在空中，猛然炸裂开来。
<br/>　　这以一尊尊三大势力的弟子，都在楚风眠的灵力下，纷纷身躯炸开，当场陨落。
<br/>　　仅仅一瞬，这所有三大势力的弟子，全部都成了死人，直接灰飞烟灭，没有任何一点留下的。
<br/>　　而在楚风眠的手中，也多了无数空戒。
<br/>　　就算是楚风眠拥有血海，财富巨大，但是谁又会觉得自己的财富多。
<br/>　　这些三大势力的人既然自己送钱来，那楚风眠当然要收下。
<br/>　　“走吧。”
<br/>　　楚风眠看了一眼白萱，身形一动，两人身形一动，直接离开这颜古龙帝的洞府中。
