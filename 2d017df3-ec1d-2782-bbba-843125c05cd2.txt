因为夏夕绾的关系，叶翎非常讨厌夏妍妍，所以当叶翎第一次看到她和顾夜瑾站在一起时，就跟顾夜瑾闹了很大的性子。
<br/>
<br/>        少年时的顾夜瑾就很受女孩子欢迎，会收到很多礼物和情书，那时叶翎还小，还曾经帮着那些女孩子将情书送给过顾夜瑾。
<br/>
<br/>        叶翎唯一不能忍的就是顾夜瑾跟她在一起，看一次就闹一次。
<br/>
<br/>        夏妍妍还记得那一年顾夜瑾生日，她去参加了生日party，party上主角顾夜瑾却消失了，大家都在找他。
<br/>
<br/>        夏妍妍找到了女洗手间，在洗手间外她就听到了叶翎的哭闹声，“哥，你明知道…我不喜欢夏妍妍，是她…害了绾绾，我不想…看到她…”
<br/>
<br/>        她听到了顾夜瑾低沉暗哑的嗓音，带着哄宠，“翎翎别哭了，哭的上气不接下气，哥哥会心疼…”
<br/>
<br/>        “哥哥骗人，那你为什么…还让夏妍妍来参加你的生日party，我不喜欢你们在一起，不要你们在一起，都是她害了绾绾…”
<br/>
<br/>        “绾绾绾绾，你不喜欢夏妍妍究竟是为了我，还是为了夏夕绾，希望你自己去想清楚，如果没有听到我想要的答案，你就会一直看到夏妍妍！”
<br/>
<br/>        顾夜瑾突然摔门而去。
<br/>
<br/>        夏妍妍对那一天的事情记忆犹新，在她印象里，叶翎很喜欢顾夜瑾，但是仅限于对哥哥的喜欢。
<br/>
<br/>        毕竟那时的叶翎还太小，一直被顾夜瑾庇佑在羽翼里，连跟男生接触的机会都没有，她还不懂什么叫喜欢。
<br/>
<br/>        至于顾夜瑾的心思…
<br/>
<br/>        这时霍璇冷哼了一声，“夜瑾哥，叶翎是不是又吃醋了，她每次都这样，需要你去哄她！”
<br/>
<br/>        顾夜瑾勾了一下薄唇，溢出几分寒芒，她不会了，现在她看到他跟夏妍妍在一起都不会闹了。
<br/>
<br/>        这时“叮”一声汽车鸣笛，顾夜瑾侧眸，一下子就看到了疾驰过来的那辆跑车。
<br/>
<br/>        红色的法拉利跑车，叶翎坐在驾驶座上，载着夏夕绾，她拿下了鸭舌帽，巴掌大的小脸上挂了一副宽大的墨镜，整个人明艳似火。
<br/>
<br/>        夏妍妍和霍璇也看到了叶翎，以前的叶翎真的是不识方向的，除了会跟顾夜瑾闹夏妍妍的事情外，叶翎粉粉糯糯的小团子，很乖巧很听话，像是温室的玫瑰花没有经历任何的风吹雨打。
<br/>
<br/>        这两年叶翎变了很多很多，整个人气场开了，现在开着跑车的她明艳水媚的格外张扬，烈焰似火。
<br/>
<br/>        顾夜瑾的眸子里都是叶翎那张扬飞舞的茶色卷发，这时夏妍妍突然尖叫道，“夜瑾，快停车，前面是红灯！”
<br/>
<br/>        一道尖锐的刹车声响起了，顾夜霖紧急停了车。
<br/>
<br/>        夏妍妍和霍璇吓得脸色惨白，差一点就闯红灯了，这里车流很急，会车祸的。
<br/>
<br/>        顾夜瑾抬眸，那辆法拉利跑车已经转弯而去了，驾驶车窗缓缓滑了下来，他见那只刚做了鲜红美甲的小手伸了出来，缓缓竖起了中指。
