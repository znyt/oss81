　　双生武魂，在炎龙大陆属于传说。只听说过存在，却不曾真正有人见过。
<br/>　　至于紫色武魂，那也是稀罕到极点的东西。
<br/>　　哪怕是萧家中的长老，也大多是黄阶武魂罢了。
<br/>　　能拥有蓝色武魂，已经是整个大陆都有名的天才武者了，更别说最高级的紫色武魂了。
<br/>　　“试试效果如何。”萧逸暗暗想了想。
<br/>　　忽然，房门外传来一阵轻微的脚步声。
<br/>　　“嗯？”萧逸耳朵一动，作为一个杀手，对周围的环境变化极为敏感。
<br/>　　“有人来了。”萧逸立刻收住了想试一试武魂的想法，在床上躺好，并假寐起来。
<br/>　　如果来者是敌人，他便能让敌人放下警惕，自己能有足够的时间和机会对付敌人。
<br/>　　吱呀一声，房门被打开。
<br/>　　萧逸用余光瞥了一眼，进来的是一个女子。
<br/>　　女子一身布衣，背对着，并随手关好了门。
<br/>　　没看到女子的样貌，但看这背影，那柔弱的身子，轻盈的玉步，想必是个美人。
<br/>　　“哎，萧逸少爷什么时候能醒呢。”
<br/>　　女子的声音很好听，如银铃般清脆悦耳，声音中充满了担忧与关怀。
<br/>　　女子手中捧着一碗药，来到床边。
<br/>　　萧逸意识到女子并非敌人，微微睁开了眼睛。
<br/>　　几乎是他睁开眼睛的瞬间，眼睛瞪得老大。
<br/>　　“我草，哪来的丑八怪，胆敢吓本少爷。”萧逸看着女子丑陋的样貌，吓了一跳，差点没忍住一拳轰过去。
<br/>　　原来这女子并非是什么美人，相反很普通，甚至有些丑，脸上青一块红一块，看着怪吓人的。
<br/>　　“阿。”女子惊呼一声，下意识地想躲开。
<br/>　　但她手里正捧着药，她马上反应了过来，没有再躲，仅仅是侧过了脸，怯生生地闭上了眼睛。
<br/>　　萧逸的反应也很快，立马收住了拳头。
<br/>　　他不是个喜欢以貌取人的人，也不会去歧视别人。
<br/>　　只不过刚才心里的想法和女子的样貌有了巨大的落差，这才让他差点儿打出一拳罢了。
<br/>　　“为什么不躲开？”萧逸收回了拳头，沉声问道。
<br/>　　女子见萧逸没有打她，这才睁开了眼睛，脸上仍旧带着害怕之色。
<br/>　　“这是萧逸少爷救命的药，不能弄洒了，萧逸少爷要快点好起来。”女子怯怯地说道。
<br/>　　“你就不怕被我打伤？”萧逸皱眉道。
<br/>　　女子摇了摇头，道，“依依的命是少爷的，少爷打就是。”
<br/>　　“你...”萧逸一愣，这时，他看清了女子的面容。
<br/>　　其实，她只是个少女，五官和脸部长得非常精致，细细去看的话，非常耐看。
<br/>　　如果不是她长得面黄肌瘦、脸上又青一块、红一块得，绝对是个绝色美女。
<br/>　　“阿。”少女忽然反应过来，道，“萧逸少爷你终于醒了，我得赶紧告诉三长老去。”
<br/>　　说罢，少女小心翼翼地将药碗放到桌面上，跑着离开。
<br/>　　萧逸通过脑海中的记忆得知。
<br/>　　这个少女名叫依依，今年15岁。
<br/>　　自幼父母双亡，萧逸的父亲看她可怜，便买回来给萧逸当贴身丫鬟。
<br/>　　小时候的依依长得非常可爱，性格乖巧懂事。
<br/>　　只不过萧逸是个典型的废物，每每在外头受了欺负，就会回来将气撒到依依的头上。
<br/>　　依依经常受到他的虐待，吃不饱，穿不暖下，自然变得面黄肌瘦的。
<br/>　　而且，就在萧逸被推下山崖的前一天，萧逸还毒打了她一顿，所以她脸上现在还青一块、红一块的。
<br/>　　“唉。”萧逸叹了口气，对自己刚才的行为感到后悔。
<br/>　　“老子一世英名，怎么就附身到这么个废物子弟身上了。”萧逸头一次对这副身体感到厌恶。
<br/>　　不多时，一个中年人便火急火燎地来到了房间。
<br/>　　萧家三长老萧重，曾经是萧逸父亲的心腹，如今担任代理家主之职。
<br/>　　年幼的萧逸，父亲神秘失踪后，便是由这位三长老抚养长大的。
<br/>　　“醒了，真的醒了，太好了。”萧重急得语无伦次，脸上止不住地喜意。
<br/>　　“要是逸儿你有什么三长两短，我日后如何与家主交代阿。”萧重坐到了床边，认真地打量着萧逸。
<br/>　　当看到萧逸气息平稳，没有大碍时，一直悬着的心才放了下去。
<br/>　　萧逸的父亲已经失踪多年，但萧重始终照顾着萧逸。
<br/>　　这时，萧重似是想起了什么，脸上的喜意忽然变为愁容。
<br/>　　“唉，逸儿，你怎么就这么不小心呢，在后山瞎逛，也能掉到山崖下去。唉，你昏迷的这些天，家族里...”
<br/>　　萧重一脸愁容，想说些什么，但看着萧逸的样子，却又忽然停下了话语。
<br/>　　“算了，你刚刚醒来，还是多休息休息吧，其他长老还等着我回去主持家族会议。”
<br/>　　说罢，萧重从怀中拿出一袋东西，放在桌面后，深深地看了萧逸一眼，才缓缓离去。
<br/>　　萧逸明显注意到，萧重的眼光很复杂，有关心、有慈爱，但更多地却是忧愁。
<br/>　　待萧重离去，萧逸才对一旁的依依问道，“依依，怎么回事？”
<br/>　　依依是随萧重一起回来的，此时正细心地吹着药，药太热了，吹暖了，好给萧逸服用。
<br/>　　听到萧逸的问话，依依抬起头来，表情有些踌躇，支支吾吾地。
<br/>　　“到底怎么了。”萧逸皱眉问道。
<br/>　　依依犹豫着，低声关怀道，“少爷你刚刚醒来，身子又虚弱，待会儿喝了药休息吧。家族里的事，三长老会处理好的。”
<br/>　　“说吧。”萧逸道。
<br/>　　“这...“依依低下头，为难地捏着自己的衣角。
<br/>　　“说！”萧逸呵斥了一句。
<br/>　　依依打了个激灵，拗不过萧逸，只能如实道，“少爷你昏迷的这几天，娇儿小姐前来拜访了，并退了你的婚事。”
<br/>　　依依怕萧逸承受不住打击，连忙补了一句，“少爷您昏迷的几天，所有人都觉得你没救了，娇儿小姐也是为自己的幸福着想罢了。现在你醒了，只要告诉娇儿小姐，她一定会改变主意的。”
<br/>　　“呵。”萧逸笑了笑，淡淡道，“我以为是什么事，她才不会改变主意，而且，那种恶毒的女人，退了婚才好。”
<br/>　　“阿？”依依吃了一惊，暗想，少爷不会是摔坏脑子了吧？娇儿小姐可是全城最漂亮的女子，慕容家的千金小姐阿，哪个男人不心动。
<br/>　　紫云城三大家族，萧家、慕容家、江家。
<br/>　　“还有其它事吗？”萧逸继续问道。
<br/>　　依依回答道，“还有家族里的长老们，说要废了您的少家主之位。”
<br/>　　萧逸恍然。
<br/>　　以前的萧逸，天赋低微、性格窝囊，根本不配当少家族，所有长老都看不起他，对他很是不满。
<br/>　　这一次，自己去后山瞎逛也能率下山崖，简直是废物中的极品，传出去，让整个萧家蒙羞。
<br/>　　长老们自然不会再让他在少家主的位置上坐下去。
<br/>　　“所谓无风不起浪，这事肯定有人在推动吧。”萧逸冷笑一声。
<br/>　　他知道就算自己说出事实，自己是被萧若寒推下山崖的，萧家也不会有任何人相信。
<br/>　　毕竟，一个是废物，家族耻辱。
<br/>　　一个是家族天才，未来前途大好。
<br/>　　“少爷，药可以喝了。”依依将药拿起，准备喂萧逸服用。
<br/>　　恰在此时，门外响起了一阵急促的脚步声和喧哗声。
<br/>　　“萧逸那个废物有够命大的，从后山摔下去都死不了。”
<br/>　　“死不了正好，那个废物死了，我们以后可没那么多乐子了。”
<br/>　　“....”
<br/>　　一阵阵嘲讽且不屑的话语传来。
<br/>　　“嘭”的一声，房门被粗暴地踢开，两个少年走了进来。
<br/>　　“萧杰、萧石。”萧逸皱了皱眉头。
<br/>　　萧杰，七长老之子；萧石，八长老之子。
<br/>　　算起来，两人还是萧逸的表哥。
<br/>　　不过，他们可从没当萧逸是表弟，处处欺凌他，抢他的修炼资源。
<br/>　　以前，轻则辱骂不休，处处捉弄；重则殴打一顿，将萧逸打得浑身是伤。
<br/>　　“果然，三长老将修炼资源给这废物了。”萧杰一眼看到了桌面上那袋东西。
<br/>　　萧家子弟，每个月都能得到家族给予的银两和丹药，以助子弟们更好地修炼。
<br/>　　“立刻给我滚出去。”萧逸冷冷地盯着二人。
<br/>　　“哟呵，小废物，长脾气了。”
<br/>　　“哈哈，这窝囊废该不会是摔下山后，把脑子摔坏了吧。”
<br/>　　萧杰和萧石不屑地嘲讽着，盛气凌人地朝萧逸走去。
<br/>　　他们已经习惯了抢夺萧逸的修炼资源，习惯了欺凌萧逸。
<br/>　　“你们两个大坏蛋，别想伤害萧逸少爷。”依依一把挡在床边。
<br/>　　依依随手拿了根床边的棍子，胡乱挥舞着，试图吓退萧杰和萧石，并保护‘弱小’的萧逸。
<br/>(本章完)
