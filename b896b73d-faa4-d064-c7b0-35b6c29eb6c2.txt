　　一切都过去了，那段最开心的日子，对于自己倒没有什么，毕竟这是自己选择的道路，但是对于少阳……
<br/>　　道风静静地看着这些小道士，他们稚气未脱，但是朝气蓬勃，他们都是法术界最普通的组成，他们之中，将来大部分都会泯然众人，但必然也有一些优秀的人才会脱颖而出，扛着法术界前进。
<br/>　　一代一代，就是这样下去的。只不过，当下需要扛着法术界前进的，是自己和少阳、四宝这些人。
<br/>　　从这些小道士身边走过，一路来到后山。
<br/>　　在后山一排道观的下方，耸立着一座玲珑宝塔，在月光下反射出清冷的光。宁静而肃穆。
<br/>　　玲珑塔门后的一块空地上，道渊真人正盘膝打坐，全身一动不动，脸上的皱纹沟壑使他看上去像是一尊古老的雕像。
<br/>　　道风没有着急走过去，而是在一边静静地等着。
<br/>　　道渊真人总算结束了一个周天的吐纳，长出了一口气，站起来，说道：“是道风吧？”
<br/>　　这又不是武侠小说，没有那么牛逼的人，不看人就知道谁来了，道渊真人又不是神仙。道风有些好奇地问：“你怎么知道是我？”
<br/>　　“我托无生给我带话找你，除了你，没人会来找我，我已经老得被法术界遗忘了。”道渊真人仰望着星空，喃喃说道，语气中有些落寞。
<br/>　　道风道：“在我印象中，你好像就没年轻过。我小的时候，你就已经这么老了。”
<br/>　　“什么话。”道渊真人瞪他一眼，“我比你师父大不了多少，最多也就二十岁。”
<br/>　　“别人都说你有几百岁。”
<br/>　　“几百岁，哈哈。”道渊真人笑起来，捋着胡须，“我是晚清的人，到现在也不过一百多岁……几百岁，那不成仙了。”
<br/>　　“你叫我来找你，有什么事。”道风不是喜欢闲聊的人，刚已经闲聊过几句，开始切入正题了。
<br/>　　道渊真人从地上扔了个蒲团给他，让他坐在自己对面，从身边拿起一个很老的茶壶，说道：“喝吗？”
<br/>　　道风摇摇头。
<br/>　　道渊真人自己倒出一股乳白色的液体，道风一看愣住了，“这是什么？”
<br/>　　“牛奶。”
<br/>　　“牛奶？你怎么喝牛奶？”
<br/>　　道渊真人很鄙视地瞥了他一眼，“你以为呢，我在这守塔，就该过青灯黄卷的生活？牛奶很有营养的，要不是营养好，我怎么可能活到这把年纪。”
<br/>　　“好吧，你赢了。”道风无语。
<br/>　　道渊真人喝了一口牛奶，抹了抹沾在胡子上的奶渍，望着道风，慢悠悠地说道：“你是在哪里遇到无生的，我听说他去了茅山。”
<br/>　　道风只好把茅山上的经历讲了一遍。道渊真人听完，表情也没什么变化，只是沉吟了许久，喟叹道：“你杀了法术公会的人，麻烦很大。星月奴不会放过你。”
<br/>　　“兵来将挡，也没什么关系。”
<br/>　　“星月奴……你对她知道多少？”
<br/>　　道风摇摇头，就算是他，对法术公会的了解也不多。
<br/>　　“星月奴，我跟她打过交道，大概……嗯，快有一百年了，那时候我还很年轻，跟你现在差不多……她济南对付，你未必是她对手。”
<br/>　　“没什么，惹都惹了，我也不怕。”
<br/>　　“这个是你自己的事，我找你来，是想问你有关叶少阳的下落。”
<br/>　　道风没把道渊真人当外人，对这尊法术界的活化石，道风内心也是尊敬的，觉得他很有原则，很纯粹，不然也不会跟他说这么多，于是把情况跟他说了。
<br/>　　“民国……一九二二年……”道渊真人的脸上第一次出现了波澜，沉默良久，说道：“不瞒你说，我最近做梦，总是梦到年轻时候的一些事，所以找你来……”
<br/>　　“我不会解梦。”
<br/>　　道渊真人不理他，继续说道：“我在梦里做那些事的时候，身边多了一个人，就是叶少阳。”
<br/>　　道风微微一怔，隐约想到什么，问道：“你梦里是哪一年？”
<br/>　　“大概就是二十年代初，具体不知道年份。”
<br/>　　道风目光闪动，想了一会，说道：“我觉得，很有可能少阳在那个时代，见到了当时的你。”
<br/>　　“当时的我……”道渊真人一时间思维转不过弯，询问道风是什么意思。
<br/>　　道风自己也说不清楚，只能给他分析了一下，道渊真人听完，久久说不出话。
<br/>　　“那……我应该怎么办？”第一次经历这种事，道渊真人自己也没了主意。
<br/>　　“不知道，好像什么都做不了，不过你可以多注意梦中的经历，争取找到一些有用的信息，可以用来分析的。”
<br/>　　道渊真人沉吟着点了点头。
<br/>　　这件事虽然离奇，但是光讨论是没结果的，两人也都不是磨叽的人，谈论了一会之后，也就放下了这个话题。道渊真人话锋一转，说道：“我找你来，是为一件事，最近星月奴的一个弟子来找我，希望我能站出来，跟法术公会搞好关系，让他们插手这次龙华会。”
<br/>　　道风看着他，没说话。
<br/>　　“我给拒绝了。”道渊真人的答复，在道风的意料之中。
<br/>　　“不过，他们告诉我一个信息，他们愿意在龙华会上提供轩辕剑，作为取胜者的信物，奖励给对方……道风，你可知道轩辕剑是什么？”
<br/>　　道风点头道：“略有耳闻。”
<br/>　　道渊真人神秘地笑了笑，道：“你听过这句话么，得先天，凑五器，开地图，太阴入。想要进太阴山，要凑齐五把各自属性不同的剑……目前叶少阳已经得到了三把，龙泉剑属水，松纹剑属木，鱼肠剑属火……这把轩辕剑，也是五大仙剑之一，五行属土，还有一把剑，我却不知是什么。”
<br/>　　道风眉头皱起来，说道：“等等，鱼肠剑为什么五行属火，不应该是属水吗？”
<br/>　　大凡神器，都有灵性，必须找个名目来取名，五行上也是各自有属性，掌握并且顺应这种属性的话，必然能将法器发挥出更大的威力，七星龙泉剑五行属水，因此操控它的剑诀，大多跟水有关。
