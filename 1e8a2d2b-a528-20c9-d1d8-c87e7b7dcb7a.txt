　　苏莫回到了客栈之中，静静的等待。
<br/>　　他不知道那位沁儿姑娘会不会来，但是，他必须等待一段时间。
<br/>　　此女，是他进入人王宫的唯一契机。
<br/>　　至于那个褚彦，苏莫没有在意，挑梁小丑而已。
<br/>　　若不是他身份特殊，不敢高调，此人他绝不会轻饶。
<br/>　　但是现如今，他只能低调行事，不能引起太大的波澜。
<br/>　　叹了口气，苏莫一边留意着外界，一边静静的参悟吞噬奥义。
<br/>　　时间一晃，便过去了三日。
<br/>　　沁儿依旧没有来，苏莫也不着急，安心的等待。
<br/>　　同时，他决定，再等三日时间，若是对方再不来，他只能再想其它办法了。
<br/>　　“咚！咚！咚！”
<br/>　　又过去了一日，苏莫的房门外，突然响起了低沉的敲门声。
<br/>　　苏莫神识一探，发现门外是一名中年人，一身锦袍，面容古板。
<br/>　　苏莫睁开了眼眸，双眸一眯，因为此人，是刚刚从人王宫出来之人。
<br/>　　他一直在注意着人王宫，绝对不会认错。
<br/>　　苏莫起身，打开了房门，看向眼前的中年人。
<br/>　　“不知前辈有何事？”苏莫沉声问道。
<br/>　　“你叫段惊天？”中年人打量了苏莫一番，淡淡的问道。
<br/>　　“正是！”苏莫点头头。
<br/>　　“那好，跟我走吧！”中年人不容质疑的说道。
<br/>　　“去人王宫吗？”苏莫轻声问道，他心中清楚，此人来应该就是邀他去人王宫的。
<br/>　　想必，沁儿没有亲自来，却是让别人来了。
<br/>　　“嗯！”
<br/>　　中年人点头，继续道：“不过，事先和你说清楚，即便你是沁儿小姐举荐，也要经过天赋测验，若是你天赋不佳，还是不能进入人王宫！”
<br/>　　“明白了！”苏莫点了点头，心中却是暗松了一口气。
<br/>　　看来，沁儿姑娘果然没有对他置之不理，安排了别人招他入人王宫。
<br/>　　“嗯！走吧！”
<br/>　　中年人招呼一声，便转身向外走去，苏莫立刻跟上。
<br/>　　随后，跟着中年人，苏莫进入了人王宫之中。
<br/>　　人王宫内，宫殿连绵，如同皇家园林。
<br/>　　苏莫跟着中年人，来到了一座恢弘的宫殿之中。
<br/>　　宫殿中人不多，中年人带着苏莫进入了一间无人的偏殿之中。
<br/>　　“段惊天，告诉本执事，你的身份背景！”中年人坐在了偏殿的主座之上，并未招呼苏莫落座，沉声问道。
<br/>　　“在下来自天池神朝，终极剑域，一阶散修！”苏莫沉声说道。
<br/>　　“你为何想要加入人王宫？”执事再次问道，这些事情都是例行询问。
<br/>　　之前，沁儿和他说了这个段惊天的大致情况，他倒是不担心这个段惊天有什么企图。
<br/>　　毕竟，加入人王宫，只是从最低微的弟子做起，没有巨大的成就、天大的贡献，不可能接触到人王宫的核心。
<br/>　　“原因很简单，只有在强大的势力之中，才能有所成就！”苏莫面不改色。
<br/>　　“终极剑域的终极神王殿，实力并不比人王宫弱多少，你为何来到轮回神朝？”中年人再次问道。
<br/>　　“我在终极剑域，曾经和一些终极神王殿的弟子有仇，所以并不想加入终极神王殿！”苏莫沉声说道，不卑不亢。
<br/>　　“嗯，你的天赋如何？拥有什么元神？”中年人再次问道。
<br/>　　其实，他已经知道苏莫的天赋不会差，毕竟，他能看出来，苏莫的年纪不足百岁。
<br/>　　不足百岁的年轻人，有虚神境四变的修为，无论在哪里，都不算天赋差。
<br/>　　“我拥有万化神剑元神！”苏莫沉声说道。
<br/>　　“哦，我看看！”中年人微微惊讶。
<br/>　　随即，苏莫便立刻释放出了混沌万化神剑元神，即便苏莫没有催动元神之力，已有冲天的锋芒弥漫而出。
<br/>　　“混沌之力！”中年人见此，再次惊讶了起来，因为，这万化神剑元神，明显是混沌之力融炼而成。
<br/>　　“嗯，我修炼五行之力，曾经得到机缘，用混沌之力融炼了万化神剑！”苏莫微笑着说道，他的话并未不妥，一切都在武道规则之中。
<br/>　　“你倒是气运不错！”
<br/>　　中年执事点了点头，随即，他也没有再询问其它，道：“你的天赋，足够加入人王宫，从今日起，你就是外宫弟子。”
<br/>　　“多谢执事大人！”苏莫闻言，顿时向中年人抱拳行礼。
<br/>　　“给我一滴你的血液，不需要精血！”这时，中年执事手掌一番，掌中出现了两枚玉配。
<br/>　　苏莫闻言，并未拒绝，普通血液而已，并不会泄露他的身份。
<br/>　　随即，他指尖逼出一滴血液，一挥手，血液飞到了中年执事的面前。
<br/>　　中年执事手掌一挥，血液一分为二，分别落在了两枚玉佩之上。
<br/>　　“嗖！”
<br/>　　中年执事，手掌一抛，其中一枚玉牌飞到了苏莫的身前。
<br/>　　“那是你的身份玉牌，至于这一枚，则会存放在宫中特殊的地方！”中年人道。
<br/>　　“明白！”苏莫抓住玉佩，沉声说道。
<br/>　　“玉牌之上，有对应的住所，玉佩之内也有很多信息，包括人王宫的宫规，宫内的各种修炼宝地等等，你回去仔细看看！”中年执事又道。
<br/>　　“多谢执事大人！”苏莫抱拳道谢。
<br/>　　“嗯！你去吧！”中年执事挥了挥手道。
<br/>　　“弟子告退！”
<br/>　　苏莫恭敬行礼，便退出了偏殿，离开了大殿。
<br/>　　看了一眼玉佩，其上有一串数字——95278！
<br/>　　苏莫明白，这就是他住所的编号，不过，人王宫很大，寻找还要耗费一番功夫。
<br/>　　苏莫意念一动，进入了玉佩之中，玉佩中有大量的信息。
<br/>　　有各种宫规，还有人王宫详细的地图。
<br/>　　查看了一番，苏莫对于自己的住处，便已经了然于胸。
<br/>　　人王宫分为外宫和内宫，他现在就身处外宫之中。
<br/>　　随即，苏莫大步而行，向自己的住处走去。
<br/>　　很快，他便走到了一处阁楼连绵的区域，这些区域，全部是一排又一排的阁楼，一望无际。
<br/>　　这些阁楼，均是不大，高约五丈，分为上下两层，每座阁楼之间相距两丈有余。
<br/>　　没过多久，苏莫找到了自己的阁楼，走进了其中。
<br/>　　阁楼相当简陋，其内除了一些简单的座椅，没有它物。
<br/>　　最主要的是，阁楼之中，虽然也有修炼室，但只是最普通的修炼室，没有任何的玄妙之处。
<br/>　　“外宫，内宫！”苏莫盘坐在阁楼之中，仔细查看玉佩。
<br/>　　玉佩之内记载的东西非常多，人王宫的格局和规矩，也远非千渊殿可比。
<br/>　　根据玉佩之内的记载，人王宫分内外宫，真神境的修为才能成为内宫弟子。
<br/>　　真神境之下，均是外宫弟子。
<br/>　　“那沁儿身份绝对不简单！”苏莫心中沉吟，此女居然能让一名执事亲自出宫，来找他做入门测验。
<br/>　　这说明什么？
<br/>　　说明那个沁儿，绝对不是一般的内宫弟子，肯定身份不凡。
<br/>　　苏莫面带思索之色，此女，或许是一个很关键的人物。
<br/>　　他可以从此女之处，接触一些人王宫的强者。
<br/>　　他想要得到人王宫的支持，首先要确定人王宫对于轮回天尊复活的看法，其次要和人王宫的高层搞好关系。
<br/>　　关系搞好了，才能得到支持，不然，人家凭什么支持你？
<br/>　　因为你天赋高？
<br/>　　这绝不可能，拓跋青云，天赋也相当逆天，人王宫为何不支持拓跋青云？
<br/>　　而拓跋青云，为何能得到地王宫的支持，就是关系的原因。
<br/>　　身为地王宫宫主的嫡系子孙，如此关系，地王宫自然会支持拓跋青云。
<br/>　　“还是先突破真神，进入内宫再说！”苏莫叹了口气，宫规之上规定，没有特殊允许，外宫之人严禁踏入内宫。
<br/>　　所以，他要要晋升真神之境再说。
<br/>
