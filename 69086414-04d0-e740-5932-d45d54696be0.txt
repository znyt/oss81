
<br/>　　<br/>　　    “大爷，您跟我说说，捞上来的那具尸体是什么样，还记得么？”王惊蛰瞥了眼脸红的陈三岁，回头问道。<br/>　　<br/>　　    抽着烟的老头想了半天，皱着眉头说道：“那具尸体吧穿的衣服是挺怪的，我只知道是古装，要说什么样呢，我还真就不太清楚了反正我确定，肯定不是现在的人，至少也不是近几十年的了……”<br/>　　<br/>　　    王惊蛰有点不甘心的追问道：“大爷，村里的人有几次发现过这种尸体啊，想来你们祖祖辈辈都住在这的吧，水下真要是有尸体的话，几十上百年也不可能就这一具吧？”<br/>　　<br/>　　    “那到确实不是，以前零零散散的也听说有尸体飘上来过，然后被带到了岸边，我就曾经听我爹讲过，他那时也碰见了，不过都几十年了，那时候人能吃饱饭就不错了，谁还会关注个死人啊？到了这些年就少了，除了那年打鱼的时候，以后就再也没有碰到过了”<br/>　　<br/>　　    王惊蛰吐了口气，随意的问道：“那这抚仙湖底是不是也有什么故事或者传说啊，不然也不会无缘无故的就有尸体冒出来吧？”<br/>　　<br/>　　    “传说啊……”<br/>　　<br/>　　    从这些村里老人的嘴中，王惊蛰零零散散的打听到，似乎在很久以前了，到底有多久没人能说得清，这抚仙湖以前是不存在的，那一片湖的水下是一座城，也不知道什么时候开始，整座城都被淹到了水下，然后日积月累下，渐渐的抚仙湖就成型了。<br/>　　<br/>　　    老人们说的也比较零散，说的这些也都是从村子里上几辈人嘴中流传下来的，但没有一点实质的证据，也没人见过那座城，因为抚仙湖中间水域特别深，得有一百多米的深度了，人类潜水的话不借助任何装备能下浅的极限也就一百多米，用潜水装备的话两三百米也有，但这湖下不过就是个传说的故事而已，自然没人会深研究这个，所以也就没人会下去那么深了，除了二十几年前传闻的那个潜水员以外。<br/>　　<br/>　　    再一个就是，村里的人说抚仙湖靠近岸边一带水域还行，但越往湖中心特别是水下，水温特别的低，寻常人是根本承受不住的。<br/>　　<br/>　　    王惊蛰打听了后，自己也上网搜了下，但也都是只言片语的，说的比较笼统，完全没有什么确凿的说法，但有一点可以肯定的是，这水下面确实有明堂。<br/>　　<br/>　　    “以后也别叫你陈三岁了，你干脆改名叫陈二狗得了，你来之前就不会打听一下么？要是知道水下有这么多事，你脑子进水了才会把尸体沉下去！”<br/>　　<br/>　　    陈三岁呐呐的憋着脸色都红了，他尴尬的挠着脑袋说道：“我，我当时也没想那么多大啊，就，就是想着这里能养尸就行了”<br/>　　<br/>　　    “行了，马后炮就别提了，咱们去湖上看一看吧……”<br/>　　<br/>　　    今个抚仙湖的天气不错，风平浪静水波荡漾的，划着小船去往湖上，倒是别有一番情调，只是划船的两个人不太搭。<br/>　　<br/>　　    王惊蛰背着手，站在一艘破船的船头眺望，陈二狗子坐在后面划着船桨。<br/>　　<br/>　　    “再往前就是了，这湖水深的有点吓人，也就这个岸边还算是浅的”<br/>　　<br/>　　    王惊蛰低头看着水面，水确实挺清的，到现在还能看见水下的情景，过了片刻之后能见度这才降了下来，水下已经是黑漆漆的一片了，他蹲下身子用手拨了下，入手即一片冰凉的感觉，这个季节才算是初春，温度还没能上来。<br/>　　<br/>　　    “你怎么下水？太凉了”王惊蛰皱眉说道。<br/>　　<br/>　　    “年轻人火力旺，这个水温其实没啥，冬泳我都游过呢，下去适应一会就好了”陈三岁说着就站起来，把身上的衣服就给脱了下来，然后直接就跳进了水中。<br/>　　<br/>　　    姿势挺优美，宛若浪里白条。<br/>　　<br/>　　    “哗啦，哗啦”陈三岁拨着水面，抬头看着王惊蛰。<br/>　　<br/>　　    半晌后，王惊蛰依旧站在船头没有动。<br/>　　<br/>　　    陈三岁迷惑的问道：“哥，下来啊”<br/>　　<br/>　　    王惊蛰为难的说道：“我就不用亲自出马了吧？”<br/>　　<br/>　　    “不是，你不下来看看，怎么知道是啥情形呢？眼见为真啊”陈三岁有点焦急的说道。<br/>　　<br/>　　    王惊蛰：“……”<br/>　　<br/>　　    眼看着王惊蛰还犹豫，陈三岁转着一双小贼眼，突然伸手猛地一拉破船的船沿，王惊蛰就在一声“哎呀我擦”中，掉进了水里。<br/>　　<br/>　　    “哗啦”王惊蛰从水里钻了出来，抹了把脸上的水渍，吼道：“你他么有病是不是？”<br/>　　<br/>　　    “哥，来都来了，走吧下去看看，也没多远……”<br/>　　<br/>　　    片刻后，陈三岁拉着王惊蛰一起沉入到了水下。<br/>　　<br/>　　    沉尸的地方确实不太深，王惊蛰一个猛子扎下去后没多久，就看见了立在水中的尸体。<br/>　　<br/>　　    说实话，这一幕挺诡异的，正常来说尸体在水里要么是浮上水面要么就是沉底，但这具尸体就是直立向上的，并且身子前倾，当水下偶尔有水流经过的时候，都能眼睁睁的看见尸体来回的摆动，就跟活了一样。<br/>　　<br/>　　    看见了尸体后，陈三岁就朝着上面指了下，两人随即钻出水面，喘了几口气。<br/>　　<br/>　　    “看见了吧？”陈三岁问道，“看的不太清楚，等会再下去一趟，我适应一下的”王惊蛰深吸了口气，然后随即再次下潜。<br/>　　<br/>　　    这一次，王惊蛰直接憋足了一口气来到诗远的旁边，仔细端详了片刻。<br/>　　<br/>　　    这具尸体，之前他曾经见过，那时尸体上毛发是干枯的，还起了一点尸斑，皮肤暗淡无光，如今在水下再见，尸体表皮的特征确实很让人难以置信，这就跟人在水下睡着了一样，似乎几个月过去后，尸体开始焕发了新生。<br/>　　<br/>　　    王惊蛰疑惑的观望了一会，回到水面换气拿起破船上的一个生了锈的船锚，等到了湖底停住身形后，王惊蛰伸出手一把就掐在了尸体的脸上。<br/>　　<br/>　　    陈三岁见状皱了皱眉，但也没有阻止他，王惊蛰捏着尸体脸两侧，稍微用力一挤压，对方的嘴就张开了。
