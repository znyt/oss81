　　叶尘得到昊天镜之后，又过了片刻，两个时辰的期限也到了。
<br/>　　哗！
<br/>　　叶尘只感觉眼前一花，下一刻已经离开第二重星域，来到第三重星域的某一颗星球之上。
<br/>　　呼！
<br/>　　一股浓郁的灵气扑面而来！
<br/>　　“好浓郁的灵气！”
<br/>　　叶尘不由得感叹了一声，同时那道宣布规则的信息，再次在叶尘的脑海之中响起。
<br/>　　“从现在开始，你们可以在此处进行修炼，三日之内，修炼所得的真元，都可以为你们所用！”
<br/>　　叶尘微微一愣，仔细思索了片刻，方才弄明白这个规则背后的含义，
<br/>　　“原来这一关，是考验闯关者的修炼天赋啊！”
<br/>　　仔细想来，第一关考验的是肉身天赋，第二关考验的是精神天赋，第三关则是修炼天赋！
<br/>　　不得不说，这前三关的考验设计，不但精妙，而且全面！
<br/>　　等于将所有人都压到同一起步线，而在肉身、精神、修炼能力三方面，天赋越高的人，所能恢复的修为就越高，从而在接下来的考验中，就会变得尤其具有优势！
<br/>　　“真是天助我也！”
<br/>　　搞清楚了游戏规则之后，叶尘顿时大喜，当即盘腿坐下，开始进入了修炼状态。
<br/>　　他所修炼的吞天神功，可是神级修炼功法，而且尤其以修炼速度快而著称，这一关对他来说，简直太有优势了！
<br/>　　当下，叶尘将吞天神功完全施展开来，
<br/>　　呼！
<br/>　　呼！
<br/>　　呼！
<br/>　　周围浓郁的灵气，顿时犹如潮涌般，纷纷汇聚到叶尘身体百骸之中。
<br/>　　伴随着叶尘，将灵气飞快的炼化真元，汇聚到丹田之中，叶尘的修为也开始飞速暴涨。
<br/>　　从一开始的筑基四重，飞快提升到了筑基五重、筑基六重……
<br/>　　一天时间，匆匆而过，
<br/>　　叶尘的修为也一举恢复到了筑基九重巅峰之境，离金丹境也仅仅只有一步之遥。
<br/>　　而且要知道，这些真元都是叶尘重新修炼出来的，并非之前被封印的真元，等到丹田内的封印一旦解开，他的修为必然可以更进一步，应该已经有足够的把握，去冲击化神境！
<br/>　　很快，叶尘再次被一股伟岸的力量，强行带离了第三重星域，进入到了第四重星域。
<br/>　　呼！
<br/>　　等叶尘再次踏足到一个陌生的星球之上，顿时猛然感觉到一股凌冽的寒意，仿佛进入了一个远古战场之中。
<br/>　　“前三关只是对闯关者资质的考验，几乎没有任何风险，接下来的考验，恐怕就没有那么简单了！”
<br/>　　“而且，第一重星域是128颗星球，第二重星域是64颗星球，第三重星域为32颗，到了着第四重星域，便只有16颗星球了，这一次，应该有极大的概率，会遇到其他人……”
<br/>　　叶尘喃喃自语了一句之后，那道发布规则的冰冷声音，也再次陡然响起，
<br/>　　“本关任务，猎杀一头冰甲猛犸象，并且取其妖丹，将其炼化，方可进入下一关……”
<br/>　　叶尘顿时一愣，眉头不由得紧紧皱起，
<br/>　　冰甲猛犸象，乃是四阶巅峰妖兽，尤其以防御强悍著称，甚至单单论防御能力，比寻常五阶妖兽还要恐怖。
<br/>　　即便是在修为没有被封印之前，叶尘想要击杀一头冰甲猛犸象，恐怕也要费一番功夫。
<br/>　　而如今，他的修为被大大压制，仅仅只有筑基境巅峰而已，也无法动用法宝，想要猎杀一头防御如此之强的妖兽，简直难如登天啊！
<br/>　　唯一值得庆幸的是，这一关似乎并没有时间的限制，倒是可以慢慢去想办法。
<br/>　　叶尘一边暗暗寻思着，同时一边开始四处搜寻冰甲猛犸象的踪迹。
<br/>　　然而，眼下的这座星球，实在是太大了，这些冰川之间，妖兽倒是有不少，可是叶尘找了大半天，却连一头冰甲猛犸象也没有遇到。
<br/>　　不过，叶尘发现，随着时间的流逝，他丹田之内的那道封印，也开始缓缓解封了些许，真元在慢慢的恢复着！
<br/>　　轰！
<br/>　　随着丹田内恢复的真元越来越多，终于一举突破了瓶颈，叶尘的修为也重新恢复了金丹境！
<br/>　　叶尘顿时大喜，
<br/>　　如果说之前，猎杀冰甲猛犸象，他还没有太大把握的话，如今实力恢复到金丹境，有了质的飞跃，基本上就已经稳了。
<br/>　　不过高兴之余，叶尘却又有些焦虑，
<br/>　　很显然，随着他在这里停滞的时间越来越多，修为虽然在渐渐恢复，可是时间也会被浪费掉！
<br/>　　而越早完成任务，进入到下一关，对之后的关卡，应该会更有利！
<br/>　　所以，他现在必须要抓紧时间了！
<br/>　　一念及此，
<br/>　　轰！
<br/>　　叶尘脚下猛然一蹬，直接飞身而起，
<br/>　　之前困在筑基境，法宝没有办法使用，连御物飞行都无法做到，所以只能两条腿地奔。
<br/>　　而一旦迈入金丹境，便可以凌空飞渡，踏空而行。
<br/>　　叶尘纵身一跃，直接飞身到数千米的高空，视野瞬间就一下子开阔了起来，足足可以观察万里之外的情景。
<br/>　　“嗯？那边好像有打斗！”
<br/>　　叶尘一眼看到在大约四、五千里开外，一片冰川雪林之中，似乎有正在打斗的迹象，于是当即身体一晃，直接飞速朝赶了过去。
<br/>　　若是能找到其他闯关者，一同联手对付冰甲猛犸象，也未尝不是一个好的办法。
<br/>　　轰隆！！
<br/>　　在离那片区域，只有数百里距离的时候，虽然还看不清楚人脸，但叶尘已经可以清楚的看到下方打斗的场景。
<br/>　　在那片冰川之间，的确有着一头体型庞大的冰甲猛犸象，不过让人感到诧异的是，此时那头冰甲猛犸象，似乎已经被人所击杀。
<br/>　　至于一旁，则是一群人正在打斗！
<br/>　　不！确切的说，是一群人，正在围攻一其中个人！
<br/>　　而且那人白裙飘飘，似乎是还是一名女子。
<br/>　　叶尘并没有着急暴露自己，直接身体一晃，返回地面之上，然后一道残影在空中划过，已经悄悄逼近那些人千米之内，
<br/>　　就在这时，一道冰冷中夹杂着愤怒的声音，陡然传入叶尘的耳朵之中，
<br/>　　“云破天！我们说好联手对付冰甲猛犸象，你竟然过河拆桥！！卑鄙无耻！”
