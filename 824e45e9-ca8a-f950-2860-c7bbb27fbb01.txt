　　众人想要四处逃窜，然而这时，
<br/>　　“桀桀桀！！”
<br/>　　血天海一阵狂笑，
<br/>　　“还想跑？本座的血日妖轮一出，你们哪个也休想活命！！”
<br/>　　“血海无涯！！”
<br/>　　伴随着血天海一道暴喝，他头顶的那轮血日，居然开始疯狂涌出海量血水，朝周围铺天盖地的奔腾而去，只一眨眼的功夫，那漫天的血水，犹如一张巨大的幕布，竟然将整个黑岩城都笼罩其中！！
<br/>　　紧接着，那些诡异的血水，开始自外往内，迅速的收缩着，而但凡是被那血水覆盖之处，黑岩城的那些人，全都被吞噬其中，化成血汁……
<br/>　　“啊啊啊啊！！”
<br/>　　伴随着那些血水朝中心位置狂涌而来，一道道惨叫声此起彼伏，犹如死神般收割着城中的一条条生命。
<br/>　　而随着那些血水吞噬着越来越多的生命，血天海的气息，竟然开始飞速的增长！！
<br/>　　“你们这群蝼蚁，能够成为本座的食粮，也算是尔等的荣幸了！哈哈哈！！”
<br/>　　才一转眼的功夫，血日妖轮已经吞噬了上万人的性命，而血天海之前被叶尘重创，身体的颓势完全消失不见，甚至气息更胜从前！
<br/>　　“小子！我这血日妖轮，乃是妖族邪物，为修真正道所不容，所以就算本座也不能轻易动用！”
<br/>　　“妖轮一出，有死无生！今天这里的所有人，全部都得死！！”
<br/>　　血天海气息恢复过来之后，整个人身上的血煞之气，变得无比浓郁，连双眸都变成了妖异的赤红色，犹如从地狱冲爬出的恶魔，似乎已经完全丧失了理智。
<br/>　　而此刻，站在九天之上的叶尘，也不由得眉头微微一皱。
<br/>　　对方所施展出来这件法宝，即便是他，都感受到了威胁！
<br/>　　不得不说，眼前的这个血天海，比之前他在天玄星域，对付的那三名化神强者，还要难缠的多！
<br/>　　当然了，倘若此刻他直接选择离开，血天海也奈何他不得。
<br/>　　可是一想到下方，灵溪那小丫头还在城中，叶尘便说什么也不能不管！
<br/>　　“开！”
<br/>　　叶尘长袖一挥，紫琼剑在空中划过一道优美的弧线，狠狠斩在下方的那道血幕之上。
<br/>　　咔！
<br/>　　血幕直接被叶尘这一剑，生生撕开了一道长达近千米的巨大裂缝。
<br/>　　叶尘身体一晃，便冲入下方的黑岩城中，落在了灵溪所在的房间屋顶之上。
<br/>　　小丫头此刻还在沉睡，还尚未清醒过来。
<br/>　　嗖嗖嗖！
<br/>　　叶尘随手一阵虚点，便在周围设下十几道禁制，足以暂时保证小丫头的安全。
<br/>　　等做完这一切之后，叶尘这才朝血天海杀了过去。
<br/>　　“桀桀！”
<br/>　　血天海看到叶尘冲了进来，顿时兴奋的狂笑起来，
<br/>　　“小子！看来你还真是不知死活啊！居然敢主动进入本座的血阵之中！”
<br/>　　“既然如此，那就留下做本座的口粮吧，吞噬了你的血脉，本座肯定能实力大增，桀桀桀！”
<br/>　　血天海说着，便主动朝叶尘扑了过来。
<br/>　　“想吞我？也不怕崩坏了你的牙！！”
<br/>　　叶尘举剑迎上。
<br/>　　轰！
<br/>　　轰！
<br/>　　轰！
<br/>　　转眼间，两人已经又交手了十多招，虽然叶尘凭借着逆生百转的效果，能稳压对方一筹，可是在血阵之中，血天海有着源源不断的补给，血日妖轮不停的在向他输送着能量，每次叶尘将他重创之中，他都能很快恢复，简直就像是打不死的小强！
<br/>　　“桀桀桀！小子，我劝你还是别费力气了！在这里，本座就是不死之身，耗也能把你耗死！！”
<br/>　　叶尘眉头紧皱，默然不语。
<br/>　　就眼下的情况来看，对他的确十分不利！
<br/>　　他的逆生百转对强大逆天，可最多的缺陷就是不能打消耗战，一旦功力加成的效果结束，倒是他就成了待宰羔羊，根本没有任何还手之力！
<br/>　　‘也罢！反正旁人的死活，我也懒得去管，我若是只带着小丫头一人离开，倒也不是什么难事！’
<br/>　　打定主意之后，叶尘正准备付诸行动，
<br/>　　然而就在这个时候，叶尘忽然察觉到体内，吞天鼎竟然莫名的躁动起来。
<br/>　　“奥维斯！你搞什么？”
<br/>　　叶尘当即通过神念，和吞天鼎的器灵沟通起来。
<br/>　　吞天鼎虽然是仙器，可并非攻击类法器，所以不到万不得已，叶尘不会动用这个底牌。
<br/>　　奥维斯语气有些兴奋道：
<br/>　　“主人，我好像感应到了噬血妖珠！就在对方的那个法宝之中！”
<br/>　　“噬血妖珠？”
<br/>　　叶尘顿时一愣，
<br/>　　“那是什么东西？”
<br/>　　奥维斯道：
<br/>　　“主人，其实我之前一直没有告诉您，现在的我，并非我的完整形态！”
<br/>　　“当初老主人在创造我的时候，是准备将我打造成一件攻守兼备的绝世神器！”
<br/>　　“除了我眼下的这个主体之外，其实还有两个附件，一为噬血妖珠，一为噬魂魔珠！”
<br/>　　“可惜在十万年前的那场仙魔大战中，老主人陨落，我也受到重创，噬魂魔珠和噬血妖珠脱离主体，下落不明……”
<br/>　　“万万没有想到，竟然会在这里遇到噬血妖珠，而且它似乎被人炼制成了其他法宝！”
<br/>　　叶尘通过神识，和吞天鼎的器灵交流，不过是一瞬间的功夫，便已经了解了事情的来龙去脉，不由得心念一动，
<br/>　　“奥维斯，既然是这样的话，那你有没有把握，将噬血妖珠抢过来？”
<br/>　　奥维斯傲然道：
<br/>　　“虽然噬血妖珠，被人打造成了别的法宝，不过老主人的精神烙印，又岂是那么容易被抹掉的？主人你只管放我出来，我定能将噬血妖珠降服！”
<br/>　　叶尘闻言大喜，当即心念一动。
<br/>　　轰！
<br/>　　吞天鼎破体而出，然后直奔血天海头顶的血日妖轮爆射而去。
<br/>　　“什么东西？”
<br/>　　血天海也愣了一下，等到他反应过来，吞天鼎已经将血日妖轮罩住，
<br/>　　“噬血妖珠！还不速速归位！！”
<br/>　　奥维斯的声音，自吞天鼎内发出，那血日妖轮，顿时开始剧烈的颤动起来。
<br/>　　而与此同时，血天海陡然发现，自己的法宝竟然有脱离自己控制的迹象。
<br/>　　“这到底是什么鬼东西？！！”
<br/>　　血天海惊骇不已，慌忙动用神念，想要将血日妖轮收回体内。
<br/>　　可惜，叶尘又岂会让他如愿？
