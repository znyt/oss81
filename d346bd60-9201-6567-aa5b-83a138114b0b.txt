        &lt;h3&gt;第三卷第一百五十一章  只要你安好，便是晴天（感谢芒果酸奶加点冰盟主）&lt;/h3&gt;
<br/>
<br/>        春姨和财叔没有想到陈子昂会回家，张函还在房间陪着孩子玩耍呢，陈子昂不是还在西城吗？
<br/>
<br/>        陈子昂因为公司最近的事情多，便坐了早上的那一班飞机回到了北京，她到北京之后就直接到公司处理事务，没有给春姨说她下午要回家吃饭。
<br/>
<br/>        陈子昂看着父子三个人坐在地毯上和谐的画面微微愣了一下神，感觉眼眶有些发热，她悄悄的退了出来。
<br/>
<br/>        春姨说：“张函来接孩子，俩个孩子有些感冒了，我就没有让出去。”
<br/>
<br/>        “没事，来就来了吧。”陈子昂知道春姨的想法。
<br/>
<br/>        财叔正在厨房倒腾着饭菜，既然张函来了，没有必要吃饭的时候还把人家赶出去，于是财叔就帮着春姨一起做了几个菜，准备留张函一起吃晚饭。
<br/>
<br/>        陈子昂回来刚好赶上吃饭的时候。
<br/>
<br/>        当春姨去喊孩子出来吃饭的时候，张函也跟着一起出来了，他看到陈子昂也是一愣，心里有些忐忑，在西城陈子昂可是将他赶出门了。
<br/>
<br/>        “子、子昂你回来了？”张函感觉有点尴尬。
<br/>
<br/>        “妈妈回来啦。”团团和嘟嘟撇下爸爸直接冲过去一人抓着妈妈的一只胳膊。
<br/>
<br/>        “感冒了？”陈子昂对着俩个孩子问。
<br/>
<br/>        “有一点，不过爸爸来了，我们的感冒就好了，爸爸你说是不是？”古灵精怪的嘟嘟走过去拉着爸爸的手。
<br/>
<br/>        张函正准备拿着背包出门，陈子昂连招呼都不给他打，自然是不想见到他。
<br/>
<br/>        团团也走过去拿下张函的背包：“爸爸，该吃饭了。”
<br/>
<br/>        陈子昂看了看俩个孩子，又看了一眼张函：“张总，到了饭点了，不如将就着吃了饭再走？”
<br/>
<br/>        “这.....“张函心里有些迟疑。
<br/>
<br/>        “来，快坐下，难得一起吃顿饭。”财叔从厨房出来招呼着张函坐下。
<br/>
<br/>        为了缓和一下尴尬的气氛，财叔拿出来了一瓶酒，对陈子昂说：“我很久没有喝酒了，今天高兴，咱爷三个喝几杯。”
<br/>
<br/>        “你想喝就喝吧。”陈子昂见财叔提出来喝两杯没有异议，她难道不理解财叔的苦心吗。
<br/>
<br/>        财叔和春姨也是用了心，五个人连汤带菜一共是十个菜，原本还是没有做陈子昂的饭菜，可见他们对张函的到来还是很用心的。
<br/>
<br/>        蒜蓉粉丝蒸龙仔虾、鲍鱼烧肉、香煎银鳕鱼、小炒黄牛肉、酱香凤爪、素菜沙拉、酸爽脆黄瓜、红豆山药膏、鸡汤辽参、椿芽笋丝。
<br/>
<br/>        财叔给张函倒了一杯酒，对张函说：“你不能喝酒就少喝点。”
<br/>
<br/>        陈子昂以前是见过张函喝酒的，不知道为什么财叔会这样说？
<br/>
<br/>        张函感激的接过了酒杯，在他的心目中，这俩个老人是陪着他的孩子和陈子昂一起度过了这么多年，不是亲人胜似亲人。
<br/>
<br/>        财叔也给陈子昂倒了一杯酒，对陈子昂说：“你想喝了可以陪我喝两杯，不想喝了也少喝点。”
<br/>
<br/>        俩个孩子眼冒星星的看着自己的爸爸妈妈，嘟嘟悄悄的对团团说：“爸爸和妈妈好般配哦，哥哥我们这算不算是团圆了？”
<br/>
<br/>        “小点声，别让妈妈听见了。”团团低声的吩咐妹妹。
<br/>
<br/>        俩个孩子因为兴奋和快乐，小脸红扑扑的。
<br/>
<br/>        陈子昂看着俩个孩子开心的模样，心里一疼，在别人的家一家人坐在一起吃饭是再正常不过的事情，可是她的孩子却觉得是世界上最幸福的事情。
<br/>
<br/>        “乖乖的吃饭，吃完饭晚上早点休息，只有休息好了，感冒才能好呢。”陈子昂宠溺的看了一下孩子。
<br/>
<br/>        “妈妈，我今天能不能多玩一会啊。”嘟嘟的鼻音很重。
<br/>
<br/>        “是啊，妈妈我也想多玩一会。”团团也囔着声音。
<br/>
<br/>        俩个孩子身体很好，很少生病，有个头疼脑热的都是一个症状，孩子生病了陈子昂有些心疼。
<br/>
<br/>        “那你们乖乖的吃饭吧。”陈子昂算是默认了让俩个孩子多玩一会，俩个人喜滋滋的低着头开始吃饭。
<br/>
<br/>        张函看着陈子昂和孩子们的互动，心里暖暖的，他拿起酒杯，对春姨和财叔说：“阿姨、叔叔，我敬您们一杯，感谢您们照顾子昂母子三人，要不是您们，也就没有他们的今天。”
<br/>
<br/>        “说这话就见外了啊，我们当子昂是女儿了，孩子就是我们的孙子。”财叔和春姨乐呵呵的笑着说，这么些年，他们对张函的印象还是不错的。
<br/>
<br/>        “子昂，我借你的酒也敬你一下，感谢你将俩个孩子教育得这么好，我问心有愧。”张函的声音有点哽咽。
<br/>
<br/>        “彼此感谢吧，没有你我也生不出来孩子不是吗？”陈子昂微微的笑着。
<br/>
<br/>        她的一句话逗得大家都笑起来，气氛瞬间缓和了。
<br/>
<br/>        陈子昂本身对张函就没有恨意，她怀孕的时候张函不知道，而且事情过去了这么多年，该面对的现实还是要面对，在孩子今后的人生中，张函肯定是要参与的，她也没有必要和张函像仇人一样，不如敞开心扉，做一个普普通通的朋友。
<br/>
<br/>        陈子昂不知道的是张函从来都没有忘记过她，而张函的婚姻无非就是一层窗户纸，一捅就破。
<br/>
<br/>        财叔的酒量不行，只是为了缓和气氛，才喝的酒，三杯酒下肚，财叔的话就多了起来。
<br/>
<br/>        “张总，你知不知道你当初在某仁医院住院的时候，子昂经常去哪个医院做产检？”财叔问张函。
<br/>
<br/>        “我不知道，那一段时间我心如死灰，没有关注别的事情。”张函叹了口气，还好，哪个时候他不知道子昂怀孕了，还好子昂进入了落家。
<br/>
<br/>        他在住院期间，有个女同学到医院探望她，出门没多远就被陆玉开车误撞断了腿，在医院也躺了大半年，陆玉仅仅就是赔了一点钱。
<br/>
<br/>        “财叔，你怎么知道他也在那里住院？”陈子昂好奇的问。
<br/>
<br/>        “你知道为什么你能在落家一呆就是几年吗？”财叔微微一笑。
<br/>
<br/>        “不是雪儿的原因吗？”陈子昂想不到还有别的原因。
<br/>
<br/>        “张总是你落白哥哥的堂哥，你进落家的第二天下午我们就知道你们的情况了。你落叔叔才将你们保护起来的。”财叔一语道出了几年前的原因。
<br/>
<br/>        “张总，感谢你。”陈子昂苦笑一下，她以为纯粹是落雪的原因，原来最主要的原因还是因为张函。
<br/>
<br/>        “子昂啊，人这一生其实有很多身不由己，所以我们更要珍惜彼此啊。”财叔有些感慨。
<br/>
<br/>        “嗯。”陈子昂闷闷的应着，她已经很珍惜了啊。
<br/>
<br/>        她珍惜她身边的每一个人、每一次机会，正因为她知道生活的不易，她才很努力、很珍惜！
<br/>
<br/>        “子昂，我没有去找你，是因为我生病住院了，等我出院之后便再也找不到你。”张函拿起一杯酒、一饮而尽、辛辣的酒就像是他的人生，火辣辣的烧得他的心疼。
<br/>
<br/>        张函从陈子昂家里出来的时候，雪已经下了厚厚的一层。
<br/>
<br/>        陈子昂在窗帘后面一直看着张函走出大门。
<br/>
<br/>        陈子昂想到了一句话：只要你安好，便是晴天！
