　　“多谢师祖！”
<br/>　　公孙轩辕一觉醒来，便看见自己的师祖正笑眯眯的看着自己，他连忙拜道。如今的轩辕，虽然外表没有什么变化，但他全身的气质，已经发生了天翻地覆的变化，之前那是凌厉，而如今却是返璞归真了。
<br/>　　这还不是最重要的，最重要的是在这一刻，他的心境修为已经被无限拔高了，如今的他，只需要不断去提升修为，完全不用怕因为修炼过快而导致心境不稳等问题。
<br/>　　“好了，如今第一个好处已经给你了，想必你也知道其中的价值，百世轮回之中无数人生经历，不仅仅只是提升了你的心境，更是让给你获得无数经验，为你节省了数万年的时间。”
<br/>　　元始淡淡开口说道，打断了轩辕的思绪。
<br/>　　“多谢师祖！”轩辕再次恭敬拜谢道。
<br/>　　“至于第二件好处，喏，你看看——”元始说道这里，他向着虚空一点，顿时虚空一阵涟漪波动，出现一个虚幻的阵法，阵门转动，玄光闪耀，顿时从涟漪的中心处，缓缓出现一柄金黄色的神剑。
<br/>　　神剑全身呈现金黄色，无论是剑柄，还是剑身，剑刃、剑锋、剑尖等，都是金黄色。剑身一面刻有日月星辰，一面刻着山川草木，剑柄一面书农耕畜养之术，一面书四海一统之策。
<br/>　　剑身微微一荡，便有无尽力量迸发，充满圣道的气息，皇者的霸道威严，让人忍不住在它面前低下头来，进行跪拜。
<br/>　　公孙轩辕一看到这剑便喜欢上了，他的目光盯着神剑，便再也移不开了，在他内心深处有一种渴望，渴望将这剑拿到手上，挥斥方遒。不知为何，他看到这剑的第一眼，便觉得这剑最适合自己，就好像……好像这剑天生便是为自己而生的。
<br/>　　“师祖，这剑……”
<br/>　　公孙轩辕虽然喜爱这剑，但师祖在眼前，他还是不敢放肆，他强忍住心中的不舍，将目光移开，眼巴巴地看上元始。
<br/>　　元始微微一笑，看着轩辕这幅模样，颇觉得有趣，轩辕如今已经有百岁了，再加上他梦中轮回的岁月，早已超过万年，可如今这幅模样，却是如同一个小孩一般。
<br/>　　不过，他也没有多耽搁，直接开口说道：
<br/>　　“此剑乃是本座采首山之铜而铸，其中蕴含着圣道的力量，斩妖除魔，不在话下。它可做你的本命神器，可随着你不断晋升，直至你证道，它便可以进化成道器。”
<br/>　　“多谢师祖！”
<br/>　　轩辕一听此话，顿时兴奋了，他向着元始道谢一声，便顾不得其他，直接一手伸向金色圣剑，想要将之抓住。
<br/>　　可是，还不等他抓住，那剑便摇晃起来，并且发出凌厉的剑光，看向公孙轩辕。
<br/>　　轩辕见此，心中一急，伸出的右手并指一挥，顿时一道剑光出现，与那金色圣剑发出的剑光相撞，轰的一声，两者相互湮灭于虚空。
<br/>　　“师祖，这……”
<br/>　　公孙轩辕扭头看向元始，他不知道这剑为何会发出剑光来攻击他。
<br/>　　“圣剑有灵，能够被他承认，方有资格执掌此剑。”元始淡淡出声道。
<br/>　　“哦？原来如此，这剑真有意思，我喜欢！”
<br/>　　公孙轩辕听罢，顿时明了，原来不是圣剑要躲避他，而是要考验他啊。知道原由后，他不再心急，而是很有耐心的重新看向圣剑。
<br/>　　圣剑凌空旋转，形成一道巨大金色的剑刃风暴，撕裂天地，形成毁天灭地之势，向着公孙轩辕席卷而来，誓要将他撕碎。
<br/>　　“来得好！”
<br/>　　公孙轩辕一声大叫，双目之间，不见丝毫波动，他目光冷静，看着那惊天动地的剑刃风暴，背后木剑凌空，一个旋转，便被他抓到手中。
<br/>　　“葬剑诀，葬天，葬地，葬众生！”
<br/>　　突然，公孙轩辕目光大盛，眼中尽是无尽剑气，全身气势凌人，剑意森寒。同时手中木剑也挥动起来，他手捏剑诀，木剑凌空，形成一道巨大的缺口，仿佛要将这片天地都埋葬进去。
<br/>　　天地如狱，我便葬了这天地！
<br/>　　轰隆！
<br/>　　两者相互交接，葬剑诀以迅雷不及掩耳之势，将圣剑斩出的剑刃风暴直接埋葬，丝毫不剩。
<br/>　　圣剑不服，继续攻击。金黄色的剑身流转，释放是强大的气势，剑尖遥遥对着公孙轩辕，轻轻一点。
<br/>　　“嗖！”
<br/>　　一点寒光万丈芒！
<br/>　　剑点形成黑色死亡剑气，以一种不可见的速度，向着轩辕点来，剑点瞬间形成万道寒芒，遮蔽这片天地。这片空间完全被封锁了，让人逃避不开，所过之处，万物寂灭，虚空枯萎，犹如死神到来一般，生灵都被剥脱了生命。
<br/>　　见到这一幕，公孙轩辕目光一凝，心头略显沉重，面对这一剑，他也不敢有丝毫的放松，若是因为大意了导致自己败了，丢脸是小，得不到圣剑的承认，那才是悲剧。因为从看到这剑的第一眼，他便认定了此剑只能自己所有。
<br/>　　“乱剑诀！”
<br/>　　公孙轩辕再次单手挥舞木剑，剑法杂乱无法，完全没有一丁点规律，就好似是一个不会使剑的人在拿着神剑随意劈砍。但就这看似杂乱无双的随意劈砍，却是每一剑中，都仿佛带着莫大的威力，那是一种混乱的规律，似乱不乱，似弱实强。每一剑都却恰到好处，不多不少，让人完全摸不清。
<br/>　　既然摸不清，那还如何抵挡？这便是乱剑诀的威能。
<br/>　　“乱中我为尊，烟雨任平生！”
<br/>　　乱剑天地，形成一方剑气牢笼，将那万丈寒芒困住，之后一股混乱之力爆发，完全将那无数死亡寒芒于混乱之中消灭殆尽。
<br/>　　公孙轩辕一步跨出，径直来到轩辕剑身旁，以一种势无可当的气势，抓住圣剑的金黄色剑柄。
<br/>　　“嗡嗡嗡！”
<br/>　　圣剑剑身在颤动，发出嗡嗡的响声，像是在反抗轩辕。紧接着，一道虚幻的圣剑破空而出，在公孙轩辕来没来得及反应过来之时，便从他的眉心没入。
<br/>　　剑刃锋锐，斩灭神魄，目标直接，毫不掩饰地劈向公孙轩辕的神魂所在之地，誓要将他的神魂绞碎。
<br/>　　斩神剑！
<br/>　　公孙轩辕先是心中一急，以为这剑誓要将他直接劈了，但看到那剑光砍向他的神魂之处，他反而不急了。若是一般人被这剑斩向魂魄之处，那完全是十死无生，但他却不同。
<br/>　　他早已将《神元九转》练到了第三转，经过此次拜师轮回，不仅让他的心境蜕变了，他的元神之力同样大增，突破到了第四转融术。
<br/>　　他的神魂现在完全是一片剑域，斩神之剑进入剑域之中，只能被同样为元始所化的剑气同化吞噬掉。
<br/>　　果然，圣剑发出的斩神剑气没入公孙轩辕的神魂剑域之中，顿时就有无数道剑光出现，它们同时斩向斩神之剑，誓要捍卫自己的领地。
<br/>　　那斩神之剑还没来得及折腾几个浪涛，便被轩辕的神魂剑域之中的剑气给生生磨灭了。
<br/>　　“嗡嗡嗡……”
<br/>　　圣剑一阵轻鸣，发出呜呜响声，似是在哭泣，又像认命了，它不再反抗，开始静静躺在公孙轩辕手心。
<br/>　　……
