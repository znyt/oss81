　　第186章
<br/>　　好一场雨，当真是：龙施号令，雨漫乾坤。势如银汉倾天堑，疾似云流过海门。楼头声滴滴，窗外响潇潇。天上银河泻，街前白浪滔。淙淙如瓮捡，滚滚似盆浇。孤庄将漫屋，野岸欲平桥。真个桑田变沧海，霎时陆岸滚波涛。神龙借此来相助，抬起长江望下浇。
<br/>　　有词赞曰：
<br/>　　『冷冷的冰雨在脸上胡乱地拍，暖暖的眼泪跟寒雨混成一块』
<br/>　　……
<br/>　　借着雨势，赵子龙冲到马车前救了林少羽，将他搭在马上，拨转马头往城门走。
<br/>　　程大雷以一当十，虽然还是那三板斧，可威力已大胜从前。现在的他已经是顶级实力，能和秦蛮、赵子龙、林冲平起平坐，不管他以前是不是个废物，如今都已能抬头挺胸重新做人。
<br/>　　莫名的杀招不用思考便从脑海中涌出，对方一刀劈来，他心底至少有四五种应对之法。现在程大雷感觉，战斗的似乎是另外一人，而自己只是操纵他前进后退。
<br/>　　成为一个高手的感觉，还真是不太习惯啊。
<br/>　　一斧头将身边的敌人劈开，鲜血泼在程大雷脸上，这种腥热的感觉为何令人如此兴奋，从什么时候开始，自己有些喜欢这样的杀戮了？
<br/>　　本来已有几分力竭，可突然间似乎有新的力量涌起，程大雷有一种冲动，想杀到敌营，将敌人个个斩尽，刃刃杀绝。
<br/>　　鬼面斧附带技能：热血。
<br/>　　不过是冲动，就该遏制，真如果冲进敌营，再热血也是死路一条。自己的任务只是掩护赵子龙救人，如今人已救，自己也该撤了。
<br/>　　“戎族的渣渣们，看我五雷法！”
<br/>　　程大雷拎起马上一个箱子，一边逃得气喘吁吁，用火石点燃，幸好这火石和火药一直用蓑衣遮着，尚未被雨水淋湿。
<br/>　　轰！
<br/>　　火药箱丢进戎族的兵丁中，死伤一片。
<br/>　　“他，他竟会掌心雷！”
<br/>　　“这真是神仙转世！”
<br/>　　兵丁惊呼一片，有几个连兵器都握不住了。
<br/>　　“混蛋，是火药！”金问道大骂：“追，放箭，给我放箭！”
<br/>　　终究还是有有见识的，金问道一眼就看穿了程大雷的把戏。
<br/>　　因这风雨之势，弓箭的威力就去了大半，就算有箭落在程大雷背心，却发出铛地一声响，如中金石。
<br/>　　“刀枪不入！”
<br/>　　“还说他不是神仙，铜皮铁骨呐！”
<br/>　　金问道皱起眉头，知道这货蓑衣内肯定穿着盔甲，可惜自己手下都是一些野蛮人，这样的把戏也能把他们唬住。
<br/>　　还真是卑鄙啊！
<br/>　　“追！有敢退后一步者，杀！”
<br/>　　金问道下了军令，看着赤眉。
<br/>　　“弟弟，该你了。”
<br/>　　赤眉点点头，带着大军冲杀出去，这一战，他为先锋。
<br/>　　逃跑程大雷还是很擅长的，关键是他座下的瘦马附带逃逸技能，逃跑时有速度加成。
<br/>　　他和赵子龙前后脚冲进城，可此时城门还未来得及关闭，赤眉已经领着大军杀进来。
<br/>　　从城内又有两队人冲出来，一人秦蛮，一人林冲。二人据城门坚守，掩护程大雷和赵子龙撤退。
<br/>　　秦蛮自不必说，林冲早就憋着一股气，做人，他不擅长，做事，也差那么点意思，但战场杀敌，是他份内之事。
<br/>　　说了马上第一，便是马上第一。
<br/>　　刚交手就灭了对方一员大将，一枪七叠，便是林少羽有隐藏属性加成，还和林冲差点。
<br/>　　“退啦，退！”已安全脱身的程大雷大吼。
<br/>　　秦蛮和林冲撤回山谷。
<br/>　　赤眉率大军闯进城门。
<br/>　　所谓久守必失，长期坚守没有不破的道理。
<br/>　　今日，蛤蟆寨城门被攻破。
<br/>　　金问道看着这一幕，骑在马上哈哈大笑。
<br/>　　任你装神弄鬼又如何，任你把人劫走又如何，我要的不是杀人，我要的是破城。
<br/>　　说了三日破城，便是三日破城，差一天，差一个时辰，差一刻都不算。
<br/>　　“草原的勇士，长生天的儿孙，随我破城！”
<br/>　　赤眉闯进城门，发现城门后还有一座山谷，谷口狭窄。
<br/>　　他率先冲过去，等待他的不是斩将立功，一雪前耻，而是一柄长矛。
<br/>　　张三爷的长矛。
<br/>　　若论单骑救主当属赵子龙，林冲马上第一，秦蛮臂力惊人，可若说一夫当关张三爷才是祖宗。
<br/>　　一矛捅天，一矛扎地，一矛赶猪，在这个时代蒙尘生锈的丈八蛇矛，第一次展现它应有的风姿。
<br/>　　刘关张皆在，隐藏属性桃园结义被激发，每个人的实力都暴涨一截，即便是一脸苦相的刘悲也能耍一耍双剑。
<br/>　　赤眉一时攻不破。
<br/>　　纷纷有人涌进城门，人越聚越多，越聚越多……程大雷修城时，并没有把瓮城建得很大，此刻人已经挤满。
<br/>　　“看我五雷法哦！”
<br/>　　一声高呼，紧跟着轰隆一声爆响，众人耳膜似乎都要被震裂。
<br/>　　赤眉猛地回头，忽然发现程大雷立在城头之上，却原来，他刚在不是逃进山谷，而是……逃上了城墙。
<br/>　　他看着周围瓮城内的景象，人挨人人挤人，人头黑压压一片，而四面高墙耸立，多像一口锅吧。
<br/>　　一口要把所有人煮死的锅。
<br/>　　有火药，滚石，金汤等物从城头上丢下来。
<br/>　　轰隆隆响声一片，在这样的地形下，火药的威力发挥到极致，冲击波震荡，手臂，内脏，残破的脑袋等人的躯体纷飞，他们原先是不是属于一个人，现在已无人知晓。
<br/>　　以城门为界，城内和城外已经被隔开，堵住他们的不是城门，而是尸体。
<br/>　　堆积如山的尸体。
<br/>　　程大雷兑换了多少火药呢，三十箱。
<br/>　　在如此惨烈的杀戮下，程大雷的恐惧值暴涨，他仍在不停兑换火药，负责轰炸的士兵发现，城头上火药好像搬不尽似的。
<br/>　　最后，程大雷一共兑换了一百箱火药，将瓮城的地皮都砸没了一尺，当然现在还看不来，因为上面堆满了尸体。
<br/>　　人们想向外逃，可是怎么逃得脱。赤眉知道，只有继续向山谷冲才有一线生机。可无论他此刻怎样嘶吼，咆哮却无一人会听他的。
<br/>　　军魂已破。
<br/>　　他看着城头上的程大雷，目光不知是愤怒还是怅然若失的悲伤。
<br/>　　又是这样，当你以为自己将要胜利时，他干干脆脆关上门，冲你甩出两个字：
<br/>　　休想。
<br/>(本章完)
