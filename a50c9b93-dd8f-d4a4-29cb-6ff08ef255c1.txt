　　青色的平台上，那些和苏莫同平台的人，不少人目光都落在了苏莫的身上。
<br/>　　虽然苏莫的修为较低，但能排在真圣榜第一，绝对不能小觑，也是威胁之一。
<br/>　　“哈哈！你们都要被淘汰！”一名巨灵族的青年朗声大笑，此人的体型和巨摩相差无几，身高四丈，一身爆炸性的肌肉，给人强大的力感。
<br/>　　此人极为自信，他巨灵族不仅战力强大，防御力亦是远超其它种族。
<br/>　　“巨煌，你就是活靶子，还想要过关？”
<br/>　　一名飞灵族的天才满脸不屑之色，对方这么大的身躯，只会承受大量的攻击，随即他自信的说道：“这一关，本平台之上，若说谁能撑的最久，非我飞华莫属！”
<br/>　　众人闻言面色凝重，这飞灵族的飞华的确是希望很大，首先其是真圣榜排名一百八十八名的天才，其次又是飞灵族之人。
<br/>　　飞灵族之人速度无双，躲避攻击的能力，不是其他人可以比拟的。
<br/>　　可以说，就算飞华的实力不如巨煌，但度过这关考验的希望，却是比巨煌更大。
<br/>　　“哼，那就以事实说话吧！”巨煌冷哼道。
<br/>　　苏莫目光瞥了一眼说话的巨煌，和那名飞灵族的飞华，心中恍然，这两人均是真圣榜的天才。
<br/>　　飞华排名一百八十八，那名巨灵族的巨煌排名还要更高，乃是一百七十九名。
<br/>　　不过这两人，苏莫完全不放在眼里，他目光转动，看向了其它的平台，发现并没有强强相遇的情况。
<br/>　　如姜神夜、巨摩、飞长云、黑锋、巫殇等人，均是有各自的平台，没有互相碰到一起。
<br/>　　苏莫还看到了尤天韩，对方所在的平台好像也没有什么太厉害的人物，不过尤天韩在真圣榜的排名不高，对方想进入第三关，估计有些难度。
<br/>　　至于巫人王，苏莫当然不会忘记，此人的运气也算不错，没有碰到超级厉害的人物，但对方的平台上估计也有真圣榜天才，想过关估计也不容易。
<br/>　　苏莫倒是希望对方过关，到时候对方来挑战，他让对方有来无回。
<br/>　　少倾之后，苏莫便不再关注其他人，静静的等待考验的开始。
<br/>　　而其他各个平台上，也是渐渐安静了下来，众人均是准备接受考验，不坚持到最后，谁也不会放弃。
<br/>　　此时，真圣碑上再次青光大放，逐渐的在天穹之上化为了一片青色的乌云。
<br/>　　乌云庞大无比，方圆数千里，覆盖了所有的平台，其内有强大的力量在涌动，随时可能降下圣雨。
<br/>　　这圣雨每一滴都威力巨大，而且会越来越强，所有人都不敢小觑。
<br/>　　所有人的面色都凝重了下来，准备迎战圣雨。
<br/>　　而此时，苏莫的心中不禁活络开了，这一关是逐渐淘汰制，谁能撑到最后，谁便能过关。
<br/>　　那若是他将其他人都打下去，平台上就留下他自己，是不是就过关了？
<br/>　　应该是这样，一个平台只能有一人过关，就剩下他自己了，不过关也要过关。
<br/>　　心念一转，苏莫心中便有了主意。
<br/>　　这个办法不错，虽然他的实力强大，但这样更节省时间，而且，这些同平台的天才，每一个都是各自种族或势力的翘楚，谁知道他们身上都有什么宝物！
<br/>　　万一有的人身上有逆天的防御宝物，他岂不是有麻烦了，一旦被淘汰出局，离开了真圣界，怕不是立刻就要被大祭司镇杀。
<br/>　　轰隆隆！
<br/>　　在苏莫思索之间，天穹之上的青云之中，突然传出轰隆隆的巨响。
<br/>　　而后，一阵浩大的青色暴雨从青云中倾洒而出，向所有平台上覆盖而去。
<br/>　　这些青色的暴雨，每一滴都蕴含毁灭般的力量，如同一柄柄青色长剑，洞穿空间，从天气之上暴刺而下。
<br/>　　轰！轰！轰！！！
<br/>　　这一刻，在八百个平台之上，所有的天才均是体内玄力涌动，强大的气息冲天而起，完全做好了迎接圣雨的准备。
<br/>　　“来吧！”
<br/>　　“哈哈，小小圣雨也想击败我，简直是做梦！”
<br/>　　“看我们谁能撑到最后！”
<br/>　　苏莫所在的平台上，巨煌、飞华等人暴喝出声，同样是个个身上玄力涌动，气势冲天，每一个都强大异常。
<br/>　　“你们都下去吧！”
<br/>　　就在此刻，一声冷漠的声音响起，是苏莫开口了。
<br/>　　话音未落，他便催动了吞噬战魂，强大的吞噬之力，瞬间将平台上的二百多名天才笼罩住了。
<br/>　　而后，他手臂抬起，一巴掌挥了出去，掌劲顿时幻化成了一只小山般的巨大手掌。
<br/>　　巨大的手掌横扫虚空，扫过整座平台，如同秋风扫落叶一般。
<br/>　　嘭！嘭！嘭！！
<br/>　　霎时之间，闷响声不断，平台上的两百五十余名天才，还没反应过来是怎么回事，顿时便被巨掌拍飞了出去。
<br/>　　一众天才仿佛是抛飞的麻袋，四散而开，七零八落的飞出了平台。
<br/>　　“啊！”
<br/>　　“怎么回事？”
<br/>　　“怎么会这样？”
<br/>　　惊呼声不断，二百五十多名天才，乍一被击出平台，只来得及呼喊一声，便个个消失不见，被淘汰了出去。
<br/>　　几乎就是一瞬间，苏莫所在的平台上，就只剩下了他一人。
<br/>　　“什么？”
<br/>　　“怎么可能？”
<br/>　　“卧槽，这是要干什么？”
<br/>　　其它平台的人，见到苏莫突然出手，以雷霆之势，将所在平台的其他人全部打了出去，顿时一脸的懵逼。
<br/>　　一时间，他们都是有些懵圈，没反应过来苏莫为何会如此。
<br/>　　但是下一息，所有人都愣住了，只见天穹之上爆射下的圣雨，那本该覆盖到苏莫所在平台的一部分，突然就消失不见了。
<br/>　　这一瞬间，所有人都明白了——苏莫已经过关了！
<br/>　　蓦地，其它所有平台上，众人顿时就炸开了锅。
<br/>　　“尼玛，还能这样？”
<br/>　　“卧槽，这就过关了？”
<br/>　　“这……这怎么可能？”
<br/>　　“还带这样的？不用承受攻击就过关了，这是作弊啊！”
<br/>　　所有人均是目瞪口呆，现场一片哗然，这种情景让所有人都感觉有些荒诞。
<br/>　　这特么还能这样？
<br/>　　在圣雨的攻击还没到来之前，将所有人打出去，然后就剩下自己，这样就过关了？
