　　大家聊起在宿舍的那段日子——虽然对叶少阳来说很是短暂，但却是人生中一段十分难得的经历，他很怀念。
<br/>　　听他们聊着工作，聊到生活上的压力，工作啊，房子啊，还有父母催婚什么的，这些烦恼在叶少阳听来却是一种难以奢望的幸福。
<br/>　　普通人的生活。
<br/>　　“对了，少阳，我们公司最近麻烦事挺多的，老板怀疑风水有问题，想找个高人看看，你有没有兴趣我给你介绍下，报酬应该不错。”
<br/>　　李多热心地问道。
<br/>　　叶少阳笑了笑道：“过完年吧，到时候再说。”
<br/>　　他不好意思说，自己早就不干这种事了，倒不是不喜欢钱，而是……真的没工夫啊，这种江湖术士的行为是再也不干了。
<br/>　　这次聚餐大家都吃得很开心，但是散了之后，就还是各忙各的，叶少阳清楚，大家根本不是一个世界的人，在一起也没有什么交集，回想自己当初刚下山不久，接连处理的几宗的灵异事件，那时候还要接触到不少普通人，而现在，基本上除了身边这帮人，自己已经很少跟别人打交道了。
<br/>　　但他并不觉得孤独，反而很充实，因为有这么一帮兄弟姐妹，他现在最怕的，就是像失去秋莹一样失去他们。
<br/>　　聚会的这天下午，叶少阳在家里继续鼓山海印，结果一个意想不到的人来了：小九。
<br/>　　叶少阳十分兴奋，跟她聊起来才知道，她是来陪自己过年的。“空界最近没什么战事，但是军师说，尸族越是平静，就越是有问题，再加上阴间已乱，尸族不可能不行动，怕是正在酝酿一场大的战争。真打仗起来，我肯定又要忙碌，正好最近也要过年了，我就找机会陪你几天。”
<br/>　　平时不爱说话的小九，讲述自己来人间的理由讲了一大串，叶少阳知道她是出于害羞，毕竟小青小白他们都还在空界，她一个人过来，也是有点害羞。
<br/>　　叶少阳也不说破，两人就在房间里说了一会话，叶少阳提出带她去逛夜市，小九欣然接受，让他等一等自己去换衣服，然后去了卫生间，出来的时候，叶少阳目瞪口呆：小九换上了自己之前给她买的那套露脐装、短裙、高跟鞋和丝袜。
<br/>　　“我来的时候，就把这些衣服带上了，喜欢吗？”小九亭亭玉立地站在叶少阳面前，表情有一点害羞。
<br/>　　叶少阳狂咽口水，眼睛都直了，很快就面红耳赤，不由自主地就挠起后脑勺来，“那个……好看是好看，但现在是冬天，你穿这个出去……不合适。”
<br/>　　“我又不怕冷。”
<br/>　　“你是不怕，但别人看着怪怪的，会把你当傻子的，还是换一身吧。”叶少阳往她穿着丝袜的两条大长腿上瞟了一眼，又是一阵心跳加速。
<br/>　　小九道：“那怎么办，我就带这一套人间的衣服，我不喜欢变衣服穿，感觉跟没穿一样。”
<br/>　　叶少阳也有点为难，突然想到什么，打开衣柜，里面全是芮冷玉的衣服，冬装也有不少，正要拿，突然想到什么，转头不太好意思地问小九：“那个，这是冷玉的衣服，你可以先穿，你……不介意吧？”
<br/>　　小九笑道：“她是我主母，有什么介意的，只要她到时候不生气就行。”
<br/>　　叶少阳找了一身衣服给她，小九换上，看上去没有一点妖精的痕迹，倒像个楚楚动人的人间女子。
<br/>　　但叶少阳的心好像被什么扎了一下，立刻沉默了。
<br/>　　小九抿着嘴巴，冲他勉强笑了笑，柔声道：“你是不是想主母了，不然我去山河社稷图里找她？”
<br/>　　叶少阳伸出双手，摸着她的脸蛋，怜爱地说道：“你以后记住了，她是我未婚妻，但你不必把她当成主母。你是你自己。”
<br/>　　小九听了这话，心里很感动。
<br/>　　看着芮冷玉的衣服穿在小九身上，叶少阳心中浮起了一股强烈的负罪感，但事已至此，他不知道将来怎么去处理跟她们的关系，也只有努力让自己开心起来，享受这一切。
<br/>　　他带小九去了商场，这次不敢擅作主张，找了一家最大的店，让店员帮小九选配衣服，店员从没见过这么漂亮的姑娘，几个店员都围上来，一起出主意，最后搭配了两套冬装，老板娘还找来数码相机，非要她拍几张照片，抱怨说他们品牌是范凉凉代言的，最近饭彻底凉了，把海报都撤了。今天难得见到一个漂亮妹子，想把照片发在朋友圈做广告，声称效果一定好。
<br/>　　还要加小九的微信，约着改天来拍一些照片，最后叶少阳加了微信，带着小九逃离了这份热情。
<br/>　　“你之前有点狼狈，为什么呀？”下楼的时候，小九好奇问道。
<br/>　　“额，他们都把我们当成情侣啊，你这么漂亮，我压力就很大，你没看他们看我的眼神就很怪，他们一定认为我很有钱……”
<br/>　　小九噗嗤一笑。
<br/>　　叶少阳让她站好，自己在远处打量了一下，黑色毛衣，蓝色牛仔裤，外面是一件米黄色的风衣，不得不佩服那些店员的眼光，这一身看着简约不花哨，又透着青春洋溢的感觉，搭配小九的颜值，简直……叶少阳想起了网上那些段子，在那些色狼眼中，绝对是死刑不亏十年血赚啊。
<br/>　　买好了衣服，两人就从商场逛下去，小九很少来人间，上次逛商场还是谢雨晴带的，对琳琅满目的店铺保持很强的好奇，几乎是过一家看一家，这次跟叶少阳一起来商场，心情又不一样，挽着退叶少阳的胳膊到处逛，来到一家娱乐城，看里头很多人在玩，叶少阳自己也不熟，就带小九进去，换了些币，学着别人的样子玩那些游戏，跳舞机，赛车，投篮什么的，玩的不亦乐乎。
<br/>　　从娱乐城出来，路过奶茶店，小九又买了一杯橙汁，喝了一口，抬头冲叶少阳笑起来，“怪不得妖精都要修成人形，人间的确太多好玩的东西。”
<br/>　　“好玩的还多着呢，我以后可以带你慢慢去发现。”
<br/>(本章完)
