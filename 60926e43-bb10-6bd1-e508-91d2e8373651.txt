　　就在成汤天乙攻打天下之时，昆仑山中，却发生了一件大事。
<br/>　　通天教主竟然搬出了昆仑山，去了东海金鳌岛，在那里建立了碧游宫。这件事情的起因便是封神榜一事，经过此事，通天发现他们三兄弟之间不再如之前那般亲密了，他们之间或多或少地多了一点隔阂，不仅仅是因为道念不同的原因，也许之前就有了，而封神一事只是起到催化作用。
<br/>　　当通天说出这件事的时候，元始还是挺惊讶的，不知道通天为何会做出如此决定，不过，他也没有多说什么，这是通天的决定，他也不想多管。但是，当天晚上他却与通天谈了一晚上的话，谁也不知道他们在谈些什么，第二天通天便带着截教众仙出了昆仑山，去了东海。
<br/>　　在通天走后，太清也带着玄都走了，去了混沌中开辟他的道场大赤天，建立了八景宫。用太清的话来说，便是昆仑山不小，但也不大，一闪不容二虎，更何况是三教，如今通天走了，他也该走了，去开辟他的道场了。
<br/>　　两人走得倒是潇洒，只留下元始苦笑不已，不过，他也仅仅只是郁闷一下，便将这些繁琐之事抛之脑后，两人走了也好，这样昆仑山也只留下玉虚宫一家，倒也宽敞。
<br/>　　就这样，东昆仑只剩下了阐教，而西昆仑倒是还有西王母，以前还有一个厉害的散仙，却不知何时消失了。至于那个散仙是谁，元始心中多少有些猜测，那人的辈分高得有些吓人，他不在了元始也省心些。
<br/>　　三清分家，在外人看来是三清闹别扭了，不复之前那般团结了。对此，自认有人高兴有人愁，提别是西方教，两位圣人便十分高兴，因为他们知道，只有三清出现分裂，他们西方教才会有机会在东方谋算。
<br/>　　三清原本是一家，如今一家不复，三清便不再如之前那般可怕了。
<br/>　　……
<br/>　　五十年后，分布在洪荒各地的阐教众仙也相继回来了，元始立刻召集众仙开会。天尊殿中，广成子、云中子、太乙、玉鼎、灵宝、道行、慈航、孔宣等人都在，就连南极仙翁这个记名弟子也在。
<br/>　　元始扫视众人一眼，见各自都没有太大变化，便开口道：“此次唤尔等归来，是要告知尔等一件大事。洪荒之中，量劫再起，此次量劫范围之广，囊括天地人三界，一切众生，皆在其中，特别是汝等三教弟子，还有西方教等，都为重点对象。若不能度过此劫，轻者陨落，死后封神，重则形神俱灭，不得轮回。”
<br/>　　“什么？老师，这可如何是好？”广成子等人皆是大惊。
<br/>　　“嗯，封神量劫，神仙杀劫么？”燃灯道人眼睛一眯道。
<br/>　　元始见此，面色不变，心中却在思索，也是时候给他们一些压力了，好让他们知道，不是圣人弟子便不会有劫难临身，圣人弟子，一样会死，让他们那日渐骄纵的心收敛一些。
<br/>　　看了一眼众仙，元始沉声道：“尔等不可懈怠，此次大劫，虽说是大劫，但又何尝不是机缘，若是汝等能够度过，必定修为大进，但若是度不过，便等着上封神榜吧，以后修为难进，还需供天庭驱使。”
<br/>　　众人一听，脸色骤变，身为阐教弟子，又怎能让天庭任意驱使，这是在侮辱他们。
<br/>　　这时元始又道：“不过，尔等也不必太过担忧，汝等有众多法宝在身，想必遇到准圣都能斗上一二，不会轻易丢掉性命。当然，也不能放松，因为大劫随时可能降临。”
<br/>　　“传令，自即日起，尔等阐教弟子可在外收些资质上好之人做徒弟，传我大教道统，也可在大劫之中，转移劫气。只有做到有神可封，神位圆满，量劫自然过了。尔等不可轻易出山，若是不听劝告，在大劫之中身陨，别怪为师不顾尔等生死。”
<br/>　　“是，谨遵老师教诲！”
<br/>　　众阐教仙人心中一凛，恭敬道。下山收取弟子，找人替代应劫，这也是一个不错的办法，既然都需要人来填满这神位空缺，别人上去总好过自己上榜，毕竟死道友不死贫道。
<br/>　　“嗯，还有，南极仙翁？”元始突然道。
<br/>　　“弟子在。”人群之中，走出一个老翁形象，手持桃木杖，额头明显凸出的仙翁，对着元始恭敬道。
<br/>　　“为师已为你签订一神位，乃六御之一的南极长生大帝之位，此次大劫你也不必担忧。”元始对他淡淡道。
<br/>　　“多谢老师！”南极仙翁闻言大喜，他一生修为也就平常，只是个太乙金仙，在众弟子之中是最差的存在，如今得了一个大帝之位，有天庭气运加身，以后说不能更进一步，总比现在好过，况且，天庭也不差。
<br/>　　“嗯。”元始点点头。
<br/>　　“咦，玉鼎，你身后之人便是你的弟子么？”元始突然看向玉鼎身后，那里站着一人，那人模样二十来岁，面目俊俏，身穿白衣，额头之上有一道银色道痕，最为明显，为他增添几分威严。
<br/>　　“不错，老师，这是弟子新收的徒弟，杨戬。”玉鼎对着元始说道，接着又转身对着那青年说道：“杨戬，赶紧对你师祖行礼。”
<br/>　　说到此处，玉鼎心中还是多少有些忐忑的，因为这杨戬是天庭昊天的妹妹瑶姬之子，瑶姬背着天庭，私自下凡，更违背天条嫁了一个凡人，生下三人。天庭知道后，派人前来捉拿。这杨戬便是在逃亡之中被他救下，他见其资质非凡，便收为徒弟。他此时也不知道元始的态度，毕竟杨戬与天庭有关，心中多少有些不安。
<br/>　　“见过师祖！”杨戬闻言，连忙对元始行大礼拜道。
<br/>　　“嗯，天生天眼，神人血脉，资质还行，可为阐教护法。”
<br/>　　元始却没有去管那些恩怨，对着杨戬点头道：“你可以去阐教仙神楼与三界楼中选一些功法秘典。吾观你炼体一道颇有些资质，你可以去选《九转玄功》或者《玄黄不灭》。”
<br/>　　“多谢师祖！”
<br/>　　杨戬闻言，大喜，师祖没有怪罪他，也没有驱逐他，他以后也有强大背景，可以与天庭对抗了。
<br/>　　玉鼎真人也同时松了一口气。
<br/>　　“嗯，好了，你们各自去吧。”元始对着众仙说道。
<br/>　　于是，阐教众仙带着一丝忐忑，纷纷下了昆仑山，各自回去道场之中，或闭关，或收徒弟，或游历……
<br/>　　与此同时，截教之中，通天开始讲道，讲道之后，对他的徒弟们又是警告一番，让众弟子多留心眼，不要轻易入劫，否则身死道消。
<br/>　　可是，截教仙人众多，形色各异，又哪里受得了束缚，对于通天的话，有些听在心里，而有些却是不以为然，依然我行我素。
<br/>　　通天心中对此有数，他也有心借此机会整顿截教之中的一些蚜虫，但对于亲传弟子，他却是下了严令，让他们闭关苦修。
<br/>　　其余各教之中，掌教也或多或少的告诫一番。洪荒之中陷入了短暂而诡异的寂静之中，量劫正在慢慢到来。
<br/>　　……
