        童筱颖沉默了许久，眸光微晃，黯然神伤地说道：“我和何明旭是高中同学，高考后的那个暑假确定恋爱关系，何明旭在跟我谈恋爱的同时，也在跟林若琴谈恋爱。是马智杰带我去揭发的他两，然后我和何明旭分手了。我直到马智杰死后，才从马智杰的父亲那儿得知，他一直在暗恋着我。我觉得他是个好男孩，不应该……”
<br/>
<br/>        童筱颖说着说着，便说不下去了，将脸偏向了车窗那边。
<br/>
<br/>        傅夜沉耐心地听着，凝眉沉思。
<br/>
<br/>        两人又沉默了一会儿，傅夜沉才开口说道：“马智杰的事情，我已经让警察局里的人在重新彻查了，所以，在这未出任何结果之前。筱颖，我希望你不要再乱来！”
<br/>
<br/>        “大哥，你相信我的话了，是不是？马智杰的死，另有原因！所以，你愿意帮我了，是不是？”童筱颖不禁喜上眉梢。
<br/>
<br/>        傅夜沉抬起手来，爱怜地捏了捏童筱颖的下巴，顺手在她鼻梁上宠溺地刮了刮：“我一直都相信你啊！我的傻丫头！”
<br/>
<br/>        蓦地，童筱颖身心一怔，再次挪动着身子与傅夜沉保持着一定的距离。
<br/>
<br/>        他刚刚说那番话，莫名其妙地让她想到了昼景。
<br/>
<br/>        昼景也喜欢说她是个“傻丫头”……
<br/>
<br/>        童筱颖心情有些复杂地看向车窗外。
<br/>
<br/>        傅夜沉也意识到自己的失态，一手撑在车门上，手背抵着脸颊，目光看向了另一边。
<br/>
<br/>        他和她之间夹杂着“大哥”与“弟妹”的关系，让他不得不对她若即若离。
<br/>
<br/>        车子行驶得很快，夜色渐深，他得带她回滨江城。
<br/>
<br/>        昨晚没睡好，白天又在折腾，童筱颖坐在车内昏睡了过去。
<br/>
<br/>        一旦她累到睡着了，睡眠一般都会很深，雷打不醒的那种。
<br/>
<br/>        傅夜沉让司机把车开回了他在滨江城的私宅，到家后，他轻轻地将童筱颖从车内抱了出来。
<br/>
<br/>        他对童筱颖的呵护与宠爱，连华生全看在了眼里，倒是心知肚明。
<br/>
<br/>        傅夜沉将童筱颖放到床上后，刚从童筱颖的房间里出来，就被不知何时守候在房门口突然“窜”出来的连华生给吓了一跳。
<br/>
<br/>        连华生不打算过问童筱颖的事情，而是双手抱臂，倚着门边的墙壁，说起了正事：“夜惑里的那个制药师全招了，他的师父就是龙半夏！”
<br/>
<br/>        “你的意思是说，阿景的母亲，有可能是道上的卧底？”傅夜沉剑眉微蹙。
<br/>
<br/>        连华生点了点头，接着问道：“我们的调查，已经打草惊蛇了。要不要弄个幌子出来，掩饰一下？”
<br/>
<br/>        “启动第二套计划吧！”傅夜沉首肯道。
<br/>
<br/>        连华生抬起手来，捏了捏自己的下巴，若有所思地说：“夜惑里的那个制药师，其实是制蛊师。而且，他只会两种蛊，一种就是你中的冥阳蛊，助兴与避孕用的。另一种就是，麻魂蛊，用来麻痹人的大脑思维服从指令的。你说，他们真的只是这么简单吗？”
<br/>
<br/>        “这种事，他们查，你盯着！最近，我公司里有点儿乱，我得先管公司。”傅夜沉头疼地揉了揉太阳穴。
<br/>
<br/>        连华生知道傅夜沉的工作压力大，偏偏爱情上又不如意。
<br/>
<br/>        “那你早点休息！”他拍了拍傅夜沉的肩膀，只能在心里替他感到惋惜。
<br/>
<br/>        “你也是！”傅夜沉点了点头，看着连华生上楼去的背影，欣慰地笑了笑。
<br/>
<br/>        他庆幸自己有一批能和自己出生入死，肝胆相照的兄弟。
<br/>
<br/>        或许，真的是什么样的出身，决定了步入什么样的路，同时也决定了以后会邂逅什么样的人。
<br/>
<br/>        傅昼景自从被朋友背叛，害得自己负债累累，现在又不得不拼命工作，还得跟一个女人耍心机，他便开始打心底里的不再信任任何朋友。
<br/>
<br/>        季思妍感觉到傅昼景在离她越来越远，追根究底，她从来不会在自己身上找原因，而是把所有的过错全都推到了童筱颖身上。
<br/>
<br/>        如果，让阿景发现，童筱颖“出轨”了，以阿景的那颗心思，定会对童筱颖大失所望。
<br/>
<br/>        季思妍陪着傅昼景出差时，傅昼景还在外地忙碌，她却找了个借口，提前回到了滨江城的家中。
<br/>
<br/>        她只有趁着傅昼景不在滨江城的这段时间里，才有机会操作自己已经计划好的事情。
<br/>
<br/>        季思妍把童筱颖的资料，递给了哥哥季运恒。
<br/>
<br/>        “哥，你不是最会玩女人了吗？我要你帮我玩这个女人！”季思妍将童筱颖的寸照举到了哥哥季运恒的眼前。
<br/>
<br/>        季运恒正躺在沙发上，嘴里叼着根香烟，手里拿着手机正和一个女人聊天打情骂俏。
