　　龙族大长老龙昌离开龙都城有可能带走了苏莫，这个消息被天王宫得知，而地王宫同样得知了。
<br/>　　不过，地王宫并未派人去追踪，一则是拓跋青云在闭关疗伤，而地渊宫主不在地王宫。
<br/>　　所以，地王宫虽然也派出了一些人手，去调查具体的情况，但是，并未出动超级强者前去追踪。
<br/>　　地王宫知晓天王宫最近动作频频，往来于各大神朝之间，地渊宫主也是不敢怠慢，同样在想办法拉拢各大势力。
<br/>　　地王宫的目标，不求能拉拢到多少势力，只求不要有那个大势力被天王宫拉拢。
<br/>　　……
<br/>　　天星神朝境内。
<br/>　　一道流光以肉眼不可见的速度，飞快的划过苍穹，撕裂虚空，最终落在了一片密林之中。
<br/>　　流光化为了一道人影，正是龙族大长老龙昌。
<br/>　　嗡~~~
<br/>　　空间一阵波动，苏莫的身形，出现在了大长老的身侧。
<br/>　　“苏莫，这里是天星神朝，到此为止吧！”大长老沉声道，他用了六日的时间，带着苏莫先是进入了混沌神朝境内，然后又绕到了天星神朝。
<br/>　　“多谢大长老！”苏莫向龙昌郑重的抱拳道。
<br/>　　“嗯！”大长老点了点头，但并没有急着离开，略一沉吟，道：“前两日，我发现好像有人追踪，不过这两日应该已经甩开了！”
<br/>　　“哦？”苏莫闻言一惊，随即心中恍然，追踪之人，应该是终极神王。
<br/>　　天王宫和地王宫的人，不太可能这么快的追来，一般人根本不可能追的上龙昌大长老，除非是天君河、地渊神王或者是帝一魂、拓跋青云等人，就一直守在龙都城之外，但这个可能性基本不存在。
<br/>　　“你自己小心一点！”龙昌叮嘱道，对于眼前的青年，他非常的看重，甚至潜力超越了龙腾。
<br/>　　“我会的，多谢大长老了！”苏莫向对方抱拳。
<br/>　　“嗯！”大长老微微颌首，随即大袖一挥，如同流星冲破苍穹，眨眼间便消失无踪。
<br/>　　见大长老离开，苏莫深深的吸了口气，并未急着离去。
<br/>　　他转头看了看四周，寂静的山林之中，偶尔有鸟鸣兽吼之身。
<br/>　　“前辈，您在吧？”苏莫轻声说道。
<br/>　　四周空空荡荡，没有半个人影，苏莫见此，眉头微微一皱，难道大长老真的将终极神王甩开了？
<br/>　　“本座说了保护你五年，自然不会食言。”
<br/>　　就在此时，距离苏莫数百丈远处，一颗古树之下，走出一道矮壮的身影，正是终极神王。
<br/>　　“呵呵，我以为您没有跟上来！”苏莫见此，不由的笑了笑。
<br/>　　“凭龙昌的速度，还不至于甩掉本座！”终极神王面色淡淡的说道。
<br/>　　“那就好！”
<br/>　　苏莫点了点头，便不再多言，辨别好方向之后，便闪身飞离了山林。
<br/>　　他的目标是时空界域，只有在那里，他才能在最短的时间之中，达到五品的修为。
<br/>　　当然，神域宝地很多，或许也有一些其他地方，能让他快速突破，但是，他只知晓时空之城，而且颇为熟悉。
<br/>　　飞行之中，苏莫改变了容貌和气息，变成了一个容貌普通的黑脸汉子，向时空界域快速而去。
<br/>　　而终极神王，身形一闪，便消失无踪，隐匿到了暗中。
<br/>　　以终极神王的修为，他暗中保护苏莫，除非有同级别的武者，不然根本无人能发现他的存在。
<br/>　　在苏莫离去片刻之后，天穹之上，一道肉眼难辨的流光，再次落了下来，停留在了半空之中。
<br/>　　“终极神王？”
<br/>　　这流光依旧是大长老龙昌，他望着苏莫离去的方向，低声自喃，心中若有所思。
<br/>　　事实上，前几日他发现有人跟踪，但这几日并不确定还是否有人跟踪，所以便在此地停留，放下了苏莫，而他并未离开。
<br/>　　只是，没想到跟踪之人，居然是终极神王，而且，终极神王居然在保护苏莫。
<br/>　　“这小子，手段不凡！”少倾，龙昌大长老暗暗点了点头，以苏莫的手段，说不定真的能翻盘。
<br/>　　随即，他也没有多留，转身离去。
<br/>　　……
<br/>　　苏莫的飞行速度不快，一边赶路，一边修炼，耗费了近四个多月的时间，才赶到时空界域的极空城。
<br/>　　在两个月的时间，他和人德神王联系了一次，对方向他保证，会尽最大的力量，得到更换神阵阵基的资格。
<br/>　　想要得到这个资格，必须是门下弟子，在千选战之中夺得第一。
<br/>　　人德神王，虽然对门下弟子，没有十足的信心，但是会让门下弟子尽量的争取。
<br/>　　人德神王的底气，是叶凌天，神域九大公子之一的叶公子。
<br/>　　苏莫对此，只能告诉人德宫主，尽力而为便可。
<br/>　　极空城极为繁华，和以往没有太大的变化，但武者的数量明显更多了，苏莫大摇大摆的走进了极空城。
<br/>　　城中喧嚣震天，强大的武者成群结队，络绎不绝。
<br/>　　“听说千选战还有不到五年时间了，不知道哪个天骄能夺得最终的胜利？”
<br/>　　“这个无法预料，九大公子个个天赋超绝，并且，有很多其他的万古天骄，修为都达到了九品的地步，比之九大公子都有希望。”
<br/>　　“永恒神宫的公孙公子希望很大啊！”
<br/>　　“我一直不明白，这些大势力，为何要争夺更换阵基的权利，有何好处？”
<br/>　　“这你就不懂了，神域的万古神阵，是有主阵基的，布置神阵的势力，可以将主阵基连通自己的地盘，日后的资源不是其它地域所能比拟的。”
<br/>　　极空城之中，嘈杂的声音四起必伏，但是，大多数的人，都是在讨论神域更换阵基之事。
<br/>　　因为，这关乎到每一个人，阵基更换之后，神域的资源、灵气等，都会大大的提升。
<br/>　　因为千选战即将到来，连前段时间苏莫和龙腾闹出的风波，都被掩盖了下去。
<br/>　　“原来如此！”苏莫走在城中的街道之上，耳中听着无数嘈杂的声音，心中也解开了不少疑惑。
<br/>　　他大步而行，毫不停留，向时空之城的入口之处走去。
<br/>　　很快，苏莫便来到了时空之城下方，那犹如一片巨大广场的空地之上，放眼望去，乌压压的人群，数之不尽。
