　　魔影分身离开客栈之后，并为大摇大摆的直接飞向阴阳神宫，而是快速的穿梭在一条条小巷中。
<br/>　　苏莫这也是比较谨慎，不想暴露自己的位置。
<br/>　　这具魔影分身，通体呈火红之色，乃是一具火属性的魔影分身。
<br/>　　苏莫并未让魔影分身的威力最大化，只是动用了一种属性的玄力，因为他也有些顾忌。
<br/>　　毕竟，以他目前的实力，最强状态的魔影分身，威力实在太强，若是自爆的话，估计会将整个天海城都给毁灭了。
<br/>　　天海城可是一座超级雄城，人口过亿，到时候将有数以亿计的无辜者死亡。
<br/>　　这是苏莫不愿意看到的情况，他虽然对于敌人狠辣无情，但还没有疯狂到，为了自己的目的，而屠杀上亿无辜之人的地步。
<br/>　　所以，他这次动用的魔影分身，实力控制在了一个相对较小的程度，差不多相当于武皇境三重武者，可以毁灭整个阴阳神宫，但又不会对天海城造成太大的破坏。
<br/>　　唰！唰！
<br/>　　魔影分身爆发出了最快的速度，不断的在一条条小巷中穿梭，不多时便接近了阴阳神宫。
<br/>　　嗖！
<br/>　　魔影分身没有任何停留，以最快的速度，闪电般的从阴阳神宫的大门中冲了进去。
<br/>　　阴阳神宫的大门之前，把守的十数名守卫修为并不是太高，最高之人不过是武王境巅峰修为，根本来不及阻拦。
<br/>　　甚至，他们都没看清魔影分身，只看到一道影子闪电般的冲进了神宫中。
<br/>　　“什么人？”
<br/>　　“敌袭！”
<br/>　　“快拦住他！”
<br/>　　霎时之间，一声声大喝声响彻而起，十数名守卫大惊失色，急忙向魔影分身追去。
<br/>　　而这时的魔影分身，已经急速冲进了阴阳神宫之中。
<br/>　　“碎！”
<br/>　　就在此时，阴阳神宫深处，蓦然飞出一只玄力大手，大手撕裂虚空，闪电般的向魔影分身抓去。
<br/>　　这只玄力大手，长约数十丈，劲力极为凝练，威势滔天。
<br/>　　显然，阴阳神宫中的超强存在，已经第一时间发现了魔影分身，立刻便出手了。
<br/>　　不过，魔影分身乃是由苏莫控制，怎么可能让对方抓住？
<br/>　　就在这一刹那，魔影分身的躯体鼓荡，猛然爆开了。
<br/>　　轰！
<br/>　　一声惊天动地的爆响，响彻方圆数千里，剧烈的爆炸，让得整个天海城都距离的震动了起来，不少建筑纷纷倒塌。
<br/>　　而阴阳神宫首当其冲，魔影分身爆开，犹如核弹爆炸，狂暴的玄力横扫八方，碾碎一切。
<br/>　　轰隆隆！！
<br/>　　气浪横扫，大地崩碎，建筑灰飞烟灭，那些追来的守卫、以及很多阴阳神宫中的人，直接被炸的连渣都不剩。
<br/>　　“可恶！”一声怒喝声如雷霆炸响，从苍穹神宫深处响起。
<br/>　　只见那只抓来的玄力大手，急速延伸，瞬间膨胀到方圆近千丈，狠狠的扣压而下。
<br/>　　轰隆隆！！
<br/>　　一阵轰隆隆的巨响，爆炸的中心被玄力大手遮盖，爆炸之力也被压制住了。
<br/>　　不过，爆炸之力已经席卷八方，玄力大手只能压制住爆炸的威能，却不能彻底免除破坏。
<br/>　　那席卷出去的冲击波，扫荡而出，将大半个阴阳神宫都破坏了，无数的建筑化为了齑粉，无数武皇境以下的阴阳神宫弟子惨死当场。
<br/>　　啊啊啊！！
<br/>　　凄厉的惨叫声不断，魔影分身的威力虽然没有多强大，但对于武皇境以下的武者来说，却是非常致命的。
<br/>　　“混蛋！”
<br/>　　怒吼声惊天动地，阴阳神宫深处，一名身穿锦袍，面容英挺，气度非凡的中年人身形冲天而起。
<br/>　　此人，正是阴阳神宫宫主，阳鼎天。
<br/>　　望着下方一片狼藉的阴阳神宫，望着那无数惨死的阴阳神宫弟子，阳鼎天面上怒气冲天，眼眸中爆露出强烈的杀机。
<br/>　　这个时候，爆炸的冲击已经过去了，冲击波也已经随风消散。
<br/>　　大半个阴阳神宫，已经变成了一片废墟，而在爆炸的中心，魔影分身之前爆炸的位置，出现了一个方圆数里的大坑。
<br/>　　若不是玄力大手的压制，这个大坑估计还能放大十倍。
<br/>　　这让阳鼎天如何不怒？
<br/>　　“是谁？”
<br/>　　阳鼎天伫立在高空上，怒声大喝，他眼眸如电扫视四方，灵识如潮水般探出，瞬间笼罩住了整个天海城。
<br/>　　其实，阳鼎天知道，此事必然是苏莫和海明江所为，只不过他没有明说而已。
<br/>　　用灵识将整个天海城扫荡了一遍，阳鼎天并没有发现苏莫和海明江的所在。
<br/>　　苏莫和海明江的实力，不弱于他，若是两人有心隐藏，天海城上亿人口，短时间内他很难找到两人。
<br/>　　阳鼎天沉思了片刻，眼眸中闪过一丝残忍的笑意。
<br/>　　就算找不多苏莫和海明江，但两人只要来了天海城，就必死无疑了，天罗地网也可以启动了！
<br/>　　“什么人攻击阴阳神宫？”
<br/>　　“肯定是天涯海阁报复了！”
<br/>　　“很有可能，天涯海阁阁主还活着，可是准帝级的强者！”
<br/>　　“……”
<br/>　　整个天海城都沸腾了，无数的议论声此起彼伏，无数的武者犹如过江之鲫，纷纷向阴阳神宫的方向而去，前去观看情况。
<br/>　　~~~~
<br/>　　客栈房间之内，苏莫和海明江两人，将自身的气息收敛到了极致，并且两人身上留露的气息，已经改变了。
<br/>　　加之两人容貌已经大变，不仔细查探，就算是准帝强者，也很难发现两人。
<br/>　　“海阁主，好像阴阳神宫中只有阳鼎天一人！”苏莫低声说道，他已经感受到一股浩荡的灵识扫荡而过，所以才开始说话。
<br/>　　刚才阴阳神宫遭受到这么猛烈的攻击，按理来说，若是有其他准帝在此，或者有什么埋伏，应该也要暴露出来了吧！
<br/>　　但现在，只有阳鼎天一人现身，并没有其他准帝强者。
<br/>　　“苏宫主，虽然老朽很想直接杀过去，但现在还不是时候！”海明江微眯着眼，目光透过窗口，望向阴阳神宫的方向。
<br/>　　“阴阳神宫也有护宗大阵，这个时候居然还不开启？显然有恃无恐！”海明江又道。
<br/>　　苏莫闻言点了点头，略一沉吟，道：“那我们走吧！返回苍穹神宫！”
<br/>　　“这……这就离开？”海明江闻言一怔，没想到苏莫居然这么果断？就这么甘心离开。
<br/>　　“给我两个月的时间，两个月之后，阴阳神宫我们随意拿捏！”苏莫淡淡的说道，他有信心在两个月之内，突破到武皇境四重。
<br/>　　回到苍穹神宫，又厚土塔的重力领域修炼，两个月时间突破，完全不是问题。
<br/>　　到时候，什么阴阳神宫、黄泉魔宗、北堂世家，通通都要毁灭。
<br/>　　“这……好吧！”海明江略有些迟疑，但还是点了点头，他也知道苏莫的实力进展极快，一日千里，时间越是拖后，对他们越是有力。
<br/>　　随即，两人也没耽搁时间，立刻便准备离开天海城。
