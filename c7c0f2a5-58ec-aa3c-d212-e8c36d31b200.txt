　　嘭的一声巨响。
<br/>　　吞灵冰鲸冲天而起，卷起百丈惊涛骇浪。
<br/>　　萧逸等五人脚下冰块，瞬间破碎。
<br/>　　五人早有准备，连忙跳跃而起。
<br/>　　“来了个大家伙。”萧逸轻笑一声，脚步在百丈巨浪上轻点，稳稳而站。
<br/>　　“交给我。”钟无忧大喝一声。
<br/>　　“参天树，现。”
<br/>　　一颗参天巨树，凭空而现。
<br/>　　那遮天蔽日般的茂密枝条，瞬间包裹了半空中的吞灵冰鲸。
<br/>　　“吼。”吞灵冰鲸发出如妖兽一般的怒吼。
<br/>　　庞大的身躯，剧烈抖动，意图挣脱参天树的枝条，逃回冰海中。
<br/>　　“孽畜，休想跑。”凌羽大喝一声。
<br/>　　双脚玄妙一动，踏水而来。
<br/>　　“星影剑阵。”
<br/>　　凌羽手中星影剑重重劈下。
<br/>　　无数星光，陡然而生。
<br/>　　星光之下，无数柄锋利剑刃，自成剑阵。
<br/>　　剑阵之内，仿佛蕴含一股极其强悍的禁锢之力。
<br/>　　原本在参天树枝条中剧烈挣扎的吞灵冰鲸，顿时挣扎幅度大减，只能象征性地抖动几下。
<br/>　　“寒冰缚。”白冰雪同样大喝一声。
<br/>　　手中一道寒霜打出。
<br/>　　寒霜飞跃至半空，渐渐化作一条寒冰灵蛇。
<br/>　　寒冰灵蛇往吞灵冰鲸而去，随后自动环绕，一圈又一圈，将吞灵冰鲸彻底束缚。
<br/>　　数秒后。
<br/>　　寒冰灵蛇，再次化为一抹寒霜。
<br/>　　不过一秒，整条吞灵冰鲸，被彻底冰封，动弹不得。
<br/>　　……
<br/>　　高空中，几个冰宫长老点头一笑。
<br/>　　对于吞灵冰鲸的出现，他们早有预料。
<br/>　　“第一头吞灵冰鲸出现了，且被裂天剑宗天骄所擒。”一冰宫长老笑了笑。
<br/>　　“真不愧是剑道武者。”
<br/>　　“他们在这极寒之地，实力被减弱不少，尚且有如此战力。”
<br/>　　“若是在外头，其战力之强，可想而知。”
<br/>　　旁边两位冰宫长老笑道，“裂天剑宗，可是第一武道圣地。”
<br/>　　“其内剑道强者，自是不凡。”
<br/>　　……
<br/>　　下方，被彻底束缚冰封的吞灵冰鲸，从半空中落下。
<br/>　　吞灵冰鲸，其实力，与七级妖兽冰鲸一模一样。
<br/>　　但，萧逸还不曾出手，就已经被白冰雪三人轻松拿下。
<br/>　　这就是剑宗弟子的战力。
<br/>　　虽这三人的修为不过地元七八重，但战力，绝对达到天元境。
<br/>　　顺带一提，这第一轮的规则，是有名额的武者间，不许互相干扰。
<br/>　　但这并没有不允许同势力间的武者联手。
<br/>　　萧逸五人，开始吸取这头庞然大物的精纯灵气。
<br/>　　对比没有任何战力的吞灵冰鱼。
<br/>　　吞灵冰鲸不仅实力强，身躯庞大，最重要的是，其蕴含的灵气精纯度，也大大上升了一个层次。
<br/>　　这些精纯灵气，本就是由极光氤氲之气，以及极寒精华而成。
<br/>　　再加上进入吞灵冰鲸体内，得到酝酿，压缩等等，其精纯程度，相当骇人。
<br/>　　上方，冰宫执事，已经给五人记录了分数。
<br/>　　捕获一头吞灵冰鲸，比捕获一天的吞灵冰鱼加起来的分数还要多得多。
<br/>　　不多时，待得这头巨大的吞灵冰鲸蕴含的精纯灵气，被五人吸收殆尽。
<br/>　　一股气势爆发，忽然而生。
<br/>　　这是武者修为突破的征兆。
<br/>　　而突破者，正是叶铭。
<br/>　　他的修为，终于踏入了地元境，为地元一重。
<br/>　　另一位突破的，是白冰雪。
<br/>　　她的修为，本是地元七重。
<br/>　　而来不夜城后，在客栈聚灵阵中，修炼一月有余，修为本就增加了不少。
<br/>　　处于地元七重巅峰。
<br/>　　这就是有名额武者的好处，可早早来不夜城等候和享受修炼资源。
<br/>　　并为冰缘大会的到来而备战。
<br/>　　现今，吸收了这头吞灵冰鲸的精纯力量后，修为立刻突破，达到地元八重。
<br/>　　而钟无忧和凌羽，修为本就在地元八重。
<br/>　　虽未突破，但显然也增加了不少修为。
<br/>　　五人中，只有萧逸，几乎没有任何修为上的变化。
<br/>　　这头吞灵冰鲸，虽然巨大。
<br/>　　但，五人吸收，被分成了五分。
<br/>　　之后，这五分之一的精纯力量，进入萧逸体内，又九成九都被冰鸾剑夺去了。
<br/>　　真正能转化为元力，并储存于气泉的，寥寥无几。
<br/>　　萧逸摇了摇头，没有多想。
<br/>　　继续调动体内‘冰山’力量，吸引着吞灵冰鱼。
<br/>　　时间，渐渐过去。
<br/>　　三天后。
<br/>　　这一千五百名参赛武者，所不断释放的真气或者元力，早已弥漫方圆数百里范围。
<br/>　　且，随着时间过去，其浓度，也愈发恐怖。
<br/>　　渐渐的，不仅萧逸这边吸引到吞灵冰鲸。
<br/>　　其他范围，也吸引到不少吞灵冰鲸。
<br/>　　就这三天时间，萧逸五人捕捉到的吞灵冰鲸，已不下三十头。
<br/>　　而其他范围出现，并被捕捉的吞灵冰鲸，也在近百头以上。
<br/>　　根据冰宫长老之前的解释。
<br/>　　这无边无际的极寒冰海中，吞灵冰鱼数之不尽。
<br/>　　而吞灵冰鲸，却有着一个限度，大概在数百头左右，不到五百头。
<br/>　　也就是说，一千五百人中，并不能人人都能得到。
<br/>　　吞灵冰鲸，实力强者，有缘者，捕之，得之。
<br/>　　另外，吞灵冰鲸，也是拉开分数的关键。
<br/>　　萧逸五人这边，虽擒获了三十多头，但平均每人，只有六头。
<br/>　　这次参加冰缘大会的武者中，不乏天骄和强者。
<br/>　　萧逸绝对相信，有不少人的现今分数，都不亚于他们。
<br/>　　时间，再次过去。
<br/>　　到了第五天。
<br/>　　不知为何，吞灵冰鲸出现的数量，以及密度，陡然间飙升。
<br/>　　以往几天，一个范围内，每隔一个时辰左右，才会吸引出一头吞灵冰鲸。
<br/>　　可现在，每隔半个时辰，就会出现一头。
<br/>　　待得时间渐渐过去，其频发程度，再次飙升。
<br/>　　已经变为了十数分钟就出现一头。
<br/>　　这个程度，已经相当恐怖了。
<br/>　　萧逸五人捕获吞灵冰鲸的速度和数量，大大增加。
<br/>　　其他范围的参赛武者也一样。
<br/>　　萧逸猜测着，估计是数百里范围内，一千五百名武者散发的真气和元力越来越浓郁的原因。
<br/>　　萧逸理所当然地参测着。
<br/>　　但高空中的冰宫长老，已经眉头紧皱了起来。
<br/>　　“怎么回事？历次的冰缘大会，吞灵冰鲸出现频繁度，都差不多。”
<br/>　　“怎地这次格外密集？”
<br/>　　冰宫长老们，也想不明白。
<br/>　　到了第六天一开始。
<br/>　　本在捉着吞灵冰鱼，和时不时捉一头吞灵冰鲸的萧逸，忽然脸色大变。
<br/>　　这数百里范围内，虽然武者散发的真气和元力越来越浓郁。
<br/>　　但，萧逸这个范围，始终是最浓郁的地方。
<br/>　　“小心，来了个恐怖的家伙。”萧逸低喝一声。
<br/>　　钟无忧等人，眉头一皱。
<br/>　　萧逸话音才刚落下。
<br/>　　一抹巨大的黑影，陡然间出现在萧逸范围的下方。
<br/>　　不，远不止。
<br/>　　连同附近风花月等人的范围也笼罩了。
<br/>　　不难想象，这抹黑影的主人，相当庞大。
<br/>　　“好大。”萧逸吃了一惊，随即大手一挥。
<br/>　　一股澎湃灵气，包裹着钟无忧四人，将他们四人送向半空。
<br/>　　下一秒，一头庞然大物，冲击而出。
<br/>　　海面大片冰块，瞬间破碎。
<br/>　　风花月等人，直接被震得吐血击飞。
<br/>　　萧逸岿然不动，皱眉看着面前的庞然大物。
<br/>　　高空中，冰宫长老脸色一变，“怎么可能。”
<br/>　　“那是吞灵冰鲸王。”
<br/>　　“每三十年一次的冰缘大会，只会出现三头吞灵冰鲸王。”
<br/>　　“这次的三头，早已被我不夜冰宫擒获。”
<br/>　　“怎么还会有一头？”
