　　“啊！”
<br/>　　方恒的一声声惨叫，还在响起。
<br/>　　他肉体上的伤势，不重要，重要的是心中的屈辱，对这等天才武者来说，跪在地上的屈辱，才是令他们无法接受的。
<br/>　　这一跪，将是在方恒的内心深处，留下一个永远不可能磨灭的影子。
<br/>　　将来每当方恒的实力，有所进步，想要突破的时候，今日的这一步便是会从他的脑海中回忆起来。
<br/>　　令这方恒，再也没有再进一步的可能。
<br/>　　“方恒这辈子是毁了，一辈子的成就，只怕就到此为止了，想要客服这等心里阴影，谈何容易，今日这一幕，便是如同心魔一般，将会一声跟随这他。”
<br/>　　“这种你心魔，想要克服，怎么可能？一千个天才中，都很难有一人可以克服，方恒算是彻底废了。”
<br/>　　“哎，本来以方恒的资质，晋升内门，已经是板上钉钉的一件事了，没想到今日居然是遭受如此灭顶之灾。”
<br/>　　“位列人的武者，居然沦落到了这样一种下场，也是凄惨无比。”
<br/>　　方恒在外门之中，也算是一尊风云人物了，位列人榜之中，将来有着极大希望，晋升内门的武者。
<br/>　　可是如今，却是被楚风眠这样轻易击败，跪在地上，这一幕也是令不少人，唏嘘无比。
<br/>　　“这凶人，不是我们能对付的了的，快逃！”
<br/>　　“走，我们去求武师兄出手，对付这小子！”
<br/>　　那剩下的几名风云会弟子，看着方恒被楚风眠镇压，纷纷要逃。
<br/>　　就连方恒都不是楚风眠的对手，他们这群人联手，也不可能是楚风眠的对手，纷纷急速飞了出去。
<br/>　　“想要逃？过来耀武扬威完就想要逃，哪有这么容易！”
<br/>　　楚风眠冷哼一声，突然大手一挥，周围的空间，直接被封锁住。
<br/>　　这些风云会弟子，无论如何逃，都无法逃出这片区域之中。
<br/>　　“你们全部都给我跪下吧！”
<br/>　　楚风眠大手一挥，一股灵力便是从天而将，压迫下来。
<br/>　　那些风云会弟子，一个个脸色苍白无比，大声喝道。
<br/>　　“武师兄，救我们！”
<br/>　　“救你们？就算是武风云来了，他也要跪！”
<br/>　　楚风眠冷哼一声，表情中尽是冰冷。
<br/>　　“砰！”的一声。
<br/>　　那些风云会弟子，纷纷在这一股灵力之下，全部都跪在了地上。
<br/>　　这些风云会弟子，在楚风眠的洞府前，跪上了一排，简直是极为特殊的场景。
<br/>　　“一个新人！居然敢跟我们风云会为敌？”
<br/>　　突然之间，在空中，一声怒喝猛然响起，一尊身影，顿时从天而降。
<br/>　　随着这一道身影的存在，更是一道强大到极点的拳风，突然便是从空中打了下来，目标正是楚风眠。
<br/>　　这一道拳风，居然是化为了一头猛虎一般的样子，咆哮着，轰然向着楚风眠扑了过来。
<br/>　　这一头猛虎，虽然是以拳风所化，但是其中的样子，却是栩栩如生，看起来仿佛是如同一头真正的猛虎一般。
<br/>　　这是只有当拳意，达到登峰造极的一刻，才会有着的无上玄妙手段，拳风化虎，这一拳便是蕴含有着老虎的凶猛，威力更胜一筹。
<br/>　　“镇邪白虎拳！这是武风云的镇邪白虎拳！”
<br/>　　楚风眠抬起头，便是看向天空之上，一眼便是看到了这一道高大身影。
<br/>　　此人的相貌普通，但是却蕴含一身威武霸气，绝对是一尊武道强者，这边是风云会的首领，人榜第十一位的强者，武风云。
<br/>　　“小子，你敢打伤我们风云会的人，更是让他们跪下！这简直是不将我们风云会放在眼里。”
<br/>　　“敢跟我们风云会作对的人，都要死！楚风眠，你一个区区新人，居然敢挑衅我的威严，今日我便镇压了你，让你知晓什么叫做天高地厚！”
<br/>　　武风云看着楚风眠，怒声喝道。
<br/>　　这每一句话中的语气，都是高高在上，仿佛是宣判了楚风眠的刑法一般，想要惩罚楚风眠，便惩罚楚风眠，目空一切到极点。
<br/>　　“武风云，你还真当你是一个人物了？一个小小的外门弟子，人榜都不入前十，就想要在外门中耀武扬威？”
<br/>　　楚风眠抬起头，看着武风云，言语之中也是毫不客气的说道。
<br/>　　“你们风云会平日嚣张也就罢了，今日既然敢嚣张到我的头上来，那从今日开始，你们这风云会，就彻底毁灭吧！”
<br/>　　楚风眠一边开口，一边大手一挥，一掌猛然对着空中打了过去。
<br/>　　楚风眠的这一掌，在空中，无数灵力凝聚，便是化为了一条金色真龙。
<br/>　　金龙白虎，两股力量，在空中相对，猛然之间，金龙发威，无上龙威凝聚而出，直接咬了过去，将这一头白虎，直接吞噬。
<br/>　　“毁了风云会？我倒是要看看你有没有这个本事！”
<br/>　　听着楚风眠的话，那武风云暴怒一声道。
<br/>　　“人榜前十的那些人，都不敢说毁了我们风云会，你一个区区新人，也敢跟我们风云会抗衡？”
<br/>　　“你以为你有这几分奇遇就可以嚣张了？武道境界，可不是奇遇可以弥补的！无上武道！唯武独尊！”
