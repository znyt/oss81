　　叶少阳尝试将真气运到右臂上，气过经脉的感觉有点麻，但没什么大问题了。
<br/>　　叶少阳开始专注地研究面前的绝仙剑，这把剑大概五米之内的范围，都是法阵的势力范围，就算是他也无法靠近。
<br/>　　毕竟是传说中能诛仙屠佛的上古大阵，靠个人的力量是没办法的。
<br/>　　“诛仙四剑，以诛仙剑为首，想要破阵，只有撼动诛仙剑，可惜我们都在阵中，很难破阵的。”叶少阳道。
<br/>　　“什么意思？”碧清问。
<br/>　　“任何阵法，一旦成形，被困在中间的人想破阵，只有靠实力的碾压，但就算是诛仙四剑，也有它的范围，法阵总是针对内部的，如果有强者在外面，想要破阵就容易得多！再说就算从外面也无法破阵，也可以击杀主阵之人，这是釜底抽薪的办法。”凡是涉及到法术的方面，叶少阳就比碧清懂得多了。
<br/>　　碧清想了一下，道：“我们太久不回去，瓜瓜他们会来找我们的吧。”
<br/>　　“等他们来了，只怕我们坟头草都三丈高了。”
<br/>　　“用魂印通知呢？”
<br/>　　“早试过了，这阵法逆转阴阳，阻断了一切联系。走，我们先上去看看情况。”
<br/>　　右手没事了，整个人轻松了许多，叶少阳运了一口真气，也不要碧清的帮忙，抵抗住了海水的吸力，朝上面扶起来，浮到一半的时候，正好看到小马和吴嘉伟沉下来，小马手中还提着清波，这小子已经有点意识不清了。
<br/>　　互相点了一下头，一起从水面上浮起来，上面还是惊涛骇浪，人根本站不住。
<br/>　　碧清的本尊，化生出了一张巨大的荷叶，将四人托在了水面上，不管浪头多大，荷叶也是水涨船高，一直托着他们。
<br/>　　对叶少阳的出现，鲲鹏等人十分诧异，他们更诧异地是叶少阳身下的那朵莲叶。
<br/>　　“是她！”影魅微微惊叹。
<br/>　　鲲鹏也想了起来，笑道：“上次雷池一战，倒是见过的，一朵莲花精而已，李浩然那厮的师妹。”
<br/>　　“若我师兄在这，你敢当他的面这么说？”碧清淡淡说道。
<br/>　　鲲鹏立刻有点尴尬，虽然李浩然人不在，吹个牛逼也没人来把自己怎么样，但这牛逼他自己都不敢吹，李浩然已经证了混元大道，跟自己都不在一个位面了，十个他怕也不是对手。
<br/>　　鲲鹏冷哼一声，道：“他李浩然在厉害，跟你有什么关系，我师公还是轩辕上帝呢，不比你师兄更厉害？我纵然杀了你，他也不能过来找我报仇吧？”
<br/>　　“好，我定然要宰了你这条大鱼！”碧清哼哼说道。
<br/>　　叶少阳四下看去，自己带来的那几百号人，差不多都被一圈气泡一样的灵光包裹起来，与水面隔绝，不再受到巨浪的侵袭。
<br/>　　他们一个个双手合十，跟着四个盘古僧念着咒语，表情也越发虔诚凝重。
<br/>　　叶少阳看到这一幕，只是无奈一笑，也没什么好说的。
<br/>　　一切都是为了活命，他们跟自己本来也不认识，追随自己，就是为了让自己带他们离开这世界，现在性命攸关，选择向轩辕山低头，也无可厚非，何况在他们内心深处，有至少一批人也并不抗拒归顺法术公会，当初没有归顺，更多是因为实力不行，想投诚人家法术公会也看不上。
<br/>　　人性如此，也没什么好说的。
<br/>　　“叶少阳，你以为有那荷花托举着你，便可逃离这阵法么，哈哈！”玉玑子突然开口，得意地五官都挤在了一起，“实话跟你说，夺取诛仙四剑，就是老夫出的主意！当日张无生那几个老不死的，用诛仙剑阵索拿道风，以为那就是剑阵的威力了？
<br/>　　错！这诛仙剑阵，是三届之内最强的法阵之一，全看控阵之人的实力，而今我们八人控阵，除了老夫，这七位可都是昆仑山的神仙，又有无色海相助，这等威力，比你们当日所见不知道强了多少倍！诛仙剑阵，变幻无穷，你们眼下所见，只是剑阵第一式守式而已！你们可见识好了！”
<br/>　　当下与鲲鹏四人陡然变幻手印，咒语也念得更快了。
<br/>　　法阵上空的“锅盖”突然摇摇欲坠，不断有电弧落下，在空中却化成了一道道的火风，却分作两个方向，如同太极双鱼，朝着两个不同的方向吹动，吹在人的脸上，一股灼热难当，仿佛身上燃起火焰，另一股吹来，又极为阴冷，渗入骨髓。
<br/>　　叶少阳三人急忙运转罡气抵挡，但只能护住心脉，不被气息入侵，这种感官上的痛苦却一点没有减轻，只能硬扛。
<br/>　　“啊……”清波缩成了一团球，在莲叶上翻滚起来。
<br/>　　叶少阳看了一眼，对碧清道：“你保护他！”
<br/>　　碧清并没什么恻隐之心，很不情愿，但是叶少阳说也也不好拒绝，当然又分出一片莲叶，裹住清波之后，莲叶卷起来，缩到了水中，阴阳风吹不到水里去，在里面是安全的，只不过仅限于碧清一个人。叶少阳他们几个普通生灵，一旦进入水中，神识就会被撕扯，后果比被阴阳风吹着还严重，因此只能咬牙扛着。
<br/>　　“哈哈，滋味怎么样！！”
<br/>　　玉玑子满面狰狞，得意洋洋。
<br/>　　“爽啊！”小马叫道，“桑拿浴里的冰火两重天也没这爽啊，哎，冰火两重天你知道吗，你肯定不知道，你个老处男，我赌你一辈子都没碰过女人！”
<br/>　　玉玑子气得脸色痛苦，也不再言语，继续念咒，驾驭着阴阳风，提升着强度。
<br/>　　叶少阳从兜里摸出数张灵符，用朱砂笔画符——法药一切都没湿，正常能用。然后分给小马和吴嘉伟，贴上之后，感觉好一点，但作用有限。
<br/>　　“这诛仙剑阵端的厉害……玉玑子，你果真只懂得三式咒语？”影魅问道。
<br/>　　玉玑子立刻收敛神色，冲影魅谄媚地笑起来，“好教仙长知道，我这诛仙九式，还是偶尔从古籍中得到只言片语，一共也就三式，后面六式，传说却是记载在诛仙阵图上，那东西不知哪里去寻啊。”
