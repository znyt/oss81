        而他们身后的李忆莎，此刻更像是跳梁的小丑一样，哑口无言。
<br/>
<br/>        其实，傅夜沉心里一清二楚，若不是李忆莎从中作梗，他说不定可以早点找到筱颖，让筱颖少受些苦。
<br/>
<br/>        傅夜沉牵着童筱颖下楼后，在宠物店的门口遇到了夏凌焰。
<br/>
<br/>        夏凌焰刚想跟童筱颖寒暄一下，结果，傅夜沉很不给他面子，直接将童筱颖给带走了。
<br/>
<br/>        童筱颖只是急匆匆地落下一句话：“夏工，若是还有私单，记得Call我啊！”
<br/>
<br/>        “没问题！”夏凌焰随即做了一个“OK”的手势。
<br/>
<br/>        他虽然不受那个男人的待见，但是看到童筱颖跟那男人的二婚的小日子似乎还处得不错，夏凌焰的脸上莫名其妙地露出了“姨妈笑”。
<br/>
<br/>        将目光从童筱颖的背影上收回来后，夏凌焰才想起自己是来接女朋友和宠物狗的正事，于是往宠物店里走去。
<br/>
<br/>        傅夜沉打算带着童筱颖去了另一家宠物店。
<br/>
<br/>        在去的途中，坐在副驾驶座上的童筱颖忍不住好奇地问道：“大哥，你怎么会在这？”
<br/>
<br/>        “我看到你停在店门外的车了。”傅夜沉一边开车，一边回答道。
<br/>
<br/>        他本来是打算回私宅的，刚巧看到她的车停在路边，所以他也就跟着停了车，进了这家宠物店里来寻她，谁知刚好看到李忆莎和她发生了争执。
<br/>
<br/>        或许是因为偏爱，才会偏心。
<br/>
<br/>        他并不知道事情的来龙去脉，只是见不得自己的女人受欺负，所以就给郑忠威打了通电话，说要买下那家宠物店，并让那老板亲自打电话给店长，让他们好生招待筱颖。
<br/>
<br/>        童筱颖接着又问道：“那……你认识那家宠物店的老板？”
<br/>
<br/>        “这条街的商铺，都是我私人的。”傅夜沉十分耐心地继续回答着童筱颖的每一个问题。
<br/>
<br/>        童筱颖不依不饶起来：“那个女人是？”
<br/>
<br/>        “一个无关紧要的人而已。”傅夜沉云淡风轻地回答。
<br/>
<br/>        童筱颖忍俊不禁道：“大哥，你的桃花可真多。”
<br/>
<br/>        “别瞎想了，我这棵树上，只开你这一朵桃花。”傅夜沉十分默契地随声附和。
<br/>
<br/>        童筱颖笑着打趣道：“要是我这朵桃花谢了怎么办？”
<br/>
<br/>        “你若是谢了，我这棵树也就死了。”傅夜沉忽然正色道。
<br/>
<br/>        他俊脸上的严肃，让童筱颖猛然间想起了连华生的话。
<br/>
<br/>        她童筱颖可以死，大哥不能死。
<br/>
<br/>        “大哥，要是我比你先死，你一定要好好活着。然后，再娶一个深爱你的好女人当妻子，你也好好去爱她。和她一起度过余生。”童筱颖眯眼一笑，把话说得十分认真。
<br/>
<br/>        傅夜沉却只是淡定地回答道：“你放心，黄泉路上，我不会让你一个人孤单去走。你去哪儿，我陪你到哪儿。”
<br/>
<br/>        “……”童筱颖语塞。
<br/>
<br/>        还能不能好好说话？
<br/>
<br/>        车子抵达另一家宠物店的附近后，傅夜沉停好了车，先下车后，走到副驾驶座旁，给童筱颖拉开了车门。
<br/>
<br/>        两人一起离了车，他牵着她的手，往侧前方的宠物店走去。
<br/>
<br/>        “大哥！”童筱颖忽然甩开傅夜沉的手，一边抚摸着怀里的小爱，一边意味深长地继续刚刚那个话题，“大哥一定要好好地活着，然后帮我在外婆家那块地上，建一座没有钢筋混凝土的空中花园。”
<br/>
<br/>        “说什么傻话！”傅夜沉突然将童筱颖揽入怀中，双手搂着她的腰肢，微微俯身，亲昵地抵上她的额头，在她秀气的鼻头上亲啄一下，正色地道，“没有我的允许，你不可以死。”
<br/>
<br/>        小爱被夹在两人中间，成了“单身汪”的夹心馅料，难受地“汪”了一声。
<br/>
<br/>        童筱颖连忙从傅夜沉的怀里退出一段距离，忍俊不禁地介绍道：“小爱，来，认识一下，这是你的二爹。”
<br/>
<br/>        “二爹？！”傅夜沉嘴角蹙动，一脸不悦，“难不成，它还有‘一爹’？”
<br/>
<br/>        “它‘一爹’是昼景。”童筱颖一时心直口快。
<br/>
<br/>        傅夜沉的脸色更加不好看了，严肃道：“从今以后，它只有我这一个爹！”
<br/>
<br/>        小爱此刻非常识趣地朝傅夜沉又“汪”了一声。
<br/>
<br/>        傅夜沉这才眉开眼笑地摸了摸小爱的头顶：“真乖！”
<br/>
<br/>        童筱颖甜甜地微笑着。
<br/>
<br/>        他们两一起走进另一家宠物店后，店长和服务员简直是把他俩当“财神爷”一样供着，服务态度超级好。
<br/>
<br/>        小爱去做护理和体检的时候，傅夜沉便陪着童筱颖一起在店内挑选狗屋、狗狗玩具、狗狗零食和狗粮等等，小爱能用上的所需品。
<br/>
<br/>        生活所需的东西都挑完后，傅夜沉还特意挑了一块镶钻狗牌，跟服务员要求：“把小爱的狗牌也换了。”
