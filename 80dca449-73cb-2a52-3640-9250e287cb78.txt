　　而等到各国高层得知，这些事情，竟然全都是由一人所为之时，顿时全部惊愕的说不出话来。
<br/>　　苏国，克里姆林宫之中，
<br/>　　“叶狂仙？据说他在半个多月前，和老血皇大战于珠峰之上，后来便失踪了，没想到他不但没死，竟然还干出如此惊天大事！”
<br/>　　有苏国大帝之称的苏国总统库京，望着前方传来的情报，深邃的眼眸之中，射出一道浓浓的惊骇之色，继而脸上又露出一抹笑意，
<br/>　　“真是想不到，此人竟然有如此恐怖的实力！不过，干得漂亮！”
<br/>　　“米国嚣张了这么多年，总算也有吃瘪的时候了！”
<br/>　　“不过话说回来，米国应该不会善罢甘休吧？”
<br/>　　“最好能因此挑起米国和华夏之间的大战，那就再好不过了！”
<br/>　　库京自言自语了一阵，不过等他看到下面的情报，顿时两眼一瞪，
<br/>　　“什么！这家伙屠杀了米国海军之后，竟然没有立即返回华夏，反而去了米国的本土，而且竟然还挡住了，米国六枚核弹的袭击？”
<br/>　　“这怎么可能？难道他真是神仙不成？”
<br/>　　库京的脸色，从之前的欣喜，顿时变成了深深的凝重。
<br/>　　他看到米国吃瘪，自然感到幸灾乐祸，可是如果叶尘的实力太强，对于除了华夏之外的所有大国，都将不是一件好事！
<br/>　　……
<br/>　　至于米国国会方面，自从叶尘踏入本国领土之后，整个国会已经彻底炸开锅。
<br/>　　有人主战，认为叶尘对于米国的威胁实在太大，必须不惜一切代价将其灭杀。
<br/>　　同时也有人主和，认为叶尘在进入米国领土之后，对米国普通民众秋毫无犯，显然并非是那种杀人不眨眼的恶魔，又或者心存顾忌，应该还有谈判的余地。
<br/>　　一时间，主和派和主战派争论不休，相持不下，而对于叶尘，自然也没有敢再轻举妄动。
<br/>　　不但没有再发动核弹攻击，连海陆空三军，也全部被勒令，不得对叶尘开火。
<br/>　　毕竟在之前，叶尘一人屠杀整个海军基地的场景，还历历在目。
<br/>　　普通军队对他，已经构不成任何威胁，就算派再多人，怕也不够叶尘杀的，只会白白牺牲，甚至反而有可能会激发他的怒火。
<br/>　　因而现在对于米国来说，只有两个选择，
<br/>　　要么动用百万吨级以上的大型核弹，应该可以轰杀叶尘，可是米国本土也必然会遭到牵连。
<br/>　　要么就是和叶尘讲和，谈拢条件之后，双方互不侵犯。
<br/>　　就在一众国会议员，争论不休之时，忽然有人惊呼了一声，
<br/>　　“不好！大家快看！这个恶魔的目标，好像是超能局！”
<br/>　　众人慌忙一看，
<br/>　　果不其然！
<br/>　　此时叶尘带着苏珊，已经跨越三千公里，来到米国大名鼎鼎的第九区。
<br/>　　而这里，则正是米国超能局的大本营！
<br/>　　“超能局是我们米国最重要的军事研究基地，是我们领先世界其他各国的根本，绝对不能出事！”
<br/>　　“此人必须全力灭杀！”
<br/>　　一时间，主战派瞬间又占据了上风，开始对叶尘口诛笔伐起来。
<br/>　　不过很快就又有人提出了异议，
<br/>　　“如果现在动用核弹，那岂不是把整个基地也一起毁了？而且根本无法保证，能否一击必杀！”
<br/>　　“倘若无法彻底灭掉此人，我们米国必然要面临疯狂的报复！”
<br/>　　“还是不能轻举妄动！”
<br/>　　……
<br/>　　就在众人争论不休之时，总统普朗克的助理，忽然急匆匆冲了进来，手里捧着一个透明状的手机，快步走到普朗克的跟前，
<br/>　　“总统阁下，波尔局长要和您紧急通话！”
<br/>　　普朗克点了点头，那助理当即在手机上一按，
<br/>　　唰！
<br/>　　一道3D投影出现在众人面前，赫然正是超能局局长波尔，除了人影有些虚幻意外，简直就像是真人站在众人面前。
<br/>　　“波尔局长，听说叶狂仙已经前往你们超能局，应该是要对你们出手，请你尽快将重要研究成果转移出去！”
<br/>　　面对普朗克的提议，波尔却摇了摇头，一脸绝望道：
<br/>　　“总统阁下，现在恐怕已经来不及！叶狂仙的实力，远远超过我们的预期！”
<br/>　　普朗克的脸色，顿时微微一变，
<br/>　　“怎么？莫非你也主张求和？”
<br/>　　不料，波尔再次摇了摇头道：
<br/>　　“恰恰相反！此人若是不死，我们米国维持了数十年的霸主地位，将不复存在，今后将会处处受制，所以此人必须得死！！”
<br/>　　普朗克两眼微微一瞪，
<br/>　　“这么说，你有把握杀掉此人？”
<br/>　　波尔道：
<br/>　　“总统阁下和诸位有所不知，在超能局下方，其实埋藏了五十枚核弹，只要能将叶狂仙引入基地之中，同时引爆这五十枚核弹，他就算有天大的本事，也必死无疑！”
<br/>　　米国国会众人听到这话，顿时纷纷露出一抹喜色，不过紧接着又有人担忧道：
<br/>　　“可是如此一来，超能局也将不复存在了！”
<br/>　　波尔却道：
<br/>　　“按照叶狂仙的性格，超能局原本也保不住了，还不在临死之前，让我们为联邦政府铲除这个祸患！”
<br/>　　“时间已经快来不及了，总统阁下，请诸位早下决定吧！”
<br/>　　“叶狂仙若是不死，我们米国今后百年，恐怕都将会在此人的阴影之下度过！”
<br/>　　波尔说到最后，已经开始催促起来。
<br/>　　而此时，眼看着叶尘，也已经来到，米国第九区上空，直奔超能局而去。
<br/>　　“诸位！”
<br/>　　普朗克霍然站起身来，
<br/>　　“现在真的已经到了生死存亡的时候，必须要做出一个决定了，波尔局长的这个提议，同意的请举手！”
<br/>　　普朗克声音落下，
<br/>　　片刻之后，
<br/>　　唰唰唰！
<br/>　　几乎超过八成的国会议员，纷纷举手表示同意。
<br/>　　最终，米国总统普朗克一锤定音，
<br/>　　“好！提议通过！波尔局长，那一切就拜托你了！”
<br/>　　波尔听到这话，眼眸之中，顿时射出一道浓浓的狂热之色，
<br/>　　“总统阁下，诸位议员，你们放心！我们超能局的使命，就是要摧毁一切对联邦政府有威胁的存在，我一定……滋滋滋~”
<br/>　　波尔的话还没有说完，3D投影忽然一阵晃动，人影便已经消失不见。
<br/>　　而与此同时，
<br/>　　另外一边的屏幕之上，
<br/>　　只见，
<br/>　　天空之上，划过一道长达千米的紫色剑芒，
<br/>　　叶尘此刻已经杀入军事基地！
