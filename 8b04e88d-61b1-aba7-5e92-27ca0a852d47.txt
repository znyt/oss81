　　“你找死。”
<br/>　　萧逸的冷喝声，几乎是响起的瞬间，便已身影一闪挡在了风圣壶与雾妖之前。
<br/>　　萧逸毕竟就在风圣壶之旁，故即便龙啸全速而来，还是比萧逸慢了一步。
<br/>　　龙啸疾速而来，还未能真正靠近风圣壶，已感觉到一股滔天杀意陡然加身。
<br/>　　这股杀意，异常澎湃。
<br/>　　澎湃得，龙啸心头为之一惊，身影也堪堪停下。
<br/>　　龙啸看向萧逸，身躯再次一震。
<br/>　　直觉告诉他，此刻易霄眼中的杀意，并非在开玩笑。
<br/>　　这股浓郁的杀意，即便是他当日放走虎护法时，气急败坏的易霄都不曾露出。
<br/>　　而今，却是杀意凛然，他毫不怀疑，若他真敢动那奇怪的壶子，易霄会真的与他生死相拼。
<br/>　　而且，他蓦地感觉到，这股杀意中，似乎蕴含着一股他无比熟悉的气息。
<br/>　　嗖…
<br/>　　恰在此时，东方铁云亦闪身而来。
<br/>　　“易霄分殿主，你无碍，太好了。”东方铁云脸色大喜。
<br/>　　龙啸没有动作，萧逸亦收回了目光。
<br/>　　大手一挥，风圣壶外最后那一小截‘尾巴’，也被萧逸强行吸摄入风圣壶内。
<br/>　　随后指尖一收，风圣壶回到腰间。
<br/>　　“东方殿主，毒袍虽死，但邪修祸患还未消，稍后再聊吧。”萧逸说了一声，目光，看向远处的十万邪修。
<br/>　　“好。”东方铁云点点头，“有紫炎易霄在，邪修祸患，算得了什么？”
<br/>　　东方铁云边说着，负手而立。
<br/>　　他没有出手，也很清楚不必出手。
<br/>　　十万邪修，数量虽多，但在绝对的实力之下，那不过是一堆土鸡瓦狗。
<br/>　　而且，猎妖殿主殿的强者、执法队等等，早已在外封锁。
<br/>　　龙啸，则愣在原地，眉头紧皱。
<br/>　　他还在思索着刚才那股杀意中的熟悉气息。
<br/>　　而且，现今各条龙形煞气上，毒袍身死前的操控和手段早已消散。
<br/>　　故他已然夺回了数条煞气的掌控权，雷煞、木煞，还有邪煞。
<br/>　　只是，其它的三条龙形煞气呢？血煞、毒煞，还有风煞。
<br/>　　风煞他知晓，刚才他亲眼看着被易霄‘抢’走，而血煞、毒煞，此刻却不知所踪。
<br/>　　在他的感知中，血煞和毒煞，仿佛凭空消失了一般。
<br/>　　故他愣在了原地。
<br/>　　而萧逸，则早已疾速飞行下，落到了十万邪修面前。
<br/>　　“你们聚集在一块，倒是省了易某不少功夫。”萧逸冷笑一声。
<br/>　　这十万邪修，之前也一并困在了巨木封印之内。
<br/>　　不过他们远远地躲开了五条煞气的暴走，故分毫未损。
<br/>　　只是，他们躲得了那时的五条煞气暴走，却躲不过如今萧逸的追杀。
<br/>　　“咕。”十万邪修，几乎是同时地脸色大变，亦是同时地咽了口口水。
<br/>　　紫炎易霄的凶猛，他们很清楚。
<br/>　　他们刚才也亲眼看着，他们赖以自信的毒袍，被易霄亲手斩杀。
<br/>　　毒袍冰冷而干瘪的尸体，此刻便凄惨地躺在地上。
<br/>　　堂堂毒袍，他们邪君府中声名等同传奇的护法，就这样死了，何况他们？
<br/>　　嘭…
<br/>　　忽然，滔天紫炎，弥漫长空。
<br/>　　萧逸没兴趣与他们废话，既然六龙地域内的邪修都聚集在一起了，他便一并杀了，倒省了他不少麻烦。
<br/>　　轰…
<br/>　　滔天火焰，瞬间降落。
<br/>　　嘭…同一时间，邪修中，一道屏障，陡然结起。
<br/>　　屏障，包裹了十万邪修，挡下了萧逸的火焰。
<br/>　　“虎护法？”萧逸皱了皱眉。
<br/>　　十万邪修，他并不放在眼里。
<br/>　　但里头，还是有几个位列绝世的强者，如虎护法，如三长老。
<br/>　　虎护法，可是位掌控9997道的绝世强者。
<br/>　　当初，萧逸兽手、兽脚增幅，且五色火焰圆环凝聚下，也只堪堪压他一头，费了好些功夫才将他重伤，捉到击杀的机会。
<br/>　　现在，同样如此，短时间内，也难以奈何这虎护法。
<br/>　　“紫炎易霄，想杀我六龙地域各分部邪修？没那么容易。”虎护法冷笑一声。
<br/>　　“爹，快救我。”同一时间，邪修中，龙天大喝一声。
<br/>　　事实上，明显可以看到，即便虎护法的元力屏障暂时挡下了滔天紫炎。
<br/>　　但滔天紫炎，仍旧以缓慢的速度不断压下。
<br/>　　元力屏障被烧毁冲破，只是时间的问题。
<br/>　　一旦滔天紫炎完全降临，对于十万邪修来说，那将会是屠杀的下场。
<br/>　　远处，本在思索的龙啸，听到龙天的大喝，瞬间反应了过来。
<br/>　　“易霄，你敢杀我儿？”
<br/>　　萧逸充耳不闻，仍旧操控着滔天火焰，双手虚压。
<br/>　　“真当我怕了你不成？”龙啸脸色一怒，便要出手。
<br/>　　身旁，东方铁云冷哼一声，“龙域主，龙天已是邪修。”
<br/>　　“你不怕易霄分殿主，但我猎妖殿总殿的怒火，你可承受得起？”
<br/>　　“你…”龙啸眼眸一冷。
<br/>　　远处，龙天急声大喝，“爹，快杀了易霄，刚才巨木封印内，我亲眼所见，他将我们龙家的血煞和毒煞一并吸收了。”
<br/>　　“什么？”龙啸脸色剧变，瞬间反应了过来。
<br/>　　他终于想明白，刚才那股杀意中的熟悉气息是什么。
<br/>　　正是他龙家的六煞之一，血煞。
<br/>　　难怪在他的感知中，他龙家的血煞、毒煞，都已消失，竟是已被吸收了。
<br/>　　事实上，龙啸不是没有想过这个可能。
<br/>　　只是，他总觉得不可能，觉得不可思议。
<br/>　　他很清楚六龙六煞每一条蕴含的力量有多么庞大恐怖，那逾十万丈的庞大身躯之内，可是孕育了百万年方才成长到这个地步。
<br/>　　如此庞大的力量，怎可能在短短一天多之内被吸收殆尽。
<br/>　　可现在，龙天的话，彻底坐实了他的想法。

<br/>　　手中五色圆环，陡然疾速转动，化作六色。
<br/>　　虎护法还有这十万邪修，他必杀之。

<br/>　　滔天紫炎，陡然威力大增，瞬间燃尽了虎护法的元力屏障。

<br/>　　不过短短一个呼吸间，整个高空，陡然在紫晶灵炎焚烧下化作虚无。
<br/>　　十万邪修，凭空蒸发了一般，悉数身死。
