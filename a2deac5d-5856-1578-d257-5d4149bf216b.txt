　　眼见着巨大的锤影镇杀而下，苏莫全身魔力沸腾，滚滚魔气冲天而起。
<br/>　　他手中的长剑已经化为了漆黑的魔剑，爆发出璀璨的魔光，体内魔力运转到了极致。
<br/>　　“死吧！”手臂一挥，苏莫瞬间连斩三剑。
<br/>　　咻咻咻！！
<br/>　　霎时之间，三道漆黑的剑气逆斩而上，呈品字形斩向巨大的锤影。
<br/>　　这三剑威势惊天，剑威盖世，这是因为苏莫沟通了体内的战魂和武魂。
<br/>　　他动用了郭剑青的白色长剑战魂，以及其它十种最为强大的剑武魂，所以这两剑威势直线暴增。
<br/>　　轰！
<br/>　　三道剑气瞬间斩中了巨大的锤影，毫无意外，锤影被瞬间撕成了碎片。
<br/>　　而后，三道剑气威势不减，划破长空，直击巫梁宇。
<br/>　　“什么？”巫梁宇见此顿时心中一惊，苏莫的实力出乎了他的预料，这三道剑气之强，让他心惊肉跳。
<br/>　　眼见三道剑气即将临身，电光火石之间，巫梁宇立刻将手中硕大的紫金铜锤向身前一挡。
<br/>　　下一息。
<br/>　　轰！
<br/>　　一声惊天爆响，三道魔气森森的剑气轰然的斩在了紫金铜锤之上，发生了猛烈的爆炸。
<br/>　　嘭！
<br/>　　紫金铜锤根本挡不住三道剑气，瞬间撞击在了巫梁宇的胸膛之上。
<br/>　　噗！
<br/>　　一股血箭从巫梁宇的口中喷出，冲起数十丈高，其内更是夹杂着大量的内脏碎片。
<br/>　　啊！
<br/>　　巫梁宇口中发出一声惨叫，身形直接倒飞了出去，随即嘭一声撞击在了战台边缘的阵法气罩之上。
<br/>　　哇！
<br/>　　身形止住之后，巫梁宇口中更是鲜血不停的喷出，显然受了极重的内伤。
<br/>　　全场一片寂静，几乎落针可闻！
<br/>　　所有人都是一脸的愕然，他们怎么也没有想到，巫梁宇不仅没有胜，居然还败的如此之快！
<br/>　　不过才两招而已，巫梁宇就败了！
<br/>　　巫尘刚才展露的实力，比上次斩杀巫烈和巫玉儿，还要强大不少。
<br/>　　而一些长老人物，则是个个面露惊讶之色，他们刚才分明在巫尘的身上，感受到了无比浓烈的剑魂之力。
<br/>　　这是什么战魂，剑威居然如此的强烈！
<br/>　　而此刻的巫梁宇，更是眸中露出骇然之色，巫尘的实力让他震惊不已。
<br/>　　他都没来的及动用战魂，就败了，身受重创！
<br/>　　“结束了！”
<br/>　　苏莫冷漠的声音响起，手中长剑高高举起，就欲一剑斩杀巫梁宇。
<br/>　　“住手！”就在此时，一声暴喝声陡然响起，震的苏莫脑中一阵茫然。
<br/>　　而后，从看台上伸出一只玄力大手，直接撕开了阵法气罩，将巫梁宇抓了出去。
<br/>　　出手之人，正是长老巫源。
<br/>　　少倾之后，苏莫脑中恢复了清明，顿时脸色一沉。
<br/>　　他没想到巫源居然如此的无耻，见巫梁宇败了，就将对方救了出去。
<br/>　　若是他败的话，可是只有死路一条了。
<br/>　　“长老，我和巫梁宇自愿一战，你为何插手？”苏莫冷冷的问道。
<br/>　　“巫尘，切磋而已，你居然还想要下杀手，你修炼魔功，莫非已经入魔了不成？”巫源冷喝说道。
<br/>　　一句话，巫源不仅推卸了插手的责任，还给苏莫按上了入魔的罪行。
<br/>　　“巫梁宇口口声声要杀我，难道我还要放过对方不成？”苏莫嗤笑道，他心中却是阴郁到了极点，对方这是只许别人杀他，不准他杀别人了！
<br/>　　“哼！强词夺理，总之切磋比试，点到为止！”巫源冷哼一声说道。
<br/>　　苏莫深深的吸了一口气，他也不想与对方争辩，没有什么意义！对方也不可能将巫梁宇再送上来给他杀。
<br/>　　随即，苏莫环视四方，朗声喝道：“还有谁不服？尽管来战！”
<br/>　　苏莫的喝声震荡四野，响彻在所有人的耳中，让得很多巫族之人的脸色都沉了下来。
<br/>　　实在是苏莫的姿态太狂了，简直是藐视一切，让人极为不舒服。
<br/>　　不少人心中都生起了战意，不为其它，只为将苏莫的傲气踩在脚下。
<br/>　　嗖！
<br/>　　破空声响起，有又一名武帝境三重的巫族青年飞落在了战台之上。
<br/>　　“我来与你一战！”青年高大壮硕，声音浑厚无比。
<br/>　　“败！”
<br/>　　迎接青年的只是一道惊天的剑气，长达数里，魔威滔天，剑威盖世，绝世的锋芒让得阵法气罩都出现了丝丝的裂纹。
<br/>　　只一剑，壮硕青年便被劈飞了出去，口中鲜血狂喷。
<br/>　　接下来的时间里，一名又一名武帝境三重的巫族天才上台，纷纷与苏莫交手。
<br/>　　但是，无人是苏莫的对手，没有人能在他的手中撑过三招。
<br/>　　嘭！
<br/>　　嘭！
<br/>　　嘭！
<br/>　　一名名巫族天才身受重伤，口中喷血，身躯抛飞。
<br/>　　没过多久的时间，苏莫已经连败了十几名巫族天才。
<br/>　　他出手极重，没有任何试探之心，每一招每一式都没有留手，混天魔功第一层的威力奇大，再加上他众多武魂和战魂的加持之下，就算是厉害的天才人物，也不可能是他的对手。
<br/>　　当然，苏莫只敢使用剑类的武魂和战魂，因为武斗场有不少的巫族长老人物，他也怕被别人看出他的底细。
<br/>　　但即便不用吞噬战魂，十几种剑类的武魂和战魂，也让他的战力达到恐怖的地步。
<br/>　　“还有谁不服？”又打败了一名巫族天才之后，苏莫身形挺拔，伫立在战台之上，朗声高喝。
<br/>　　他没有杀人，将所有人击败之后，便不再出手了。
<br/>　　因为他知道，有巫源和众多的长老人物在此，他想杀人很难困难了。
<br/>　　所以，他索性不再下杀手，击败之后便收了手。
<br/>　　听闻苏莫的喝声，武斗场一片安静，不少人都面面相觑。
<br/>　　众人都彻底被苏莫的实力震惊了，不少人更是心中暗想，这个以前名不经传的巫尘，是怎么崛起的？
<br/>　　他修炼的是什么魔功？威力居然如此的强大！
<br/>　　巫天御面无表情，他眼帘低垂，眸中冷光闪烁。
<br/>　　苏莫的实力虽强，但他有自信，如果自己也是武帝境三重的修为，未必弱于苏莫。
<br/>　　而巫源早已脸色铁青，一个小小的巫尘，难道真要要同阶无敌了吗？
<br/>　　“无人敢战了吗？”苏莫再次出声，目光环视四方，他已经不想再继续战斗下去了。
<br/>　　但是，既然战斗到了现在，他自然要横扫所有同阶的巫族之人，彻底扬名巫族。
<br/>　　他相信，今日之战以后，他也会进入那些巫族真正高层的视野，日后成为少司将更加容易。
<br/>　　这时，看台上所有的人，都转动了目光，望向了同一个反向。
<br/>　　那是看台的最上方，端坐着一名身穿黑袍的青年，此人身材消瘦，脸庞罩在黑袍之下，让人看不清面容。
<br/>　　苏莫见此，也转头看向此人，随即眼眸微眯了下来。
<br/>　　他搜索脑中记忆，很快便找到了此人的信息。
<br/>　　巫魔，黑耀城的巫族之中，最强大的几名天才之一，正是武帝境三重的修为。
<br/>　　此人的实力恐怖异常，杀人不见血，手段无比的诡异，不出手则以，一出手必无活口，堪称敌人的噩梦。
