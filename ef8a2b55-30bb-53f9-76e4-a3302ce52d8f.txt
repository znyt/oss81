        将这小女人紧紧地搂在怀里，傅夜沉覆到她耳旁，温柔地呢喃：“筱颖，我爱你。”
<br/>
<br/>        “我还想听。”童筱颖仰了仰头，微笑着与傅夜沉四目相对。
<br/>
<br/>        “我爱你。”
<br/>
<br/>        “不够。”
<br/>
<br/>        “我爱你、我爱你、我爱你、我爱你、我爱你……筱颖，我爱你。”
<br/>
<br/>        他一遍又一遍地告白，充斥着磁性的声音，好听得像是在清唱一首能催眠的摇篮曲。
<br/>
<br/>        这天晚上，傅夜沉终于看到他深爱的这个女人，是带着微笑，在他怀里睡着的。
<br/>
<br/>        傅昼景和季思妍的婚礼这天，是在滨江城里最豪华的酒店里举办。
<br/>
<br/>        整个大厅，红玫瑰、粉百合、红地毯，水晶吊灯星光闪闪，各桌酒水全部成双成对，现场布置得像爱丽丝的仙境般梦幻，厅堂里回荡着单曲循环的婚礼进行曲。
<br/>
<br/>        傅夜沉是一个人去的，毕竟，父母不在，长兄为父，爷爷傅炎廷上年纪了，晕车没能来参加他傅昼景的婚礼，傅昼景也丝毫不在意。
<br/>
<br/>        礼成之后，傅昼景带着季思妍前往各桌敬酒，来到傅夜沉这桌时，他还特意关问起了童筱颖。
<br/>
<br/>        傅夜沉只是莞尔一笑，淡淡地回答：“她没必要来参加你的婚礼。”
<br/>
<br/>        “也对。没必要。确实没必要。”傅昼景与傅夜沉碰了碰杯子，将杯中的白酒一饮而尽，讥笑了一番，“我看，是大哥和筱颍过得并不好吧！”
<br/>
<br/>        对于傅昼景的话，傅夜沉只笑不语。
<br/>
<br/>        季思妍因为有孕在身，所以喝的是果汁，对于傅昼景关心童筱颖，又给傅夜沉找茬这事，她似乎已经麻木了。
<br/>
<br/>        婚礼上，傅夜沉是第一次见到小尚轩。
<br/>
<br/>        大人之间的矛盾，并未因此而牵涉到这个小家伙身上。
<br/>
<br/>        在傅夜沉的眼里，小尚轩的眼睛十分水灵，与小时候的阿景几乎像是一个模子刻出来的。
<br/>
<br/>        但是，小尚轩的皮肤有点儿偏病态白。
<br/>
<br/>        傅夜沉蹲下身去，拉了拉小尚轩的手。
<br/>
<br/>        季思妍让小尚轩叫“大伯”。
<br/>
<br/>        小尚轩便乖巧地唤了声：“大伯。”
<br/>
<br/>        “乖。”傅夜沉轻轻地拍了拍小尚轩的肩膀，接着从衣兜里掏出了一个红包，递给了小尚轩，“这是大伯给尚轩的见面礼。”
<br/>
<br/>        小尚轩下意识地抬起头来看了季思妍一眼，然后才抬起瘦小的双手，接过红包，稚嫩地说了声：“谢谢大伯。”
<br/>
<br/>        傅夜沉会心一笑，并未在这里久留，而是提前离席了。
<br/>
<br/>        婚礼过后，傅昼景以“总裁”的身份接管了季氏集团，彻底成为了大哥傅夜沉的商业竞争对手。
<br/>
<br/>        因为最近要赶体育馆的设计图，得和部门里的团队一起加班，童筱颖从私宅里搬了出来，住进了傅氏集团大厦马路对面隶属于傅氏集团旗下的员工公寓里。
<br/>
<br/>        而她和傅昼景之前养的那条博美犬小爱，一直被傅夜沉寄养在了落以柔的家中，童筱颖也不打算去接回来了。
<br/>
<br/>        因为她没时间陪伴小爱，大哥就更不要说了，跟着落以柔，比跟着她要好。
<br/>
<br/>        只是令她没料到的是，她才搬来这员工单身公寓才住两天，这天，季思妍忽然找上了门来。
<br/>
<br/>        童筱颖拉开门后，一时间还未认出来面前这女人是谁。
<br/>
<br/>        只见她穿着米白色孕妇装，小腹微隆，素颜，双颊上有雀斑，看起来很憔悴。
<br/>
<br/>        “童筱颖，我们谈谈。”
<br/>
<br/>        季思妍开口说这句话的时候，童筱颖才认出来这女人居然是季思妍。
<br/>
<br/>        只是，她现在这副模样，是要跟她谈什么？
<br/>
<br/>        “有话直说。”虽然谈不上去憎恨季思妍这个女人，但是也没法对她坦然相见。
<br/>
<br/>        童筱颖对季思妍的态度，不冷不热，但又见她穿着平底鞋，双手捧腹，肩挎手提包，她还是好心邀请她进了屋。
<br/>
<br/>        在季思妍心里，现在真是风水轮流转。
<br/>
<br/>        童筱颖刚邀请她在沙发上坐下，她便从手提包里掏出一个信封，摆在了茶几上，示意童筱颖自己看看。
<br/>
<br/>        和曾经的那次，她来找她坦白小尚轩的情形是那么的几分相似。
<br/>
<br/>        季思妍总是一副“正宫娘娘”的姿态，摆出高贵优雅的气势，来给一个正在受宠的嫔妃下马威。
<br/>
<br/>        童筱颖淡定地拿起那个信封，将里面的东西抽了出来。
<br/>
<br/>        竟然是一叠照片！
<br/>
<br/>        而且还是一张又一张高清晰的床照。
<br/>
<br/>        照片里的女人，长得跟她一模一样，而这里面的男主角，自然是傅昼景了。
<br/>
<br/>        “这不是我。”童筱颖并未将照片看完，而是直接塞回了信封里，放到茶几上一脸淡定道。
<br/>
<br/>        “你就不要睁眼说瞎话了！我这怀着孕，也不打算跟你闹。我来只是想跟你开个价，你要多少钱，才能离开滨江城？彻底离开阿景。”季思妍一边抚摸着腹部，一边理直气壮地质问。
