　　涂小安发现自己亲身参与之后，信徒的崇拜，那些信仰之力就会归功在他的身上，比如那个晕迷者，看似是被蛇王水救活，实则不然。
<br/>　　他悄无声息的在那碗蛇王水中加了一点宝贝。
<br/>　　那么就是杀人蜂的蜂蜜。
<br/>　　杀人蜂的蜂蜜对人的身体有非常好的强身健体功效，但其中还有一大特点，就是能解毒。
<br/>　　能解毒，解什么样的毒，其实涂小安并不清楚，今日一试，效果显著，没想到这蜂蜜连罕见蛇种的毒都能解，这就不得了了。
<br/>　　杀人蜂的蜂蜜，彻底的在涂小安的心中提升了自我的价值。
<br/>　　其实每年白镇都有很多人死在蛇毒上面，他的杀人蜂蜂蜜要是能解蛇毒，用处是非常的大了。
<br/>　　别的都不说，拿来赚钱，都是一项非常好的项目。
<br/>　　无成本，纯利润，而且绝对非常的抢手。
<br/>　　当然，就算要做，也都是后话，涂小安现在首先是将信仰之力收集十万点数，开启商城，看看里面有没有什么能让自己化形的宝贝。
<br/>　　搁在以前他没指望自己还能变成人，但拥有了飞行术，变形术，那么所谓的化形术，也就不在是痴人说梦了。
<br/>　　此时他放眼望去庙内都是跪地的人头，很是欢喜。
<br/>　　现在有一点比较麻烦的是，他如何取而代之，成为真正的蛇王，然后在这个庙内建筑属于自己的蛇王雕像，他也不可能每天都待在蛇王庙中。
<br/>　　唯有把蛇王雕像换成自己的样子，才能收集信徒的信仰之力。
<br/>　　到时候，任何来庙内磕头许愿的信徒，都能为他贡献信仰点数，源源不断。
<br/>　　不过，想做成这件大事，他需要找一个帮手，来帮助自己完成这项壮举。
<br/>　　毕竟这个蛇王雕像在蛇王庙供奉了数百年，说换就换，谁也不会答应。
<br/>　　旋即，涂小安在这黑压压的人头中，锁定了一个目标，是那个道袍男子，也是蛇王庙的负责人。
<br/>　　刚才产生的信仰之力点数中，就有这个人，足足提供了100点信仰之力，说明这个人非常信仰蛇王。
<br/>　　他叫李顺，白镇人基本都认识他。
<br/>　　比涂小安年长一轮，三十出头，至今单身，也是一位蛇王的忠实信徒。
<br/>　　说起李顺，那是有一段故事的。
<br/>　　早年的时候，他还有一个哥哥，叫李文，是一位土生土长的捕蛇人，有一日，李文上佘山捕蛇，忽然在一棵大树根部发现了一个蛇洞，这个蛇洞本来被茂密的草遮住，平常没人发现。
<br/>　　他眼贼，拨开茂密的草，发现里面居然有一窝的蛇蛋。
<br/>　　李文顿时喜出望外，忙把上衣脱下来铺在地上，轻轻的把蛇蛋一个一个拣在衣服里。
<br/>　　足足十六颗蛋。
<br/>　　做完之后，他隐晦的听到一阵沙沙的声响由远及近直奔他而来。
<br/>　　李文立即警觉起来，放下蛇蛋，拿着蛇钩提防着。
<br/>　　一霎时，顿见一条一米多长的毒蛇窜到蛇洞口，下身盘起护着蛇蛋，上身威武地直竖起来，蛇信子像火焰般吞吐着，对着李文摆出进攻的架势。
<br/>　　李文退后两步，举起蛇钩，两眼住视着蛇的动静，相持了几分钟后，他决定主动进攻，挥舞起蛇钩，呼的一声下去，正中蛇头，蛇的上身立刻萎靡了下去，接着李文又是一个蛇钩砸在蛇身上，没想到力气过猛，把毒蛇给打死了。
<br/>　　身为捕蛇人上山是为了抓蛇，不是杀蛇，不过死了就死了，这里还有一窝的蛇蛋，他也收获颇丰。
<br/>　　之后，李文将一窝蛇蛋带回家，没多久十六条毒蛇孵化出来。
<br/>　　李文为了赚更多的钱，决定先将这些毒蛇给养大，只有成年的毒蛇，才有价值。
<br/>　　这一养，出事了，三个月后，李顺一日回家，发现自己的哥哥身上爬满了毒蛇，全身溃烂，早就气绝身亡了。
<br/>　　而这些毒蛇，正是哥哥从佘山带回来蛇蛋中孵化出来的毒蛇。
<br/>　　小蛇养大，蛇复仇。
<br/>　　这个故事破有点武侠小说中主人公的遭遇。
<br/>　　母亲被杀了，自己被仇人养大，然后报仇雪恨。
<br/>　　只不过让人毛骨悚然的是，这次不是人，是蛇的复仇。
<br/>　　李顺吓坏了，而且他感觉到这十六条毒蛇身上充满了怨气，甚至还想杀他灭口。
<br/>　　他转身就跑，一直跑一直跑，总觉得后面有蛇在追自己，也不知道是不是阴差阳错，他跑到了蛇王庙附近。
<br/>　　顿时灵光一闪，躲进了蛇王庙中。
<br/>　　进了蛇王庙，那种蛇的怨气，蛇的阴影才慢慢的消失掉。
<br/>　　李文明白是蛇王救了他，从此成为了蛇王坚定的信徒，由于哥哥也死了，他也没了家人，干脆就留在蛇王庙中。
<br/>　　这一待，就是十几年。
<br/>　　现在他赫然成为了蛇王庙的负责人，要是和尚的话，就是主持级别了。
<br/>　　蛇有灵性，会报恩，会复仇，很多惹了蛇的人，渐渐的，会选择来蛇王庙躲灾，看似迷信，但你也不得不信。
<br/>　　当然，这个故事有点灵异的赶脚，用科学的方式来解释，蛇为什么会复仇，是蛇的生殖腺会排除一种特殊液体来吸引异性，你要是想弄死它，只要你的身上染了这种味道，那么不管这条蛇有没有被你弄死，你都会有潜在的危险。
<br/>　　都会被它或者它的同类追踪。
<br/>　　一些老道的捕蛇人就是利用这一点，在自己的胶鞋上擦上这个分泌液，来引蛇的。
<br/>　　此时的李顺跟着众人毕恭毕敬的磕头，但他没有许愿，因为他并没有什么愿望。
<br/>　　唯一想的就是蛇王庙的香火能越发的昌盛，蛇王被更多的人信仰。
<br/>　　他坚定蛇王是存在的，是神灵。
<br/>　　忽然，李顺浑身一僵硬，感觉有一股无形的压力笼罩在自己的身上，让他动弹不得。
<br/>　　这时，他的眼睛慢慢睁大，脑海中出现了一道诡异的声音：“李顺，你可是本蛇王最忠诚的信徒”
<br/>　　李顺既惊既喜，蛇王这是在跟他对话，在他面前显灵，莫非是自己十几年如一日的在蛇王庙中伺候，得到了蛇王的眷顾？
<br/>　　惊喜，宛如中了五百万，让李顺整个人都亢奋起来了，他不害怕，只有惊喜。
<br/>　　“蛇王大人万岁，小人就是你最忠诚的信徒，可以为蛇王大人赴汤蹈火，万死不辞”
<br/>　　李顺激动的呢喃自语。
