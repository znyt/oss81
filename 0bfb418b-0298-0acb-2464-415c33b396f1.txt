　　“你们怎么来了。”林峰看到外面陆勇和周华带着几个大头兵走了过来，还真是稀客。
<br/>　　陆勇瞧了眼店内，“怎么，林峰小兄弟这是不欢迎我们？这不，听手下一个人说，林峰小兄弟开了个店，特意前来补个花篮。”
<br/>　　林峰恍然大悟，估计是之前那个牛队长把这里的事情告诉了上面。
<br/>　　“哪里，只是有些意外，事情都处理好了？”
<br/>　　“差不多了，不过这群人太狡猾了，大老虎都跑了，就剩下几个小苍蝇，如今也都已经绳之以法了。”陆勇道。
<br/>　　“不查不知道，一查吓一跳，我军部居然这么多敌方卧底，还真是可怕。”
<br/>　　跑了是预料之中的事情。林峰问道：
<br/>　　“可问题是他们跑哪里去了？”
<br/>　　陆勇摇了摇头，“这个我也不清楚。”
<br/>　　离开曙光基地外面都是怪物丧尸，能跑哪去。
<br/>　　“难道是那个地方？”
<br/>　　林峰忽然想到了巨树之森，不出意外，战门的总部就在那里，这群人既然是战门的人，那么很可能也是去哪里了。
<br/>　　“巨树之森？”
<br/>　　陆勇和周华对视一眼。
<br/>　　“这是基地附近唯一的一个S禁地，若真的如林峰所说，是战门的大本营，那么真的不好对付。”
<br/>　　“这件事情还是要从长计议，对了，林峰小兄弟，我们今天来不是谈论这些的，这是给你的，也算是一点心意。”
<br/>　　陆勇忽然拿出了一枚勋章。
<br/>　　“这是？”
<br/>　　林峰接过好奇道。
<br/>　　“金龙勋章！”
<br/>　　陆勇笑呵呵道。
<br/>　　“这个东西，一般人拿不到，必须要有杰出贡献的人才可以拿到，在荣誉勋章之中，也是仅次于战神勋章的高级别勋章了，这种勋章可不只是一种象征性的东西，他也是有很大实权的，像林峰小兄弟这么年轻拿到这枚勋章的，几乎没有。”周华道。
<br/>　　“这么说我也是个将军了？”林峰道。
<br/>　　“林峰小兄弟，这枚勋章可不只是荣誉，他是有实权的，你是可以随意调动基地内一个团的兵力而不需要跟任何人打报告。”陆勇笑道。
<br/>　　我去？这么爽？
<br/>　　一个团少说也1000多人了。
<br/>　　“啧啧，说实话，我都羡慕的很啊。”周华笑道。
<br/>　　“既然如此，那我就收下了。”林峰收入物品栏。没想到他居然一下子得到了两个荣誉勋章，一个金刀勋章相当于尉官，这个则直接是相当于将军的金龙勋章了，若是林峰父母还在的话，那估计要笑得合不拢嘴了。
<br/>　　“嗯。”
<br/>　　陆勇随后和林峰又闲聊了几句然后一群人就离去了。他们军务繁忙尤其是在这最近出了这么大事情的档期。
<br/>　　店外面的人叽叽喳喳的都一脸好奇。看到这么多人聚集在这里。
<br/>　　李德和他爹也都走过去好奇的看看到底发生什么了，然而就是这么一眼把他们给吓得魂魄都没了。
<br/>　　一群看热闹的都凑过去要看一看这花篮。
<br/>　　一看都吓了一跳。
<br/>　　“我去，中将陆勇送？真的假的？”
<br/>　　“周华是谁啊，这名字怎么这么熟悉？”
<br/>　　一群人都议论纷纷。
<br/>　　“我想起来了，这周华不就是那个排行榜上第一的强者吗？九阶的存在！！”
<br/>　　“什么？那个周华给这个店送花篮？这店什么来头。”
<br/>　　“我去，中将亲自来送花篮，这店逆天了，瞧瞧隔壁的那个小队长儿子的店，来送花篮的都是小队长级别的都已经要牛气的上天了。”
<br/>　　李德和他爹对视一眼都看出了彼此的不可置信。
<br/>　　接下来的几个时辰内。
<br/>　　不断有得到消息的人来送花篮。有军部的有战士会的。甚至还有古家的。
<br/>　　纷纷都来恭喜林峰开店大吉的。
<br/>　　“这军部的人还真是大嘴巴啊。”
<br/>　　林峰站在店门口看着一大堆的花篮是无奈道。
<br/>　　古月祝烤肉小哥哥开店大吉！
<br/>　　战士会干事刘峰祝林峰兄弟开店大吉！
<br/>　　战士会副会长罗敏祝林峰干事开店大吉！
<br/>　　战士会会长寒鸦祝林峰干事开店大吉！
<br/>　　安保部队中队队长XXX祝林峰开店大吉！
<br/>　　安保部队大队副队长XXX祝林峰生意兴隆！
<br/>　　安保部队大队队长XXX祝林峰小兄弟财源滚滚！
<br/>　　战斗部队大队长XXX祝林峰……
<br/>　　侦查大队副队长朱民祝林峰……
<br/>　　基地司令部总参谋长XXX祝林峰……
<br/>　　基地司令部中将XXX祝林峰……
<br/>　　基地司令部总司令XXX祝林峰……
<br/>　　各式各样的大人物，在这里来给林峰送花篮的，没有一个人的是低于中队长的，小队长级别的人都不好意思来送花篮。
<br/>　　一开始也有小队长来了，但是一看之前送的花篮，这一眼扫过去，很自觉的就把自己的花篮给拉走了，当然花篮拉走了，祝贺还是要嘴巴上送到的，他们拉走花篮只是因为摆在店门口的花篮都是大人物的，他们个小队长怎么可以和大人物的放在一起，这算什么事。
<br/>　　放在最前面的，自然是最吸引人的基地总司令的花篮。
<br/>　　这是个上将，也是基地内唯一的一个上将，比陆勇还要高一级的存在。
<br/>　　当然了，他没有亲至，而是派出了自己的贴身秘书来送祝贺的。
<br/>　　总而言之，新开的店，一日爆红。
<br/>　　甚至有人称之为，平民区最牛逼的店！没有之一！！
<br/>　　李德和他爹咽了口唾沫回到自己店内，当即决定，关店大吉！然后第二天去亲自上门赔罪！
<br/>　　看着门口一大堆的花篮。
<br/>　　林峰叹了口气。
<br/>　　“我本无心装逼，奈何无形装逼最为致命。”
