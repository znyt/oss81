　　战龙之爪一处，横扫天地。
<br/>　　这一根长矛，被瞬间粉碎。
<br/>　　但是看着长矛粉碎，毒手魔君的脸上，却是没有任何的惊讶，反而是露出几分冷笑。
<br/>　　“敢硬抗我的毒手魔功，你还真是活腻了！”
<br/>　　毒手魔君冷喝一声道。
<br/>　　“今天你的龙爪，就留下吧！腐蚀！”
<br/>　　长矛破碎，灵力却是融化，化为了一滩滩的剧毒，直接向着楚风眠的战龙之爪侵蚀过去。
<br/>　　这剧毒，就连周围的空间都腐蚀的滋滋作响，全部都贴在了楚风眠的战龙之爪上。
<br/>　　楚风眠一下子便是都感觉到了，这战龙之爪，在受到这剧毒的腐蚀。
<br/>　　他清楚，这便是毒手魔君的最强手段，他的剧毒，这剧毒，足以腐蚀一切，天级灵器都足以腐蚀。
<br/>　　“这是毒手魔功！毒手魔君的最强手段！”
<br/>　　“毒手魔君一出，几乎无法抵挡，只能退缩，稍微沾染上，便是极大的灾难。”
<br/>　　“这楚风眠真是愚蠢，居然打算以身躯硬抗，这下可糟了。”
<br/>　　“隐藏的强者，应该要出手了，不然真的要让楚风眠死在这毒手魔君的手中？”
<br/>　　“楚风眠死了也好，北域少一个天才，我们才能有更多崛起的机会！”
<br/>　　一尊尊武者，心思各异，其中一些跟楚风眠有些仇怨的武者，更是都盼着现在毒手魔君出手，直接将楚风眠轰杀才好。
<br/>　　战龙之爪，在这剧毒的腐蚀之下，嘶嘶作响。
<br/>　　“毒手魔功？没什么了不起的，旁门左道罢了，在九魔城都称不上是什么高级的功法，还想要来对付我？可笑至极！”
<br/>　　楚风眠冷哼一声。
<br/>　　他的右手一动，就在那战龙之爪上，无数灵力却是凝聚成了一道道鳞片的样子。
<br/>　　其中的每一枚鳞片，都是如同阵眼之一，组成了一道庞大无比的大阵，将远古龙王身的力量催动到极致。
<br/>　　顿时这剧毒，腐蚀在战龙之爪上面，居然是无法腐蚀其中分毫，反而是让战龙之爪现在变得更加闪亮一般。
<br/>　　这剧毒，虽然强大，但是想要腐蚀战龙之爪，怎么可能。
<br/>　　楚风眠凝聚了远古龙王身，更是经历过天晶这等神物淬炼身躯，他的身躯坚韧到了极致的地步。
<br/>　　这剧毒腐蚀上来，非但是无法腐蚀楚风眠的战龙之爪，反而是如同一种锻炼，令战龙之爪变得更加坚韧。
<br/>　　“破！”
<br/>　　楚风眠口中冷喝一声，战龙之爪再次挥动。
<br/>　　上面染上的剧毒，被一下子摔了下来，众人目光看去，这战龙之爪上，居然是毫发无损。
<br/>　　这剧毒，都对楚风眠的战龙之爪，无可奈何。
<br/>　　“毒手魔功，不过如此。”
<br/>　　楚风眠口中，淡然吐出八个字。
<br/>　　这毒手魔君，可是北域成名已久的天才强者了，这种实力，的确让人失望。
<br/>　　“若是你就这点实力的话，那今日你们地魔门的所有弟子，都要死在这里了！”
<br/>　　楚风眠的声音，虽然平淡，但是无数众人都足以听得出来，其中的认真。
<br/>　　楚风眠开口，从来都是说到做到，他说要杀了葬天，便是不顾任何威胁，也要将葬天斩杀。
<br/>　　将在场所有地魔门弟子，全部斩杀。
<br/>　　这句话，虽然狂妄。
<br/>　　但是从楚风眠的口中说出来，却是没有任何一人会去质疑。
<br/>　　“好一个狂妄的小辈，毒手魔功，不过如此？我不过才动用了一成的力量罢了，本来就是要给你一个机会，既然是你自己赶着来送死，那我就成全了你！”
<br/>　　毒手魔君听到楚风眠的话，勃然大怒道。
<br/>　　被一个小辈，出言讥讽。
<br/>　　他的脸色，已经是极为恼怒，就在他的背后，无数紫色毒瘴气，纷纷出现，将毒手魔君围绕在其中。
<br/>　　令他整个人看起来都如同一尊毒人一般。
<br/>　　他的一双毒手，更是开始膨胀起来，其中的颜色，也从那种绿色，向着更深的颜色开始转变。
<br/>　　这剧毒一旦出手，在场的武者，被波及到，不死也都要重伤。
<br/>　　“毒手魔君，请你住手！”
<br/>　　就在这毒手魔君，即将出手的一刻，一道古老的声音，猛然响起。
<br/>　　这古老的声音，从天而降，却是一尊一直以来隐藏的老古董开口了。
<br/>　　这声音的主人，隐藏在虚空之中，但是从这声音中，楚风眠都足以听出来一种超越生死的味道，显然也是一尊已经踏上生死台阶，碎命之境的老古董。
<br/>　　“武王城中，不许争斗，两位的恩怨，还请在外解决。”
<br/>　　这声音平静的开口道，但是话语中的意思，也是极为威严。
<br/>　　这古老声音的主人，也是北域的大人物，老古董一般的存在。
<br/>　　听到这声音，就算是桀骜不驯的毒手魔君，都只能是点了点头，收回了自己的灵力。
<br/>　　毒手魔君的实力再强，也依然无法跟真正碎命之境的强者抗衡。
<br/>　　“楚风眠，在武王城中，这次我饶你一命，不过也只能让你多活几天罢了，除非你一辈子在这武王城中，不然的话，只要你敢出城，我必杀你！”
<br/>　　毒手魔君看着楚风眠，怒喝一声，身形一动，回到了看台之上。
<br/>　　“杀我，就看你有没有这个本事了。”
<br/>　　楚风眠依然不惧，冷冷的开口，回应一声，他也平息了自己的灵力。
<br/>　　武王城，并非是生死之战的地方，其中无数强者瞩目，楚风眠也不敢真正动用全力。
<br/>　　这毒手魔君的实力，的确是楚风眠在碎命之境下，见到的最强者。
<br/>　　碎命之境的强者，以现在楚风眠的实力想要击败，还没有任何可能。
<br/>　　但是这毒手魔君，却并不一定，只要是楚风眠动用人皇罐，也未必不能将这毒手魔君轰杀。
<br/>　　不过在武王城中，人皇罐是万万不能暴露了，极道仙兵的力量，太过特殊，想要隐藏都不可能。
<br/>　　至少要离开武王城，楚风眠才可以动用。
<br/>　　这短暂交手，只是一个插曲。
<br/>　　几乎是在楚风眠跟毒手魔君交手的瞬间，擂台上便是再度分出了胜负。
<br/>　　夜天君毫无意外的击败对手，晋级决赛。
<br/>　　下一场，便是这一届武斗大会的决赛，真正的北域年轻一代的第一人，也要就此分出。
