　　别说冥家众人懵逼，就连叶尘也有些意外。
<br/>　　不过仔细一想，三大家族的这种行为，也就并不觉得奇怪了。
<br/>　　之前，叶尘一刀灭杀十多名化神强者的那一幕，深深的震撼到了这些人。
<br/>　　而叶尘和冥沧荨的关系，只要是正常人一看，就知道很不简单！
<br/>　　倘若冥家请叶尘出手，凭他那强悍逆天的实力，灭掉他们三大家族，还不是小菜一碟？
<br/>　　三大家族的家主一合计，与其坐等冥家灭了他们三大家族，他们倒不如主动认怂，在冥家没有对他们动手之前，讨好叶尘这个煞星。
<br/>　　毕竟伸手不打笑脸人，这也是保全他们家族的最好办法。
<br/>　　冥家众人都还处于懵逼状态，完全没有反应过来怎么回事，三大家主又纷纷开口了，
<br/>　　“叶前辈，晚辈之前目睹了前辈的风采，心中佩服之至，此次前来，特地奉上一份薄礼，还请叶前辈笑纳！”
<br/>　　雷家家主一边谄笑着说完这话，从乾坤镯中取出一个金色锦盒，缓缓打开，双手捧到叶尘的跟前，
<br/>　　只见，那锦盒之中，赫然是一株赤红色的果子，形状模样犹如一个刚刚出生的婴儿，通体上下透着难以形容的药香，单单是闻上一口，都觉得神清气爽，浑身说不出的舒服。
<br/>　　而周围众人看到此物，顿时纷纷大惊失色，尤其是冥家的众人，几乎都要跳起来了，
<br/>　　“这是……血婴霸王参！”
<br/>　　“据说此物乃是雷家的至宝啊！曾经有人出到三千万块极品灵石，雷家都没舍得卖！”
<br/>　　“雷家主这是疯了吗？居然把如此珍贵的至宝，给了这小……”
<br/>　　……
<br/>　　就在冥家众人纷纷惊呼之时，却见田、宋两大家族的家主，也纷纷取出了各自的礼物，双手奉到叶尘跟前，虽然比雷家家主手中的血婴霸王参略有不如，但也是一等一的极品宝贝！
<br/>　　到了此时此刻，就算是傻子也能看出来了，眼前这个叫叶尘的年轻人，绝对很不简单！
<br/>　　叶尘两眼微微一眯，在三人手中的宝贝上扫了一眼，嘴角不由得微微一扬，他自然知道，眼前这三个人，为何会如此讨好他。
<br/>　　“既然三位家主如此客气，那叶某就却之不恭了！”
<br/>　　叶尘倒也没有拒绝，直接伸手将三人的宝贝一一收了。
<br/>　　三大家主顿时纷纷松了一口气。
<br/>　　正所谓，吃人嘴软，拿人手短，既然叶尘拿了他们的宝贝，应该也就不好意思再对他们出手了。
<br/>　　而另外一边，冥家家主冥战目睹了这一幕，早已经背后冷汗直冒，不由得冲着一旁的冥沧山低吼道：
<br/>　　“混账东西！你到底招惹到了什么样的存在啊？为什么连三大家主都对此人如此畏惧？”
<br/>　　冥沧山此刻真的要哭了，
<br/>　　“父亲大人，这、这……我也不知道啊！难道三大家主被他收买了，在配合他演戏？对！一定是这样！父亲大人，您可千万不能就这么认输啊！您……”
<br/>　　“啪！”
<br/>　　冥沧山的话还没有说完，冥战直接抬手就是一巴掌，险些将他扇飞了出去，
<br/>　　“混账东西！”
<br/>　　在冥战看来，即便真如冥沧山所说，三大家主是被叶尘收买了，可是能够收买三大家主的，又岂是一般人？
<br/>　　更何况，冥战身为四大家主之一，对其他三人简直太了解了！
<br/>　　从三人的眼神举止之中，冥战看到了畏惧，而且是深深的畏惧！
<br/>　　而且，三人拿出的礼物，也全是各自家主的顶级至宝，一点也不像是在做戏。
<br/>　　种种迹象表明，这个看似修为不高的年轻人，要么是有着足以碾压三大家族的实力，要么是他背后的势力，拥有碾压三大家族的实力。
<br/>　　而无论是哪种可能，对于小小的冥家来说，都是绝对不可以招惹的存在！
<br/>　　这些念头，在冥战的脑海之中飞速闪过，当下不敢再耽搁，当即快步走到叶尘跟前，深深一躬身，道：
<br/>　　“晚辈冥战，方才有眼不识泰山，还望看在小女的面子，海涵恕罪！！”
<br/>　　这冥战也不愧是老狐狸，不但反应极快，而且拿得起放得下，最重要的是，他一眼就看出，叶尘和冥沧荨的关系不一般，直接将冥沧荨搬了出来。
<br/>　　叶尘冷冷一笑，
<br/>　　“冥家主见风使舵的本领，还真是让人叹服啊！不过你们家的那位大公子，刚刚好像说过，要让我不能活着离开这里呢！”
<br/>　　看在冥沧荨的面子上，叶尘自然不会把冥战怎么样，可是冥沧山这个祸患，他必须要除掉！
<br/>　　冥战听到这话，顿时脸色大变，而后方的冥沧山，更是脸色煞白，全身都忍不住颤抖起来。
<br/>　　而三大家主听到这话，不由得对视了一眼，纷纷露出一抹喜色，
<br/>　　他们原本还以为，冥家的这些人，是在迎接叶尘进城，却怎么也没有想到，原来叶尘和冥家之间，似乎刚刚发生过矛盾！
<br/>　　如此一来，即便叶尘看在冥沧荨的面子上，不会对冥家怎么样，肯定也不会为冥家所用了。
<br/>　　‘冥战啊冥战，亏你聪明一世，没想到也有犯傻的时候啊！上天送给你这么一颗大树，你都不知道死死抱住，真是蠢到家了！’
<br/>　　“哼！竟然有人敢对叶前辈如此无礼，我雷家第一不答应！！”
<br/>　　雷家家主当即暴然开口道。
<br/>　　而宋、田两家的家主，也都是不嫌事大的主，也纷纷附和起来，
<br/>　　“不错！叶前辈是何等人物？岂能由一个小辈如此侮辱？”
<br/>　　“冥战，你还真是生了一个好儿子啊！！”
<br/>　　“噗通！”
<br/>　　原本就已经处于崩溃边缘的冥沧山，听到三大家主的这番话，顿时再也支撑不住，直接瘫软在了地上。
<br/>　　叶尘淡淡瞥了一眼冥沧山，便将目光移开，转而望向一旁人群中，同样脸色无比难看的那位金长老。
<br/>　　他要做的，可不仅仅是杀了冥沧山这么简单，还要还冥沧荨一个清白。
<br/>　　“噗通！”
<br/>　　金老被叶尘的目光扫过，顿时吓得险些魂飞魄散，瞬间就跪了，
<br/>　　“叶前辈！大小姐！我错了！我真的知道错了！求求您，饶了我这条老命吧！”
