<br/>　　“叶公子，我是少阳门长老宋天一，只要叶公子愿意加入我们少阳门，我可以保证给你提供最好的资源！最好的功法！”
<br/>　　“叶公子，我正玄宗位列十三方势力之一，也诚挚的邀请您的加入，只要您愿意，我正玄宗圣子之位，就是您的了！”

<br/>　　最最重要的是，叶尘无门无派，还是一阶散修！
<br/>　　几乎不用怀疑，任何一方势力，只要得到了叶尘，并且叶尘只要不陨落，在不久的将来，肯定可以一飞冲天！

<br/>　　对于各方势力的疯狂邀请，叶尘并没有理会，直接从人群中穿行而过，缓步来到擎苍宗的阵营，目光最终落在了，早已经完全呆滞的凌嚣身上，嘴角微微一扬，
<br/>　　“我们之间的赌约，你应该还记得吧？”

<br/>　　尤其是那些和擎苍宗不怎么对付的宗门，更是全都一副看好戏的神情。

<br/>　　甚至，即便是叶尘侥幸赢了，也不能真的把凌嚣怎么样。
<br/>　　毕竟，凌嚣可是擎苍宗的天才！
<br/>　　可是现在，今非昔比了！
<br/>　　以叶尘如今的声势，虽然依旧没有和擎苍宗叫板的资本，可即便是擎苍宗，也不敢轻易去动叶尘。

<br/>　　已经逆天到，连八大仙门，也不得不慎重对待的地步！
<br/>　　毕竟，没有任何一个宗门，愿意招惹一个具有无穷潜力的超级妖孽！

<br/>　　谁能保证，他不会成长为下一个昊天大帝？
<br/>　　凌嚣的脸色，顿时一阵青，一阵白，说不出的难看，

<br/>　　“叶公子，之前凌嚣确实有做的不对的地方，我代他向你道歉，还请叶公子，看在擎苍宗的面子上，不要和他一般见识！”
<br/>　　叶尘没有说话，仿佛压根没有听到。
<br/>　　孙长老顿时气得牙根直痒痒，
<br/>　　如果不是叶尘之前的表现，实在太过于逆天，以他的身份地位，又岂会如此卑躬屈膝的向一个小辈道歉？
<br/>　　可是形势比人强！
<br/>　　在这种局面之下，谁也不愿意去招惹一个，将来可能成为无上巨头的存在！
