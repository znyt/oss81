　　一座漆黑的小岛之上，一座阵法光芒闪烁，照亮了整座岛屿。
<br/>　　少倾，阵法之光消散，四道身影出现在了阵法之中。
<br/>　　苏莫抬眼打量四周，顿时心中微惊，只见他现在身处的小岛，方圆不过十里，坐落在一片黑色的海面上。
<br/>　　岛屿漆黑如墨，海水同样的漆黑，显得阴森无比。
<br/>　　抬头看向上方，只见天空昏暗无比，天穹之上均是黑色的云层，仿佛山雨欲来，让得整个世界都阴暗了下来。
<br/>　　魔气！
<br/>　　苏莫心中震惊，这处地域无论是天空、岛屿还是海水中，都充满了浓郁的魔气。
<br/>　　这些魔气之浓郁，是他平生仅见，比十亿魔剑中的那魔煞之气，要强大太多太多了！
<br/>　　而此刻，他们身处的这处小岛上，被一个巨大的透明气罩完全笼罩。
<br/>　　这里是神武学府为历练的弟子设立的据点，只有神武学府的弟子可以登上此岛，在此岛修养，或者离开。
<br/>　　而其它生灵，无论是魔甲族还是凶兽，别说登上此岛了，只要距离此岛千里之内，就会被阵法之力绞杀。
<br/>　　此时，岛上还有几名神武学府的弟子，正在距离传送阵不远的地方盘膝疗伤，个个身上浴血，显然经历过血战。
<br/>　　“我们走！”
<br/>　　四人到来之后，高大青年招呼一声，率先飞出了阵法，向岛外飞去。
<br/>　　苏莫三人便也立刻跟了上去。
<br/>　　很快，四人便飞出了岛屿，飞到了一望无际的海面上。
<br/>　　看着下方漆黑如墨的海水，苏莫心微微有些凝重，这个魔云海，估计比神武录上介绍的还要危险，他必须打起十二分警惕了。
<br/>　　几人一路飞行，前往海域深处，苏莫暂时也没有和三人分开，准备稍稍摸清楚魔云海的具体情况，在做打算。
<br/>　　飞行之中，苏莫与聂乾和韩盈随意的闲聊，也自报了姓名。
<br/>　　从聂乾口中，苏莫得知高大青年名为廖云剑，天赋不凡，实力强大。
<br/>　　轰！
<br/>　　几人刚刚飞出半个时辰，下方的海水猛然一震，一只庞然大物突然从海水中暴冲而起，血盆大口张开，一口闪电般的咬向苏莫。
<br/>　　这是一条黑色的巨蟒，蟒头足有一座宫殿般大小，体表的颜色和海水几乎一模一样。
<br/>　　或许是苏莫的修为气息，是四人中最弱的存在，巨蟒不攻击别人，单单选择了苏莫。
<br/>　　“找死！”苏莫见此，眸中杀机一闪，手掌一拍而下。
<br/>　　霎时之间，浩瀚的掌劲如同一座大山，镇杀而下，狠狠的击在了巨蟒的脑袋之上。
<br/>　　轰！
<br/>　　一声爆响，巨蟒硕大的头颅顿时被轰碎，化为漫天残渣，迸射八方。
<br/>　　庞大的巨蟒，立刻死亡，身躯跌落于海水之中，掀起滔天大浪。
<br/>　　“嗯？”聂乾、韩盈以及高大青年三人见此，顿时眼眸一凝。
<br/>　　要知道，刚才死亡的那条巨蟒，可是八级一重的凶兽，相当于武帝境一重武者。
<br/>　　而苏莫居然挥手之间，便将巨蟒镇杀，不由得让三人微微惊讶。
<br/>　　当然，三人也没有太过震惊，毕竟，能进入神武学府的人，哪一个不是绝世天才！哪一个没有越级战斗的实力！
<br/>　　苏莫以准帝境的修为，能随意杀死巨蟒，虽然让三人有些惊讶，但也没出三人的意料之外。
<br/>　　“呵呵，苏莫你果然实力不凡！”聂乾笑着说道。
<br/>　　“一只妖兽而已，不足挂齿！”苏莫道。
<br/>　　“苏莫，这可不是妖兽，而是凶兽！而且是魔化的凶兽！”
<br/>　　聂乾摇了摇头，道：“妖兽是妖族，可以随意化形，这种凶兽不能化形，没有多少灵智，凶残无比！”
<br/>　　“原来如此！”苏莫心中恍然，原来还有不能化形的凶兽。
<br/>　　“呵呵，就这么点实力，就自得意满了？”廖云剑瞥了苏莫一眼，不屑的冷笑一声，眸中尽是藐视。
<br/>　　苏莫闻言脸色一冷，这个家伙是杠上自己了吗？简直是不可理喻！
<br/>　　“苏莫你别介意！”
<br/>　　韩盈见此急忙开口，她也是怕等会闹僵，队伍解散，随即她继续道：“廖兄快言快语，你别放在心上！”
<br/>　　苏莫闻言心中暗暗摇了摇头，便不在言语，他本想警告对方一番，但想想还是算了。
<br/>　　若是此人找死的话，他不介意让对方成为他的养料。
<br/>　　廖云剑见苏莫不敢反驳，嘴角勾起了一丝不屑的笑容，他眼眸瞥了一眼韩盈丰腴的娇.躯，眸中闪过一丝莫名之色。
<br/>　　随即，四人都不在说话，继续飞行。
<br/>　　魔云海很大，按照神武录的记载，此地足有方圆亿里，比之整个苍穹大陆，都小不了多少。
<br/>　　“大家小心，前方有魔甲人！”飞行了两个时辰之后，韩盈突然说道。
<br/>　　三人闻言立刻放慢了速度，不过苏莫仔细观看了一番，甚至催动了幽冥魔瞳武魂，也没有看到魔甲人的存在。
<br/>　　“魔甲人在哪里？”苏莫疑惑的问道。
<br/>　　“在二十万里之外，共有十人，修为最高的是武帝境三重，其他的都是武帝境一重或两重！”韩盈面色凝重的说道。
<br/>　　苏莫闻言惊奇，此女有什么本事？居然感知如此的敏锐！
<br/>　　聂乾见苏莫面带惊疑，立刻解释道：“韩姑娘的战魂是‘洞听耳’，可以听清方圆二十万里之内的任何声音，连身上的气息波动都能听的一清二楚。”
<br/>　　原来如此！
<br/>　　苏莫立刻恍然，原来此女的武魂，也算是一种特殊武魂。
<br/>　　“十个魔甲人而已，不足为惧，我们直接杀过去！”廖云剑傲然说道，随即他身形加速，如同幻影一般，急速向前方飞去。
<br/>　　“走！我们也杀过去吧！”聂乾招呼一声，随即三人立刻跟向廖云剑。
<br/>　　聂乾心中有些无奈，他本想暗中出手对付这十名魔甲人，没想到廖云剑居然如此自信。
<br/>　　既然如此的话，那他们只能直接杀过去了。
<br/>　　嗖嗖嗖！！
<br/>　　四人的身形，如同四道闪电，划破长空，瞬息数千里。
<br/>　　片刻之后，苏莫终于看到了前方的十名魔甲人。
<br/>　　魔甲人的体型和普通人族无二，但身高比人族要高出两尺有余，全身更是漆黑如墨，厚重的鳞甲覆盖全身，背部还长满了锋利的骨刺。
<br/>　　“哈哈！一群蝼蚁，死吧！”廖云剑狂笑一声，面色兴奋，急速冲了上去。
