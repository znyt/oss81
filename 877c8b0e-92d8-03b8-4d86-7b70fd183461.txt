        对她大献殷勤也就算了，还把自己的色相给卖了，结果就换来了她这么一句没心没肺的话。
<br/>
<br/>        明司寒岔气地远离连星儿，极其没好脾气地开始自顾自地扣起衬衫，不仅被她气到脖子粗了，就连耳根子也跟着红了。
<br/>
<br/>        连星儿见明司寒脸色不好，于是凑了过去，双手背在身后，踮起脚尖，仰望着他，声音忽然软糯软糯地示好道：“喂，别生气嘛！”
<br/>
<br/>        “我去洗澡，懒得理你了。”明司寒瞥了连星儿一眼，转身进了卧室。
<br/>
<br/>        连星儿偷偷地笑了笑，然后心情大好地坐回餐桌前，继续喝粥。
<br/>
<br/>        洗澡的时候，明司寒有些纳闷地摸了摸自己的腹肌。
<br/>
<br/>        这可是他辛辛苦苦练了三个月才出来的，居然还被那丫头嫌弃！
<br/>
<br/>        明司寒洗完澡出来后，连星儿已经吃完早餐并换上了他给她买的新衣服。
<br/>
<br/>        他起初给她买了哪些衣服后，又特意去了一趟洗衣店，让服务员将她的新衣服洗干净了烘干，然后他去买了早餐再回来接的。
<br/>
<br/>        一大清早，他便一直在为她奔波，等到他看到她穿着自己给她买的衣服时，心里又特别的高兴。
<br/>
<br/>        有个时候，他心里还有一种冲动就是，他还想给这丫头买房子、买车、买珠宝首饰，恨不得把这世上最好的东西都买给她。
<br/>
<br/>        明司寒随后快速地解决掉了早餐，将连星儿送回了学校宿舍。
<br/>
<br/>        他看着连星儿平安地上了宿舍楼，自己才转身跑回自己的宿舍里。
<br/>
<br/>        连星儿和顾千凝她们在上课的期间，班里所有的男生突然被班主任给叫了出去。
<br/>
<br/>        到了课后，她们才打听到，原来今天全校的男生都被叫去操场上跑步了。
<br/>
<br/>        只是，所有人都不知道其中到底是什么原因。
<br/>
<br/>        班上的男生们跑完步回来后，女生们就忙着跟他们打听内幕消息，结果，就连他们男生自己也不知道是怎么一回事。
<br/>
<br/>        当所有女生都在好奇打听这件事情，将男生们群围的时候，只有连星儿格外淡定地坐在自己的座位上不为所动。
<br/>
<br/>        就在连星儿的周遭四下没人的时候，一只又粗又短的手，拿着一个粉色的信封，递到了连星儿的眼皮子底下。
<br/>
<br/>        连星儿坐在座位上，视线顺着这只粗糙的手，下意识地抬起头来，直到与一个留着板寸头，戴着黑框眼镜的男生四目相对后，淡淡地问了一句：“同学，你有事？”
<br/>
<br/>        同在一个班上，这相处下来也有两三个月了，她竟然只是管他叫“同学”。
<br/>
<br/>        不过，连星儿确实不认识面前的这个男生。
<br/>
<br/>        男生推了推鼻梁上的眼镜，自我介绍道：“我叫洪志业，希望你能记住我的名字，毕竟我们是同班同学。”
<br/>
<br/>        “噢！”连星儿应了声，未接洪志业递过来的信封，而是自顾自地继续低头看书。
<br/>
<br/>        “连星儿，这是我给你的信，希望你看一下。”洪志业憋红了脸，这次将信封直接放在了连星儿的书上。
<br/>
<br/>        连星儿无奈地咬了咬下唇，刚想拿起这封信，岂料一只修长好看的大手却先将这封信给夺走了。
<br/>
<br/>        接着“嘭”地一声，一个人影一屁股坐在了桌子上。
<br/>
<br/>        连星儿怔愣地抬起头来，只见明司寒坐在桌面上，一脚踏地，一脚悬在半空中，意味深长地看着手上那封粉红的信。
<br/>
<br/>        “你叫洪志业是吧？”明司寒的星眸微微眯起，将跟前站着的洪志业上上下下打量了一番，语气冰冷得质问。
<br/>
<br/>        原本聒噪的教室，瞬间鸦雀无声，几乎所有人的目光，齐刷刷地落向这边，气氛变得格外严峻。
<br/>
<br/>        洪志业原本只是脸红，此刻被明司寒这么一问，连带着脖子处也红了。
<br/>
<br/>        就在所有人等着看他的笑话时，连星儿却突然从笔袋里翻出一张十元的纸钞，塞到洪志业的手中：“谢谢你啊！洪同学，帮我跑了一趟腿，买来了我想要的粉色信封。我正急着用来装贺卡，送给我的爱豆。”
<br/>
<br/>        “哦，不、不用谢。”洪志业聋拉着脑袋应道，看着连星儿塞到自己手中的十块钱，原本想说的话，此刻却支支吾吾地一句话也说不上来了。
<br/>
<br/>        周遭原本等着看他洪志业笑话的同学，纷纷收回了目光，又恢复了刚刚的聒噪。
<br/>
<br/>        连星儿趁机从明司寒的手中夺回了那封粉色的信，然后若无其事地从抽屉里拿出一张贺卡大小的卡片，接着打开信封，将卡片塞了进去。
<br/>
<br/>        她的动作一气呵成，自然得不能再自然。
<br/>
<br/>        洪志业见状，赧然地打了声招呼：“那我先走了，你们慢慢聊。”
