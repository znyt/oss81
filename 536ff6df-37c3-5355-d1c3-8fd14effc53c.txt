　　第280章
<br/>　　火光明灭，窦竹仝看到吕春花被绑在柱子上，手脚都被紧紧捆着，嘴里还堵着麻布。
<br/>　　“姓程的，枉我还把你当个人物，你竟然冲女人下手，你还算不算男人！”窦竹仝咬牙切齿。
<br/>　　“快别废话了，你不给活路，也别怪咱们兄弟把事情做绝！”程大雷隔着河水大喝道：“你究竟退不退，不退的话，这女人你就见不着了！”
<br/>　　窦竹仝双目圆睁，眼底恨不能喷出火来。他现在还不知道，云中龙已经把他们都卖了，所以程大雷才能对症下药，知道应该绑谁。
<br/>　　“姓程的，你有什么冲我来，放过我的女人！”
<br/>　　“想要女人，就乖乖撤退，我们好吃好喝好照顾，你要是不退的话，哈哈，我们兄弟是什么人，你大概还不知道吧。”
<br/>　　程大雷举着火把，脸上露出淫·荡的微笑，其他几人也嘎嘎的笑着。
<br/>　　窦竹仝将牙咬得咯吱咯吱响，恨不得当场杀了程大雷。
<br/>　　“窦竹仝是不是没考虑清楚，还是让你的女人陪你聊聊吧。”
<br/>　　程大雷扯开了吕春花嘴上的麻布。
<br/>　　“窦竹仝，你进攻啊！我生是你的人，死是你的鬼……老窦，别让我瞧不起你！”
<br/>　　刚扯开嘴，吕春花就冲着窦竹仝大喊。
<br/>　　程大雷吓了一跳，倒是小看了这个女人，他挥挥手，忙让手下把吕春花的嘴堵上。
<br/>　　“姓窦的，你女人的话你都听见了，她不让你进攻，你还不撤兵等什么。”
<br/>　　窦竹仝咬着牙：“你先把人给我放了。”
<br/>　　“你先退兵，嫂夫人安然无恙的送回，放心，我这个人向来一诺千金，你去江湖上打听打听，看我蛤蟆大王的信用如何。”
<br/>　　“姓程的，谁不知道你寡廉鲜耻，言而无信，你在江湖上的信用已经败尽了。”
<br/>　　“谣言，绝对是谣言，谣言该止于智者，江湖传言也能信么。”
<br/>　　双方展开骂战，隔着河僵持不下。
<br/>　　正这时候，一支箭隔空而来，正射向吕春花，直直的射入吕春花的胸口。
<br/>　　程大雷一怔，哇呀呀大叫道：“姓窦的，你还真敢对自己女人下手呐！”
<br/>　　窦竹仝也是一愣，就见身后立着个一身戎装的妇人。他目瞪口呆，呐呐道：“你杀了她，你为什么要杀她？”
<br/>　　这妇人就是窦竹仝的老婆，莫鸣米的表妹，江湖上人送绰号花皮虎。
<br/>　　“死了就死了，你鬼叫什么，难道让你和这个狐狸精白头偕老嘛！”花皮虎一声冷喝：“所有人听我军令，进攻！”
<br/>　　喽啰都看着窦竹仝，不知该不该听花皮虎的。窦竹仝脸上阴晴变换，时而是彻骨的悲伤，时而是冲天的愤怒。
<br/>　　“一个大老爷们，你还要在这里哭一场么！”花皮虎冷哼一声：“那娘们是因这帮山贼而死，要报仇往前看。”
<br/>　　窦竹仝将牙一咬，大吼道：“给我杀，杀，杀！”
<br/>　　船分水浪，十艘船一同攻来。
<br/>　　程大雷撇撇嘴，这突然的变故也是出乎他预料，不过，程大雷也不是一年前的程大雷，一个女人的死还不足以扰乱他的思绪。
<br/>　　眼下只能硬碰硬。
<br/>　　“弟兄们准备，等他们落水时放箭，准备白刃战。”
<br/>　　“吼！”
<br/>　　一群人齐声怒吼，军威振天。
<br/>　　真是展开白刃战，众人也没有怯的道理，所有人都已磨刀霍霍，热血沸腾。
<br/>　　船接近滩涂时已经行不进，窦竹仝已经红了眼，带人从船上跳下来。众人都有好水性，这是蛤蟆寨的山贼不能比的。
<br/>　　扑面而来的箭雨，这些人顶着箭雨冲锋，见他们距离岸边越来越近，眼看就是近战格杀。
<br/>　　大家心里都燃烧着战意，都是趟着尸山血海出来的弟兄，早已将生死放在一旁。
<br/>　　而近战并不是程大雷想看到的，无论怎样的胜利，都不得不面对伤亡，他不喜欢看到己方的伤亡。
<br/>　　这时候，变故突生。河里攻来的水贼突然发出一声声尖叫，身体抽搐，尸体从水中浮了起来。
<br/>　　窦竹仝不知道这一幕为何发生，诧异的看着周围。只见有个手下高声惨叫，身体不停颤抖。
<br/>　　“蛇，是蛇！”
<br/>　　程大雷倒是忽略了一件事，他用雄黄将蛇从岛上赶走，这些蛇也不会凭空消失，就在岸边的浅滩筑巢。
<br/>　　本来大家相安无事，这些毒蛇也是凑合活着，可安居之处突然被人打扰，它们下意识就开始攻击。
<br/>　　这些蛇都是剧毒，见血封喉，窦竹仝还没到岸边就死伤大半。
<br/>　　花皮虎在船头看着这一幕，眉头紧紧皱起，高声喝道：“退，快退。”
<br/>　　窦竹仝是红了眼，硬生生泅水过来，躲过毒蛇的攻击，直扑到吕春花身上。
<br/>　　“花娘，花娘！”
<br/>　　吕春花气息微弱，见到窦竹仝时脸上忽然浮现一种笑意。
<br/>　　“死人，你那袍子我……我做不完了。”
<br/>　　说罢，她合上双目，就此一命呜呼。
<br/>　　窦竹仝仰起头来，泪如雨下，口中发出一声怒吼。
<br/>　　“姓程的，你还我花娘命来！”
<br/>　　花皮虎皱眉：“哼，他和这娘们感情还挺深，去，被他抢回来。”
<br/>　　窦竹仝提着刀站起来，要找程大雷拼命，程大雷握紧了剑，心道：兄弟，今天就是今天了，我送你们一对苦命鸳鸯在黄泉重逢。
<br/>　　“大哥，大哥……”
<br/>　　有手下拦住了窦竹仝，发觉他此刻身体硬得像石头，他卖力挣扎，过程中抱住吕春花的尸体不放手。大家强拖着他和吕春花的尸体上了一艘小筏子，几个人迅速撤上了大船。
<br/>　　窦竹仝退去后，岸边漂浮着几十具尸体，蛤蟆寨这边却是没有丝毫损失。
<br/>　　只是程大雷的心情，实在是好不起来，今日有毒蛇拦路，大家才侥幸获胜。可若没有这些毒蛇，今天的情况对己方就不妙了。
<br/>　　今天是躲过了，但下次呢，想要对付浅水区的毒蛇其实也有很多法子。窦竹仝虽然今日退兵，以他对自己的恨意，怕是很快就会卷土重来。
<br/>　　必须尽快将山寨升到二本，不然的话情况不妙。程大雷最近一直在加紧做这件事，可是山寨只有百十来人，纵然有建筑精通的技能加成，想快也快不了。
<br/>　　只能去周围的村镇抢人。
