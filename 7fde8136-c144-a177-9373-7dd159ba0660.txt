　　不过李承乾所料想的反对意见并没有多少，大部分的意思还是想知道为什么。
<br/>　　是啊，为什么呢，这武德多少年，贞观多少年不是挺好的么，为什么改成皇历呢？八百多年多难记啊。
<br/>　　面对一群臭老九的喋喋不休，李承乾就觉得头疼，于是便不耐烦的问道：“那么你们谁能说清楚汉武帝是什么时候死的，光武帝又是什么时候死的！”
<br/>　　“回陛下，汉武死于后元二年，光武死于建武中年。”某老头儿想都没想回答道。
<br/>　　这特么好尴尬，突如其来的回答让李承乾有种拿砖头砸自己脚的感觉，半晌硬是一个字没说出来，良久之后才咬牙说道：“朕说的是大家都知道的年份，你这后元二年谁知道是哪一年，建武中年又是哪一年。”
<br/>　　这同样又是一个尴尬的问题，多少有恼羞成怒的成份在里面，下面的老头儿也不是傻子，总不会在这个时候再跟他呛火，于是下面便没了声音，大殿上只有李承乾一个人的声音在咆哮。
<br/>　　“朕不想管什么后元，什么建武，同样当初你们记这个东西的时候有多麻烦你们自己难道都忘了？所以那就统一一下，统统叫皇历不好么，好学，好记，省得我大唐百姓忘了本，朕要让我大唐的百姓都知道，历史上曾经发生过什么，我泱泱华夏千年传承，总不能只有几个人只知道！”
<br/>　　“陛下圣明！”对于一个歇斯底里，而又报复心强的皇帝，朝堂众人理解的没有再与他争论。
<br/>　　皇历就皇历吧，不就是多记一份历法么，这也没什么了不起，和个人又没有什么切实的关系，用得到的人还是用以前的记法，用不到的就是再简单，你告诉他了估计也没用。
<br/>　　只单单公布了一件修改历法的事情，便已经让李承乾觉得有些焦头烂额，想想老头子不知颁布了多少条法令，这其中经过了多少次的斗智斗勇还真不好说。
<br/>　　想到这些，李承乾不由暗暗叹了口气，想了想便又说道：“既然这皇历的事情已经定下来了，那么礼部就抓紧时间把时间排一下，所有的年份都统一统一，今后再刊印历书，必须加上皇历年份注释。”
<br/>　　“喏！臣等遵旨。”
<br/>　　……
<br/>　　搞定了皇历的事情，李承乾小小的松了一口气，这还是他第一次以皇帝的身份来颁布发令，如果说没有一点小兴奋那才是怪事。
<br/>　　所以刚刚他已经做好了与下面那些老货们死磕的准备，结果没想到，只是吵了几句便过关了，看来这些老东西还是比较给自己这个新人面子的。
<br/>　　正琢磨着，下面忽又走出一人：“陛下，臣有本凑！”
<br/>　　“讲来！”李承乾看着这位有着李猫之称的中青年男人，眉头不自觉的皱了皱。
<br/>　　“陛下，既然已登帝位，这皇后和太子的人选是否也要定下？正所谓……”
<br/>　　“所谓个屁，滚！”一提皇后和太子，李承乾顿时炸毛了！
<br/>　　后宫那几位这几天就已经开始唇枪舌剑的拉帮结派了，难道这李义府还嫌弃这火烧的不够大？还特么立皇后、太子，这不是扯蛋么！
<br/>　　“陛下……”
<br/>　　“陛个屁下，滚出去，否则朕马上灭你全族，你信不信！”李承乾正在气头上，哪里还想听李义府说什么，拍着桌子吼道。
<br/>　　“陛下，臣觉得李侍郎说的没错，正所谓国不可一日无君，这后宫同样不可一日无主啊。”
<br/>　　“是啊陛下，您是一国之君，怎可辱骂朝臣呢！”
<br/>　　“陛下，难道您刚刚登基便要学隋炀帝那暴君么？”
<br/>　　“呼啦”一下，李承乾的话音刚落，下面已经站出来一大堆人，呜哩哇啦就开始对他一顿狂喷。
<br/>　　看着下面一群不知所谓的家伙在侃侃而谈，李承乾终于还是忍不了这口气，他和李世民不一样，老李要面子，可是他不要，面子那东西生不带来死不带去，没必要为了它受活罪。
<br/>　　而且他与老头子并不一样，老头子和长孙皇后那是从小一起长大，皇后之位根本没有什么选择的余地，所以老头子登基之后便确立了。
<br/>　　可是现在他可没有老头子那时候的条件，现在立了皇后就是给自己找麻烦，后宫那几位谁也不服谁，最后搞不好会出大乱子，所以眼下他无论如何也不可能同意册立皇后和皇太子。
<br/>　　想到这里，李承乾的脸色渐渐冷了下来，看着下面一群人目光越发阴郁，等到那些家伙全都闭嘴才缓缓说道：“想必你们都知道后宫不得干政这一条吧？”
<br/>　　下面没人说话，全都看着李承乾等着他把话说道。
<br/>　　“既然知道后宫不得干政，那么你们是否知道朝中不可干涉后宫！你们一个个口若悬河，为国为民，朕想知道，你们凭什么管朕的私事？别用天家无私事来敷衍朕，那是道德绑架！”
<br/>　　李承乾越说越生气，娘、、的，这帮王八蛋看来是真以为自己刚刚登基羽翼未丰，想要从中讨野火。
<br/>　　不过他们也不撒泡尿照照自己是什么德性，还真以为法不则众？
<br/>　　“陛下……”
<br/>　　“闭嘴，朕的话还没有说完！”眼看着下面一群人又要开口，李承乾猛的站起身来，与下面那些人狠狠对视着，一字一顿的说道：“从今日天始，但有再言及立后，立太子者，流放岭南，遇赦不赦！退朝！”
<br/>　　“退朝！”‘啪’的一声净鞭响起，执事的内侍尖细的嗓音响起。
<br/>　　惹不起咱还躲不起么？朝堂上想不出办法，难道咱还不会回去想办法么，还真以为老子会傻fufu跟你们针尖对麦芒？李承乾一边往自己的东宫走，一边想着。
<br/>　　不过，这件事情躲得了初一躲不过十五，早晚都是要解决的，一句流放显然还吓不住那些个老狐狸，所以眼下必须找一个能说得过去的理由来搪塞，否则……非被老头子笑掉大牙不可。
<br/>　　而且更重要的是，后宫的事情到底怎么解决，眼下已经要乱成一团了，再这样下去指不定哪天就特么会闹出人命。
