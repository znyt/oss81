　　“你知道这个程序？”
<br/>　　齐凡看向莱昂，把手机递给他。
<br/>　　“这是一个定位系统。”莱昂从齐凡手中接过手机，把程序打开，解释道：“这种系统的定位装置只有指甲盖这么大，但是它却能接收到卫星信号，即便是在极地，信号也不会受到干扰。我还在海豹的时候，这个东西是军方专用的，后来又发布了低配的版本投放到民用市场，深受一些军事资源公司、安保公司以及警察的欢迎，因为这种定位系统比起以前的定位系统更加精确，而且不会受到地理环境的影响。”
<br/>　　很快手机上就出现了城区的地图，地图上一个红色小点正沿着街道快速的移动着。红色小圆点在转过一个街角后，停在了一栋建筑前，停了差不多有几分钟的时间，然后又继续开始移动；红色小圆点穿过几个街区，又在一栋建筑前停了下来，并且又是停了几分钟的时间。
<br/>　　此后，红色小圆点按照之前的规律在道路上行驶着，行驶一段距离就在一处建筑前停下几分钟的时间。最后一次停下后，红色小圆点开始快速的移动，进入到一栋建筑中，彻底停了下来。
<br/>　　而此时，莱昂手中手机的电量也是快要耗尽，只剩下不到百分之二十的电量。
<br/>　　莱昂退出了定位程序，对布谷问道：“布谷鸟，把刚才那些红点停留的位置都记录下来了吗？”
<br/>　　布谷点了点头，把自己的PDA推到桌子中央，说道：“这上面就是刚才红色小圆点停留过的位置，以及它最后停下的地方。红色小圆点停留的位置全都是超级市场、以肉类食品为主的食品加工店，而最后这个红色小圆点停下的地方，是一家冷藏公司。”
<br/>　　“根据刚才红色小圆点的移动速度，还有她停留的位置，基本可以确认，这个红色小圆点应该是一辆运送肉类食品的冷藏运输车。这也符合P病毒低温储存的条件，在P病毒上安装定位装置，并且把P病毒藏在冷藏运输车中，让这种致命病毒始终处于移动中，布克尔特只需要用这部手机就能随时了解P病毒的位置，而别人没有跟P病毒上定位装置相连的定位程序，根本不可能知道P病毒在哪里，真是好算计啊。”
<br/>　　“但是这样做的也会出现问题。”
<br/>　　陈应天提出不同的意见，说道：“这样做的确可以把P病毒很好的隐藏起来，可同样的，这样做的不可控因素也很多。冷藏运输车不是二十四小时不断工作的，一旦冷藏运输车不制冷，很有可能会因为温度导致P病毒失活；而且如果冷藏运输车发生交通事故，也有可能会使P病毒破损乃至泄露，这对于布克尔特来说是得不偿失的。”
<br/>　　“你说得这些问题其实很容易解决。”
<br/>　　齐凡手指轻敲着桌面，说道：“首先，关于冷藏运输车不制冷的问题，只要把P病毒装进一个低温液氮瓶中，就能解决掉这个问题；第二，如果驾驶这辆冷藏运输车的人是布克尔特的手下，甚至这家冷藏公司的幕后老板就是布克尔特的话，你说得这些问题也就不存在了。”
<br/>　　陈应天轻轻点头，的确自己刚才所提出的不可控因素其实很容易解决，以布克尔特在这里的势力，这些东西或许根本就算不上问题。
<br/>　　“别管这么多，我们立刻去那家冷藏公司找找就知道了。”梅庆伟说道：“实践才是检验真理的唯一标准。”
<br/>　　齐凡把慕容飞留下的那支M200狙击步枪交给了布谷，目前这里只有他一个是狙击手。好枪也要到高手手中才能发挥最大的作用。
<br/>　　这种秘密搜查的行动不需要太多的人，所以齐凡只带了孙涛、陈应天和布谷三个人。
<br/>　　四个人进入市区的时候已经是傍晚时分，几个人就着矿泉水和干粮凑合着填饱了肚子，在冷藏公司附近转悠。
<br/>　　齐凡驾驶着汽车沿着马路行驶着，对坐在车内的其他三个人问道：“看到那辆停在冷藏公司门口的红色轿车了吗？”
<br/>　　“注意到了。”陈应天说道：“车里有三个人，刚才警察靠近过去的时候，我看到了副驾驶上那个人手中的望远镜，他们也应该是在监控那家冷藏公司，坐在主驾驶上的那个人跟我们在山林里面交过手，所以他们应该跟我们有着相同的目的。”
<br/>　　“可是他们没有手机，又是怎么知道布克尔特的P病毒有可能会藏在这里的呢？”坐在后排的布谷不解的问道。
<br/>　　“他们的确是没有手机，也定位不了P病毒的位置，但是他们可以通过调查布克尔特名下的产业进行排查。”陈应天解释道：“他们目前也只是怀疑，所以才没有行动，如果确定了P病毒就在这家冷藏公司里面，特种部队早就动手了，哪里还轮得到我们。小子，你还太年轻了。”
<br/>　　“我们必须要抓紧时间了。”齐凡驾驶着汽车冷藏公司背后，说道：“美国人既然盯上了这里，就一定会有所行动，我们必须要抢在他们之前动手，如果被他们抢先，那么就真的没我们什么事情了，而且那样一来，B组的人也会有危险。”
<br/>　　当地时间六点钟，天色逐渐暗了下来，冷藏公司的员工从后门离开下班。汽车内，齐凡、陈应天和孙涛在自己的uSP.45手枪上拧上消音器，检查了弹匣中的子弹后，藏起手枪推门下车；布谷则是背着装有XM2010增强型狙击步枪的背包朝着冷藏公司对面的楼房走去。
<br/>　　齐凡、陈应天、孙涛三个人避开了安装在冷藏公司后门围墙上的监控探头，翻围墙进入到冷藏公司中。
<br/>　　冷藏公司中还有人在值班，只不过这些值班人员却是带着AK-74S短突击步枪，这家冷藏公司果然是没有表面上那么简单。
<br/>　　齐凡打开了布克尔特那部手机，通过手机中的定位程序很快就找到了那辆红色小圆点所示的冷藏运输车。
