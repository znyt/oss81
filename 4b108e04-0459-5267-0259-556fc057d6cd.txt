        唐倩兮拉着童筱颖走过来后，将她往傅夜沉身上一按，硬生生地逼着她坐在了傅夜沉的大腿上。
<br/>
<br/>        面对突如其来的香玉满怀，傅夜沉身心一怔，那种异样的感觉，再次窜上他的心头。
<br/>
<br/>        童筱颖惊得慌忙起身，低着头，挪步站在了一旁。
<br/>
<br/>        众人只见，这位大金主搂住“小谢”的那双手，可是在半空中停顿了一下，才恋恋不舍地从容地收回。
<br/>
<br/>        童筱颖悄悄抬眸瞄一眼这位“连少”，接着这昏暗的光线，看清楚对方的脸后，赫然吃了一惊。
<br/>
<br/>        原来，他就是连少啊！
<br/>
<br/>        这么多天来，她都不知道他就是那个每晚都有给她高额打赏的“连少”。
<br/>
<br/>        不过，回想起昨晚他强吻她还差点强上了她的事情，她瞬间便对他没好感了。
<br/>
<br/>        “说句话啊！小谢。”唐倩兮一边向傅夜沉陪着笑脸，一边向童筱颖使了使眼色。
<br/>
<br/>        童筱颖瞄了一眼唐倩兮，极其不情愿地说道：“连、连少……今晚一切事端都是因我而起，我、我给你赔不是。希望您大人不记小人过。放过我们‘夜惑’的人。”
<br/>
<br/>        “你打算怎么赔偿我？”傅夜沉黑眸微眯，遮住了眼里原本透着的戾气，薄唇微扬地打趣。
<br/>
<br/>        童筱颖顿感语塞，声音哽在喉咙里，一句话也说不上来。
<br/>
<br/>        唐倩兮立即吩咐服务员去把“夜惑”里自酿的葡萄酒拿过来，给童筱颖倒了一杯，让她端去给这位连大少爷赔不是。
<br/>
<br/>        童筱颖不禁皱了下眉头，她刚吃过感冒药没多久，就让她喝酒，这岂不是要要了她的命。
<br/>
<br/>        “小谢，别愣着啊！赶紧给连少赔不是！”费尔南催促道。
<br/>
<br/>        童筱颖愤愤不满地瞪了老板费尔南一眼，只好端着高脚杯，走到傅夜沉的跟前，敬酒并道歉：“小谢在这给连少赔不是！”
<br/>
<br/>        说完，童筱颖便将高脚杯里的葡萄酒一饮而尽。
<br/>
<br/>        傅夜沉点了下头，韩剑锋便命人放了费尔南手下的那几个保镖。
<br/>
<br/>        至于这季运恒，傅夜沉直接让韩剑锋将他和他那群狐朋狗友送警局里去了。
<br/>
<br/>        季运恒被带走前，恶狠狠地瞪了童筱颖一眼。
<br/>
<br/>        童筱颖刚巧无意间对上他那目光，一脸不悦地皱起了眉头。
<br/>
<br/>        这事情，总算是平定了下来。
<br/>
<br/>        既然季运恒已经被带走了，而“夜惑”的保镖也被放了。
<br/>
<br/>        童筱颖觉得这儿没她什么事了，刚准备只身离开的时候，忽然感到一阵头晕目眩，甚至有点儿觉得恶心想吐。
<br/>
<br/>        她一个踉跄，未站稳脚，整个人晕了过去。
<br/>
<br/>        傅夜沉眼疾手快，伸手便将她揽入了怀中。
<br/>
<br/>        他刚抱她的时候，就发觉她身子骨有点儿凉，现在再抱住她的时候，她竟然全身发烫。
<br/>
<br/>        傅夜沉心一急，二话不说便将童筱颖打横抱了起来。
<br/>
<br/>        唐倩兮担心童筱颖会“出事”，连忙上前拦住了傅夜沉的去路，刻不容缓地说道：“对不起，连少。小谢不是出台小姐，哪怕您给她打赏再多，也不能将她带走！”
<br/>
<br/>        “滚！”傅夜沉眸光微沉，没好气地冷斥，直接抱着童筱颖，撞着唐倩兮的肩头继续往前走。
<br/>
<br/>        唐倩兮咬牙切齿地甩了甩手，刚准备追上去的时候，被费尔南伸手给拽了回来。
<br/>
<br/>        “小倩，你就别管这么多了！”费尔南顿了顿后，又接着说道，“我看这位连少大有来头！咱们这种小喽喽，可是招惹不起的！”
<br/>
<br/>        “我不能让那男人糟蹋了小谢！我答应过小谢的哥哥，要好好照顾小谢的！”唐倩兮急得恼火。
<br/>
<br/>        费尔南好心劝阻：“你没看连少那气势吗？你拦得住？”
<br/>
<br/>        唐倩兮一跺脚，气呼呼地说道：“就算是拦不住！我也得拦着！”
<br/>
<br/>        她答应过司峻的事情，就一定要做到！
<br/>
<br/>        哪怕筱颖不是司峻的亲妹妹，但是，司峻一直把筱颖当亲妹妹一样看待。
<br/>
<br/>        唐倩兮甩开了费尔南的手，疾步追了出去。
<br/>
<br/>        迷迷糊糊之际，童筱颖恢复了点意识，抬眸看到是那个男人的俊脸，惊愕得挣扎了起来。
<br/>
<br/>        傅夜沉被她这突如其来的动作，给招架不住，不得不先将她放下来，并让她依旧靠在自己怀中。
<br/>
<br/>        “你醒了？我现在带你去看医生！”他难得用温柔地语调跟她说话。
<br/>
<br/>        童筱颖却毫不领情：“谢谢，我自己会去看医生。”
<br/>
<br/>        如果不是他硬逼着她赔礼道歉，她就不会喝那杯葡萄酒了。
<br/>
<br/>        身后不远处，唐倩兮气喘吁吁地追了过来。
<br/>
<br/>        “小谢！连、连少！你不能带走小谢！”
<br/>
<br/>        在唐倩兮的眼里，这位连少抱走筱颖，无非是因为看上了筱颖，想带筱颖去开房罢了。
<br/>
<br/>        她不能让筱颖糟蹋在别的男人胯下！
<br/>
<br/>        所以，她此刻必须得带筱颖回去。
<br/>
<br/>        童筱颖听到唐倩兮的呼唤，立即试图推开傅夜沉，但她浑身软绵绵的，哪有力气推得动他。
<br/>
<br/>        “嫂嫂——”童筱颖气若游丝地唤了声。
<br/>
<br/>        唐倩兮见童筱颖像是要挣脱这男人的怀抱，于是主动上前，立马将童筱颖从傅夜沉的怀里拉了出来。
<br/>
<br/>        “连少，你请回。我会照顾小谢！”唐倩兮咬了咬下唇，凝起眉头，说出恭送的话。
<br/>
<br/>        傅夜沉看了看童筱颖的脸，见她那副假面眼镜下，双眸微垂，不愿多看他一眼，似乎明白了什么，便毅然转身，淡漠地离开。
<br/>
<br/>        他的那群手下，也尾随而去。
<br/>
<br/>        唐倩兮如释重负地松了口气。
<br/>
<br/>        童筱颖拉住唐倩兮的手，弱弱地说了一句：“嫂嫂，送我去医院吧！我……好难受……”
<br/>
<br/>        唐倩兮这才缓过神来，发觉童筱颖浑身发烫，急忙扶着她往自己的私家车那边走去。
<br/>
<br/>        当天晚上，季家的人急得团团转，因为无论他们怎么去公关，季运恒和他的那群狐朋狗友必须得在拘留所里待十五天。
<br/>
<br/>        季思妍急匆匆地跑到拘留所来看季运恒的时候，发现他嘴角还有淤青，一看就知道他这事喝酒闹事打架惹的祸。
<br/>
<br/>        “哥，你这段时间，在‘夜惑’里都瞎混了些什么啊？还有，你这是招惹了谁？怎么就被关进来了呢？害得爸妈都急死了！”季思妍瘪着红唇，连连抱怨。
